Late-breaking information about this release of BRO ASAP™.

This document contains information that may not exist in other ASAP information products. It discusses some of the significant changes in this release of software.

-----------------------------------------------------------------------------------------

Release Build 05V1R1

KERNEL --

Enhanced fidelity of raytracing in birefringent media.

Added support of finite-difference, time-domain (FDTD) simulations via FDTD Solutions™, by Lumerical (Optical Module)

Improved the DISPLAY command to better handle singular data sets.

LOCAL axes now work correctly for SPREAD NORMAL.

Fixed the COLLECTION command.

Added more informative diagnostic messages.

Integrated scripting support for alternate languages, such as Python, VBscript, and JScript.

Further memory-management improvements.

Fixed several Kernel run-time errors.

GUI

Tolerancing

Expanded tolerancing of optical systems within the ASAP script language.

Builder

Expanded Builder capabilities including new compact builder file format.

Macro Builder: a new tool for creating user macros.

Display Viewer

Updated the Display Viewer to allow rectangular region selection with WINDOW command.

Expanded Display Viewer has new ‘Tile’ option for viewing multiple displays.

Modules

New CAA V5 plug-in for CATIA (CATIA/CAD Module)

New Solid Works Add-in module for exporting Solid Works CAD geometry (CAD Module).

Other GUI improvements/features

New Polar Intensity Plot feature

Enhanced CIE features

New Light Source Library Wizard

3D Viewer includes now the fast transformation mode.

Chart Viewer has new chart styles manipulation with the Annotation toolbar.

Quick Start toolbar enhancements: drag-and-drop sources, glass properties, scatter models (including new diffuser sheets), user script INRs, and optical coatings from catalogs onto Builder/Editor.

New predefined "Diffuser Sheet" scatter models.

Translators

Improved IGES import options.

Improved Zemax Translator.
