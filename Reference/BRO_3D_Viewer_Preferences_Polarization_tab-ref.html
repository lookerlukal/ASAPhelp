Polarization Tab on 3D Viewer File Preferences Dialog

The File Preferences dialog consists of three tabs: Ray Set, Ray History, and Polarization. This topic describes the Polarization tab.

The File Preferences dialog is accessible from the File menu by selecting 3D Viewer File Preferences.

Property Type Property Value Property Description
Data Markers    
Default Default list of available data markers Select one of the ten markers for use in identifying rays with different wavelengths. If multiple wavelengths exist, ASAP selects additional markers to display different wavelengths.
Size Default = 3 Select the size of the data marker; 1 is the smallest and five is the largest size.
OK Button Select to accept values.
Cancel Button Select to close dialog without changing values.
