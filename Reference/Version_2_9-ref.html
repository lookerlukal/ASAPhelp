Version 2.9

June 1989

Easy-to-use GALOP interactive optimizing preprocessor is now available: 
   Graphical 1st-order layout, real-ray design and analysis
   Contact us for specs and prices
Renamed RABET commands: 
   DIFFUSE     ==> INTERFACE ... RANDOM 	
   CONSTRAINTS ==> BOUNDS (Avoids any conflict with CONSIDER)
New or enhanced RABET commands: 
SURFACES
   HORN  with polynomial cross sections
   OPTICAL ... EXPAND conic into aspheric terms 
      TEST for desired surface branch
      LOCAL coordinates and limits box
      BEND a surface parabolically 
      RENORMalize coefficients 
      MULTIPLE power entry
EDGES; CHARACTER 
   SAG with default axis override 
EXTREME rays in set listed 
GRID ... optional RANDOMizing of points 
FRESNEL 3 causes both polarizations to be traced 
POLARIZ sets initial state 	PLOT POLARIZATION displays current state e
   ... TEXT can be added to most graphics display 
LEVEL of random scatter 
ARROWS on rays 
OPDMAP of current wavefront 
DISPLAY part of complex vector FIELD
   DIRECTIONAL (polar or angular) diagram
SELECT enhanced 
RAY enhanced with KEEP 
COATING LAYERS and groups of layers can be physically defined 
TRACE ... OVERLAY/TEXT options added 
   XMEMORY controls paging of extend ray/beam information
Miscellaneous changes or enhancements to RABET:

Basic polarization capability addedSeveral diffraction orders can be generated at each interfaceMore than one random ray and LEVEL of scatter allowed now

Order doubled symmetric polynomials (within LOCAL coordinates)

Parabasals stored in relative form (more accurate)

Title record added to distribution file header

Miscellaneous changes or enhancements to utilities:

ACADIN creates AutoCAD DXF files from 3D vector files

HIDDEN program now has on-line graphics and autoscaling

Composite terminal driver with optional before pause

Miscellaneous changes or enhancements to input language:

Nested user macros (up to 6 levels)

Relative jumps with $GO

Labels for $ERR and $GO can now be up to 8 characters

Single line $IF structure

Transcendental math functions (e.g. SQRT,SIN,LOG,EXP,ATAN,BJ0 ...)

Repeated entries @ with alternating signs

$ASK, interactive prompting for any register value

Interactive prompting for unspecified macro arguments

$MENU executes current user macros from a menu

$SCR allows editing any set of registers via a programmable screen

Still in the works:

Non-Lambertian COATING/MODELS in STRAP

$ITER internal macro varies any group of program inputs and plots or minimizes any program output
