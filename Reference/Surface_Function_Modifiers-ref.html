Surface or Function Modifiers

Categories of surface/function modifiers are listed in this topic.

Surface/function modifiers fit into these categories:


  • Options that distort or deform the surface functions including ASYM, and BEND
  • MULTIPLE and ARRAY commands that create multiple versions of the surfaces
  • LOCAL command that bounds the surface
  • TEST command that controls the selection of (mathematical) branches
  • RENORM command that controls the computation and normalization of the polynomial coefficients
  • Linear transformations that shift, rotate, scale and others the edge points
