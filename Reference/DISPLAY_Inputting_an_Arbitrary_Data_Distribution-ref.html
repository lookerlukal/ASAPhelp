DISPLAY - Inputting an Arbitrary Data Distribution

Creating a user-defined distribution of data for plotting.

Syntax
 
DISPLAY 0 [ ’title’ ]

        u
  zlabel z flabel  ylabel y y’ j  xlabel x x’ i  [ ’title’ ]
 -f f’ f" ... 
       :
           jlines of i entries each
[ x x’ x" ... ] 
[ WRITE name ]
      :
            other DISPLAY subcommands
Remarks

You can create your own distribution of data for plotting (and optionally write it to the BRO00u.DAT or name.DIS file) with the above command sequence. If the header x and x’ values are equal, then the program looks for the additional line of i coordinates values (ordered but not evenly spaced) after the j line of functional data. Another way to create a DIS file is by writing your own external program to do this.
