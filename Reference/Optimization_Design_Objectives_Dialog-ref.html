Optimization Design Objectives Tab

The Design Objectives tab on the Optimization Setup Summary window lists currently defined design objectives for the optimization session. At least one item in the list must be enabled to run optimization.

Property Type Field Property Description
Name Read-only Displays unique name from the selected text in the INR script.
Objective Target Editable Displays the desired target for evaluating the expression.
Objective Weight Editable Displays the design objective weight and has the default value of 1.
Enabled State Drop-down menu Switch the state of any design objective between enabled and disabled. At least one objective must remain enabled. Default menu selection is Enabled.
Delete Button Deletes selected design objective from the Optimization Summary window. Multi-select rows with the Shift key to delete more than one row.
Close Button Closes the Optimization Setup Summary window, and ends the current optimization session.
Help Button Opens Help topic that describes the dialog box.
