Commands and Macros (V)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “V”.

  • VALUES (ASAP Command)
    Lists the functional values of the current distribution data file.
  • VANES (ASAP Command)
    Creates an anisotropic scatter model that simulates the scatter from a vane structure.
  • VARIABLES (ASAP Command)
    Declares which lens construction parameters are to be varied during optimization.
  • VCAVITY (ASAP Command)
    Creates a scatter model as a random collection of v-cavities.
  • VERSION (ASAP Command)
    Prints the version of the ASAP Kernel.
  • VIEW (ASAP Command)
    Make all future plots a true perspective view.
  • $VIEW (ASAP Macro)
    Immediately processes the system geometry in the current 3D vector file (*.vcr) using the operating system command defined with the VIEWER switch or environment variable.
  • VIOLATION (ASAP Command)
    Controls the handling of paraxial departure and positivity violations during future SPREAD and FIELD commands.
  • VOXELS (ASAP Command)
    Accumulates on the next TRACE, in a 3D array of volume elements (voxels), the flux-per-unit volume or irradiance on an area.
  • VUFACETS (ASAP Command)
    Shortcut for faceting and viewing current objects.
