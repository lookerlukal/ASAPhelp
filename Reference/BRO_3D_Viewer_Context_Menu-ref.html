3D Viewer - Context Menu

This topic describes the context menu that is accessible in the 3D Viewer.

Right-click the Tree View pane to access the context menu.

Property Type Keyboard Keys Property Description
Properties Enter Opens the Object Properties dialog box for the currently selected object.
Optional Properties Shift-Enter If applicable, opens Optional Properties dialog box for the currently selected object. Applies to rayset and DIS files, and the test scene.
Close File Delete Closes selected file (applicable only when a top-level object is selected)
Make Visible V Makes the currently selected object(s) visible.
Hide H Hides the currently selected objects(s).
Wireframe W Changes draw mode of the currently selected object(s) to wireframe view.
LitWireFrame L Changes draw mode of the currently selected object(s) to illuminated wire frame.
Flat Shaded F Changes draw mode of the currently selected object(s) to flat shading.
Smooth Shaded S Changes draw mode of the currently selected object(s) to smooth shading.
Highlighted Edges E Changes draw mode of the currently selected object(s) to highlighted edges.
Zoom Extents N Zooms the view so that the currently selected object(s) fit within the window.
Isolate and Zoom Extents I Makes visible only the current selection, and zooms the view so that it fits within the window.
Isolate w/o Zoom Extents M Makes visible only the current selection without (w/o) zooming the view.
Poincaré Sphere Options   Opens the Poincaré Sphere Options dialog to select options for displaying the Poincaré sphere. Polarized data must be present in the current DIS file for this command to be enabled on the menu.
Display Polarization Information   Displays the S1, S2, and S3 polarization state data, polarization reference vector, ray flux, and ray direction for a selected point on the Poincaré sphere. This command is enabled when you have selected one or more points. The data is displayed in the graphic pane in a popup box.
Save Polarization Information   Opens the Windows Save As dialog to save ray data for multiple selected points to a user-specified text file if points are selected on the Poincaré sphere.
Computer Ellipse   Computes the polarization ellipse parameters if the current DIS file contains polarization data in the Stokes Vector form. The data is displayed in the Command Output window.
