Tips
  • Isolate your data (objects or ray data) using the CONSIDER and/or SELECT commands.
  • Set windows orientation, sampling and others.
  • Enter the specific graphics command.
  • If the desired graphic is an overlay (or composite) of several individual graphics, this process may be repeated for each one.
