Commands and Macros (R)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “R”.

  • RACETRACK (ASAP Command)
    Creates a racetrack or rectangular edge with smooth (continuous) rounded corners.
  • RADIAL (ASAP Command)
    Rotationally averages the current distribution data, and optionally computes the encircled energy.
  • RADIANT (ASAP Command)
    Calculates the far-field incoherent intensity or radiance of the currently selected ray data.
  • $RAN (ASAP Macro)
    Resets the random number seed for the "~" operator to the integer i (default 2000000001).
  • RANGE (ASAP Command)
    Adjusts the vertical plotting scale of current distribution data.
  • RAWDATA (ASAP Command)
    See the command BSDFDATA.
  • RAY (ASAP Command)
    Defines and traces a single ray.
  • RAYS (ASAP Command)
    Sets the total number of rays in the virtual memory paging.
  • RAYSET (ASAP Command)
    Specifies an arbitrary set of rays.
  • $READ (ASAP Macro)
    Start reading future records from the beginning of the given file name (with default extension .INR).
  • RECTANGLE (ASAP Command)
    Creates a rectangular edge/curve.
  • REDEFINE (ASAP Command)
    Adds options to an object.
  • REDUCE (ASAP Command)
    Selects a subset of the current distribution data and rejects remaining data.
  • $REG (ASAP Macro)
    Displays the contents of registers and associated variables.
  • REMOVE (ASAP Command)
    Subtracts best-fit polynomial.
  • RENDER (ASAP Command)
    Creates rendered graphics of the system geometry and ray trace.
  • RENORM (ASAP Command)
    Renormalizes coefficients of surface/function polynomials.
  • REPEAT (ASAP Command)
    Repeats previously defined entity data.
  • REPLICATE (ASAP Command)
    Replicates (copies) a distribution.
  • REPLOT (ASAP Command)
    Plots data stored in the 3D vector file within the currently defined graphics window.
  • RESET (ASAP Command)
    Reinitializes all control settings to those at program startup.
  • RESTRICT (ASAP Command)
    Further controls ray search within lens.
  • RETURN (ASAP Command)
    Returns to ASAP top command level (that is, the ASAP> prompt).
  • REVERSE (ASAP Command)
    Reverses the propagation direction of all currently defined rays.
  • REVOLUTION (ASAP Command)
    Creates a surface specified by spinning a 2D curve around a given axis.
  • REVOLUTION (Fitted) (ASAP Command)
    Creates a surface by spinning a 2D curve fit to data points.
  • RIGHT (ASAP Command)
    Creates a simple prism that deviates the rays by 90 degrees.
  • RMS (ASAP Command)
    Scatter model given surface variations for RMS, fall-off, and autocorrelation.
  • ROOF (ASAP Command)
    Two rectangles that are "hinged" at a common side.
  • ROTATE (ASAP Command)
    Rotates an entity about a point.
  • ROUGHNESS (ASAP Command)
    Assigns a roughness to the entity of an object, and conserves energy between specular and scattered beams.
  • ROUNDED (ASAP Command)
    Creates a rounded-corner, rectangular edge/curve.
  • RPM (ASAP Command)
    Defines a realistic polarizer.
  • RRM (ASAP Command)
    The RRM command defines a realistic retarder model in ASAP.
