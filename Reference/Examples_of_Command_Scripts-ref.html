Examples of Command Scripts

You can open any one of the hundreds of example scripts in the form of ASAP INR files. These scripts are accessible in three ways: from the command help topics, from your install directory, or directly from the Quick Start toolbar. Select the Example Scripts tab on the toolbar, and then select the command name or example file name you are interested in viewing. More than one example may be listed for a command or category of commands. The script will open in the Editor window.

If the Quick Start toolbar is not visible in your ASAP window, go to the View menu and select Properties Bar. Examples are categorized into these groups:


CategoriesGroups of ASAP commands
CommandsIndividual ASAP commands
JScript Examples of JScript commands
KeywordsCommands grouped by keywords
VBScriptExamples of VBScript commands

You can also access the Index of Example Scripts on your computer in your asap installation area under projects\examples\example_scripts.html.

You can copy all or portions of these examples to the Clipboard, paste them into an Editor window, and use them as templates for creating your own ASAP files.
