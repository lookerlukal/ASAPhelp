Optimization Objective Constraints Tab

The Objective Constraints tab on the Optimization Setup Summary window displays the list of currently defined objective constraints and penalty functions for the current optimization session. Objective constraints and penalty functions are optional.

Property Type Field Property Description
Name Read-only Displays name from selected line in the INR script.
Minimum Editable No default; optionally enter a value.
Maximum Editable No default; optionally enter a value.
Enabled Drop-down menu Switch the state of any objective constraint between enabled and disabled. Default menu selection is Enabled.
Penalty Function Editable Specifies derived scalar real-typed named variables, which contribute to a Penalized Figure of Merit (PFOM).
Delete Button Deletes the selected objective constraint from the Optimization Summary window. Multi-select rows with the Shift key to delete more than one row.
Close Button Closes the Optimization Setup Summary window, and ends the current optimization session.
Help Button Opens Help topic that describes the dialog box.
