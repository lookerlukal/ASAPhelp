Commands and Macros (XYZ)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letters “X, Y, Z”.

  • XEQ (ASAP Command)
    Specifies immediate application of current linear transformation.
  • XMEMORY (ASAP Command)
    Specifies the type and amount of ray data to store, subject to available hardware and operating system limits.
  • XY[Z] and Other Plot Window Overrides (ASAP Command Argument)
    Sets the window to the given directions and size for only the currently active plot.
  • ZERNIKE (ASAP Command)
    Creates a surface specified by an explicit Zernike polynomial.
