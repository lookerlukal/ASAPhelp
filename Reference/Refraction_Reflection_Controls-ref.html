Refraction/Reflection Controls

The HALT command and the following interface controls become the defaults used by any object for which they are not explicitly set.

FRESNEL

SPLIT

LEVEL

Since they are identical to the OBJECT subcommands, you should not use them at the OBJ> prompt. If you are, issue a RETURN command first.
