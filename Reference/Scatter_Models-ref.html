Scatter Models

ASAP supports the scatter models listed in this topic for wide-angle (far from specular) scatter.


LAMBERTIAN simple Lambertian scatter
HARVEY Harvey smooth optical surface scatter
POLYNOMIAL/TRINOMIAL/BINOMIAL scatter defined by a polynomial or fitted data
USERBSDF user-defined Bi-directional Scattering Distribution Function (BSDF) model scatter)
VANES scatter from vanes along an axis
PARTICLES scatter from particulates on a surface
VCAVITY scatter from a surface of v-shaped cavities
BSDFDATAinterpolates scatter from entered values
SUMscatter a composite of BSDF models

Scatter models are entered after the MODELS command and become part of the models database. These are later referenced by the SCATTER command during the definition of an OBJECT.

Scatter models are intended for use with the TOWARDS command for importance sampling.

  • SCATTER MODEL and TOWARDS Overview
    This topic gives several tips for scattering rays towards a specified small area.
  • Scattering with ROUGHNESS RANDOM
    This topic gives several tips for using ROUGHNESS RANDOM, which simulates scatter from a rough surface by randomly perturbing the direction of a reflected or refracted ray. The actual surface of the object is not altered.
