Commands and Macros (B)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “B”.

  • BEAMS (ASAP Command)
    Sets the default coherence, propagation, and shape characteristics for future ray creation.
  • $BEEP (ASAP Macro)
    Causes computer to emit a beep.
  • BEND (ASAP Command)
    Bends the last surface along a parabolic curve.
  • BEZIER (ASAP Command)
    Creates a Bezier edge/curve in the plane.
  • BICONIC (ASAP Command)
    Creates a cubic or quartic biconic surface.
  • BILATERAL (ASAP Command)
    Folds data around a coordinate axis/axes to reduce the number of rays needed for a calculation where symmetry is expected.
  • BOUNDS (ASAP Command)
    Assigns surfaces and edges as boundaries to objects.
  • BRANCH (ASAP Command)
    Specify an absolute or relative branch of the object name tree for future object definitions.
  • BSDFDATA (ASAP Command)
    Indirectly or directly interpolates from entered BSDF values.
