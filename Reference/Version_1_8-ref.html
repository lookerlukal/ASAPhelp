Version 1.8

February 1988

COMMAND CHANGES:
PAGE==> PIXELS
DUMP==> SAVE (now with optional file name)
ENHANCED COMMANDS:
GENERALrequires reference point
RAYinteractive input
FRESNELpolarization selection
DEFORMnew default definition and AXIS option
SHAPEadditional parameter
INTERFACcan reference COATING definitions
WAVELENGmultiple wavelengths, UNITS option
ROTATEnew default center of rotation
SOURCEmultiple types now allowed
NEW COMMANDS:
$FASTinternal macro for fast numerical input
ALIGNnew linear transformation
PLACE" " "
XEQ " " "
SELECTselects ray set or source number
COATINGSdefine spectral properties of interface
USERSURFuser-programmable surface intersection
CODE CHANGES:

VIRTUAL.PGS and *.SYS files are incompatible

TEK10 and REGIS drivers are buffered to reduce I/O

Graphics codes 4, 5-30, 31, and 127 are different

Hundreds of new error/warning messages have been added

Numerous bugs have been squashed

Commands can now be entered while in graphics mode

Higher-order Hermite-Gaussian beams can be propagated

More general ways to specify direction vectors

OBJECTS and MEDIA can now be referenced by name
