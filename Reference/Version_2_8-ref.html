Version 2.8
Version 2.8

July 1988

RENAMED COMMANDS:
 
                   STATS OBJECTS         ==> PATHS
                   INTERFACE ... SCATTER ==> INTERFACE ... RMS/BSDF
                   INTERFACE ... RANDOM  ==> OBJECT; # RANDOM ...
                   DISPLAY;REPLACE       ==> DISPLAY;WRITE
ENHANCED COMMANDS: 
                   $GO       can jump directly to a particular macro line
                   $IF       AND/OR constructs now permitted
                   MEDIA     Inhomogeneous (GRIN) added
                   MOVE      TO PLANE added
                   CUTOFF    Upper limit on # of objects encountered
                   CURVE     Additional entry and least-squares fit
                   FFAD      Ray SPOTS can be added to display
                   PARABASAL Beam scaling entry replaced by beam divergence
                   RAY       SEARCH reset and interactive tracing
                   OBJECT    One edge faceted objects allowed 
                   TRACE     Can STEP a given number of objects
                   LIST      Info on SOURCES and all parabasal rays
                   SPOTS     Plotting all parabasals, distribution file
                    END       Automatic system save to LASTEXEC.SYS
                    SYSTEM    FROM defaulted to LASTEXEC.SYS
                   RAYSET    can be completely recovered from last run
                   SELECT    based on previous ray objects added
                   USERSURF  rotations now saved and passed to user
                   SOURCE    can use last GRID to define distribution
                   DISPLAY   Allows the naming of distribution files
                   SHIFT     a distance along a given direction
                   FIELD     Saves specified distribution to file
                   PROFILES  New default operation
NEW COMMANDS: 
                   WIDTHS    Sets beam width scaling factors
                   EDGE;LINE Equally divided straight line
                   USERFUNC  User-programmable function (GRIN, surface)
                   IRRADIANC Sets irradiance direction for SPREAD/FIELD
                   DISPLAY
                      MODIFY  Allows modification of data regions
                      CONTOUR Contour plot
                      RADIAL  Averaging distribution radially about a point
CODE CHANGES: 
                   VIRTUAL.PGS and *.SYS files are incompatible
                   Nested parentheses can be used in input arithmetic
                   Input registers can now be accessed by assigned literal
                   On-the-fly macro definitions require 2 prefixes, e.g. $$
                   Up to 6th-order 84-term SURFACES and 25-point EDGES
                   Parabasal ray errors now halt trace of that beam
                   New header for unformatted distribution files (FOR009)
                   Improved accuracy in OPL calculations
                   Numerous bugs have been squashed (SOP)
                   Default DEFORM and USERSURF definitions changed
                   General clipping and device support added to VIEW3D
                   ACADIN program for 3D file to AutoCAD-9 DXF conversion
IN THE WORKS: 
                   Non-lambertian COATING/MODELS in STRAP
                   $ITER internal macro varies any group of program inputs
                      and plots or minimizes any program output
                   More "bells and whistles" (SOP)
                   Interactive optimizing graphical preprocessor (4th qtr.)
                   ACADIN program for 3D file to AutoCAD-10 DXF conversion
