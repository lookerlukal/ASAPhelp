3D Viewer Uses

The BRO 3D Viewer™ displays ASAP geometry data that is stored in the vector (*.vcr) file.

You can also use the 3D Viewer for:


  • Calculating and viewing the far-field incoherent radiant intensity pattern (flux per solid angle) of a rayset (written from ASAP with the DUMP command).
  • Displaying two-dimensional, positional, distribution data files in their proper spatial context (for example, files produced by the SPOTS POSITION or SPREAD NORMAL commands in ASAP).
  • Binning, in volume space, a collection of ray paths that are stored in an ASAP ray history file (generated by the SAVE command in ASAP).
