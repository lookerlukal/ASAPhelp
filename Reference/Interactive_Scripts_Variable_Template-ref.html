Interactive Scripts - Variable Template

To create a template file, you must first create an INR file in ASAP. Designate variables for the template with the following syntax. 

<|VariableName|>

  • The variable must start with “<|”.
  • The variable must end with “|>”.
  • The VariableName can be any name you designate.
  • No spaces can exist.
  • No duplicate variable names can exist.

You can then add the template to the Interactive Scripts page on the Quick Start toolbar, using the methods described. When you drag and drop a template file to open it, you are prompted with a dialog to fill in the variables.

Example - Template INR Script 

SYSTEM NEW
RESET

MEDIA
  COATING PROPERTIES
    1.0   0.0  'REFLECT'
    0.0   1.0  'TRANSMIT'
    0.5   0.5  'BEAMSPLITTER'

!! Create geometry
SURFACE
  PLANE Z 0 RECT 10 10 
OBJECT 'ELEMENT_1'
  INTERFACE COATING REFLECT <|MEDIA1|> AIR
SURFACE
  PLANE Z 1 RECT 10 10 
OBJECT 'ELEMENT_2'
  INTERFACE COATING TRANSMIT <|MEDIA2|> AIR

!! Verify the properties of the object
PRINT OBJECTS

!! Change the properties associated with the first object
OBJECT 1
  INTERFACE COATING BEAMSPLITTER <|MEDIA3|> AIR
  
!! Verify that the changed properties
PRINT OBJECTS

RETURN

When you drag/drop the script, you are prompted to fill in the missing variables in the Interactive Scripts - Geometry Template dialog.

When you click OK, you are prompted to save the file under a new name so that the original template file can be preserved. The new file is then opened as: 

SYSTEM NEW
RESET

MEDIA
  COATING PROPERTIES
    1.0   0.0  'REFLECT'
    0.0   1.0  'TRANSMIT'
    0.5   0.5  'BEAMSPLITTER'

!! Create geometry
SURFACE
  PLANE Z 0 RECT 10 10
OBJECT 'ELEMENT_1'
  INTERFACE COATING REFLECT SCHOTT_BK7 AIR
SURFACE
  PLANE Z 1 RECT 10 10 
OBJECT 'ELEMENT_2'
  INTERFACE COATING TRANSMIT AIR AIR

!! Verify the properties of the object
PRINT OBJECTS

!! Change the properties associated with the first object
OBJECT 1
  INTERFACE COATING BEAMSPLITTER AIR AIR
  
!! Verify that the changed properties
PRINT OBJECTS

RETURN

To edit the actual template file, you can double-click it or right-click and select Edit from the pop-up menu.
