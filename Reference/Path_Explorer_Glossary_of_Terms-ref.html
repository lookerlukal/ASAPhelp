Path Explorer - Glossary of Terms

This glossary applies to the Path Explorer feature in ASAP.

Path-sets (*.pxp): A set of files containing path query results that are generated by a Path Explorer kernel session.

Ray history file (*.his) A file containing path data created by the SAVE command

VCR file (*.vcr): A vector file containing a 3D representation of the distribution in the system created by the VIEW or VUFACETS command.
