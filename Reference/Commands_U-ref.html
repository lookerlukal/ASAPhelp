Commands and Macros (U)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “U”.

  • UNITS (ASAP Command)
    Defines the system units.
  • $UNVAR (ASAP Macro)
    Determines the type of message to be issued for future uninitialized (used before set) variables/registers.
  • UPDATE (ASAP Command)
    Controls the entity storage updating.
  • USERAPOD (ASAP Command)
    Specifies a beam irradiance or intensity apodization.
  • USERAPOD ANGLES (ASAP Command)
    Specifies 3D angular apodization of volume emitters.
  • USERAPOD BOTH (ASAP Command)
    Apodize in position and/or direction the full 3D radiation patterns of volume emitters.
  • USERBSDF (ASAP Command)
    Specifies an isotropic or anisotropic scatter model.
  • USERCURVE (ASAP Command)
    Creates a piecewise linear curve (or bilinear surface) from a $FCN function.
  • USERFUNC (ASAP Command)
    Creates a surface specified by a user-defined function.
  • USERSAG (ASAP Command)
    Creates a surface with a user-defined radial or toric profile.
  • USERSURF (ASAP Command)
    Creates a user-defined surface with a user-defined surface intersection.
  • UVSPACE (ASAP Command)
    Tags current curve to be defined in parametric-space of an object.
