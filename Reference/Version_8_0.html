Version 8.0
Version 8.0

Release Notes for ASAP 8.0 Kernel as of October 23, 2003.

Changed commands (affects backward compatibility)
 
ABERRATIONS
  STORE passes sign prefix on .INR/.MAC entry to generated EXPLODE command.
XMEMORY page file for Pro is incompatible with previous and Basic versions.
New Commands
 
LENSES
  IMPORT simple centered lens and operating conditions from a ZEMAX file.
  RESTRICT controls ray "search" range within a LENS entity.
  ABERRATIONS
    ANALYZE lens after changing any of the first-order operating conditions.
DISPLAY
  TERRAIN lists number/spacing of peaks/valleys on a line (fringe statistics).
  REMOVE (subtract) best fit polynomial of given degree from data.
SURFACES
  ROOF is two rectangles with common edge and controllable angle between them.
DOPL controls ray optical path lengths from DIFFRACTive surface INTERFACEs.
Enhanced Geometry Commands
 
LENSES
  SEQUENCE can contain reflecting and/or refracting "paraxial" planes.
  ALTER can now rotate one or more conicoids within a LENS entity.
EXPLODE uses new convention for generating unique and meaningful object names.
CURVES
  ALTER
      Rotation of one or more points is now an option.
      Can now specify a user-defined $FCN to remap one or more points.
SURFACES
  BICONIC
    Can also blend conic profiles of implicit using crossection "q" factor.
    6th-order singularity-free implicit approximation to standard explicit.
  ALTER can now increase degree before setting any non-zero terms.
  can remap points using a $FCN instead of just coordinate reordering.
  ARRAY optional RANdomization similar to that used by a GRID.
  USERFUNC option to be EXPLICIT (f(x,y)-z=0 instead of general f(x,y,z)=0).
CADEXPORT IGES
  Levels (drawing layers) set to main branches of name tree.
  Actual BRO color definitions are now included and used in file.
Enhanced Properties Commands
 
MEDIA
  Maximum overall name (catalog_glass) increased from 16 to 24 characters.
  Catalogs
    CONRADY and HERZBERGER interpolation added.
    Valid spectral range (highly absorbing outside) can now be specified.
  Negative refractive indices (double negative metamaterials) are now allowed.
  Basic isotropic optical ACTIVITY (circular birefringence) has been added.
OBJECT
  HALT axis can now be the plus or minus normal direction (one-way tracing).
  REDEFINE THICK more fully implemented now.
COATINGS can now reference anisotropic surface MODELs.
MODELS
  VOLUME scatter logarithmic PLOT has been added.
Enhanced Analysis Commands
 
SELECT
  By "element" number or parametric coordinate added.
  Number of cached bit flags is now limited only by available virtual memory*.  
DISPLAY
  plot also writes maximum function and/or total integrated values.
  ENCLOSED plot also writes total integrated value.
  DIRECTION plot now shows global polar axis orientation.
  OFFSET of actual distribution data values is now an additional option.
  Can accept much larger distributions depending on available virtual memory*.
  PLOT3D tilt and rotation angles of view can now be specified.
MOVE (iterate) rays TO a nearby SURFACE entity.
SEED derived from system clock if no options given.
FFAD LIST option prints flux, centroids, and RMS sizes of each source.
STATS
  SOURCE lists the current number of rays and total flux from each source.
  Added option to sort and truncate below a given percentage of total flux.
EMIT DATA
  Will directly take a grey-scale or color BMP bitmap file.
  Can accept much larger distributions depending on available virtual memory*.
FIELD
  Can process larger distributions depending on available virtual memory*.
  Option to free-space PROPAGATE result a finite distance or to far-field.
VOXELS total number can be much larger depending on available virtual memory*.
DECOMPOSE accepts larger distributions depending on available virtual memory*.
PROFILES will now show all parts of a "deep" LENS conicoid.
SHAPE for external GAUSsian extended to include supergaussian beams.
ABERRATIONS
  Successive PLOTs can be put into quadrants instead of full window.
  Has new option to place image solves on any surface not just last.
Macro Language

Can now reference comment (name string) argument in a macro using #00.

$MSGS will turn ON or OFF warnings and informative (but not error) messages.

$ITER (and $EVAL) maximum number of variables, functions, and steps increased.

$GRAB (and $PLAY) buffer size increased from 2,000 to 10,000 lines of output.

$EVAL will assume first variable value if no functions given.

$SCR field can be forced to always display number in exponential format.

Other Changes

.SYS binary files are incompatible with previous versions.

GES import will use Subfigure (308) name if Instance (408) not supplied one

Standard CADEQUIV.LIB has been updated to take advantage of new features.

Default ARROW size has been halved (high-resolution graphics is now the norm).

Maximum number of sources (distinct wavelengths) quadrupled*.

Polychromatic raysets colored in plots by wavelength instead of source number.

New GLA2CAT utility for converting ZEMAX .AGF and OSLO .GLC glass catalogs.

Numerous minor "tweaks", refinements, speed, and memory optimizations.

* Pro version only.
