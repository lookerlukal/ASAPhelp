Ray Distributions - Apodizing

APODIZE operates immediately on all the currently selected or considered rays and, therefore, can be used before or after a TRACE. USERAPOD applies only to rays when they are created and just for the combinations described in this topic.

Note:

An "X" in the above table indicates an apodization method that should usually produce acceptable results. However, there can be some recommended restrictions. For example, with EMITTING OBJECT, the intensity distribution must be rotationally symmetric for apodization in direction. It is also possible to attempt to use a choice without an “X” in the above table. ASAP does not prevent you from attempting such an apodization, although the results are generally not useful.
