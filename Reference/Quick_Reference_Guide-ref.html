Quick Reference Guide

The Quick Reference Guide is a group of topics that summarizes key commands and user interface locations.

The topics are not inclusive of all ASAP commands.

  • ASAP Syntax - Quick Reference
    This topic provides a reference for commonly used ASAP syntax.
  • Optical System Model - Quick Reference
    The following information is based on starting a new Builder line, or entering command syntax in the Command Input window.
  • Verifying Optical System Model - Quick Reference
    This topic provides a list of commonly used commands for verifying an optical system model.
  • Source Models - Quick Reference
    This topic provides a list of commonly used commands for verifying an optical system model.
  • Verifying Source Model - Quick Reference
  • Ray Tracing and Analysis - Quick Reference
    This topic provides a list of commonly used commands for ray tracing and analysis.
  • Miscellaneous Useful Commands - Quick Reference
    This topic provides a list of commonly used commands for re-initializing system databases and the virtual.pgs file, as well as for restoroing an ASAP session.
  • Macro Formatting - Quick Reference
    Macro format for both library (filename.LIB) and on-the-fly macro definitions: use the $IO LIBRARY filename to associate that macro library with an ASAP session.
