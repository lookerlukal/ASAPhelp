Optimization Design Variable Tab

The Design Variable tab on the Optimization Setup Summary window lists currently defined design variables for the optimization session. At least one variable must be defined to begin optimization.

You can change the value of any of the editable fields, delete selected variables, and reset the current value of the selected variables to the nominal values.
Property Type Field Property Description
Name Read-only Displays unique name of design variable from selected text in the script. The name can include up to 32 characters. If the name is not unique among existing design variabales, an informational message is displayed. Spaces in the name must be excluded.
Alias Editable Defaults to the unique design variable name. Enter a different unique name if desired.
Nominal Editable Displays value from the selected text in the script.
Minimum Editable Default is infinitely negative. The entered value must be less than the maximum.
Maximum Editable Defaults to infinitely positive. The value must be greater than the minimum. A warning is issued when no more variables can be defined.
Step Size Editable Enter a value that is less than the absolute value of the difference between the maximum and minimum value.
Manufacturing Tolerance Editable Enter a value that is a non-negative real value. If a value is not entered, the default value is 0.
Enabled State Drop-down menu Lists options to enable or disable the design variable during the current optimization session. When enabled, the variable is considered in optimization; when it is not set, the value is static and does not change from one trial to another.
Design Variable Current Value Read-only Displays current value of the design variable after the last trial run, if you have previously run optimization. If no trial solutions have been calculated, it displays the nominal value from the script.
Delete Button Removes the selected design variable from the Optimization Summary window. Multi-select rows with the Shift key to delete more than one row.
Reset to Nominal Button Sets the current value of the design variable to the nominal value, if you want to rerun optimization. Multi-select rows with the Shift key to reset more than one row.
Close Button Closes Optimization Setup Summary window, and ends the current optimization session.
Help Button Opens Help topic that describes the dialog box.
