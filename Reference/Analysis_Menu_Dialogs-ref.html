Analysis Menu Dialogs

Topics in this section of the Contents tab include dialogs accessible from the Analysis menu.

Select one of the commands on this menu that is followed by an ellipse (...).

  • Consider Objects/Rays Dialog
    Select the objects to which the CONSIDER command should be applied.
  • SELECT Rays Dialog
    Designates current set of rays that ASAP is to consider in calculations and output. This dialog box is accessible when you right-click in the Display Viewer window.
  • Discard Rays Dialog
    ASAP can discard inactive rays when several rays are traced and only a small subset is analyzed extensively.
  • Sort Rays by Path Dialog
    Produces a table of summarizing propagation paths of currently defined rays.
  • Show Ray History Dialog
    Recalls ray data from a previously saved binary file for plotting or listing.
  • FOCUS Dialog
    Lists the best 3D focal point and RMS deviations of the current ray data, from this point.
  • Calculate Flux Distribution Dialog
    Creates a geometric spot diagram, and, if selected, creates a far-field incoherent intensity or radiance for the currently selected ray data.
  • OPD Map Dialog
    Creates a geometric wavefront map in the plot window of the currently selected ray data.
  • Full Field Aberration Display Dialog
    Formats currently selected ray data and plots full-field aberration displays (FFAD). These plots are useful for identifying aberration nodes on the image surfaces of non-centered or perturbed systems.
  • CIE Color Analysis Dialog
    The CIE Color Analysis dialog box is accessible from the Analysis menu in ASAP. Use it to select settings for CIE calculation and to initiate the calculations.
  • Calculate Field Energy Distribution Dialog
    Calculates the exact coherent/incoherent energy distribution.
  • Calculate Field Properties Dialog
    Calculates the exact complex field distribution.
  • Incoherent RADIANT Calculation Dialog
    Calculates the far-field incoherent intensity and radiance of selected ray data.
  • STATS Dialog
    Lists statistics of the currently selected ray data.
  • Decompose Coherent Field Dialog
    Decomposes an existing field into a new set of Gaussian beams.
