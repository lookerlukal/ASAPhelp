Commands and Macros (T)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “T”.

  • TABLE (ASAP Command)
    Creates a table of the current data.
  • TELESCOPE (ASAP Command)
    Creates a one- or two-mirror telescope.
  • TERRAIN (ASAP Command)
    Prints number of peaks/valleys and FWHM (Full-Width-Half-Maximum) of highest peak.
  • TEST (ASAP Command)
    Selects a particular branch of a surface.
  • TEXT (ASAP Command Argument)
    Prints text annotation within the graphics window.
  • TEXTFILE (ASAP Command)
    Writes the current data to a user-definable text file.
  • THRESHOLD (ASAP Command)
    Resets all data values.
  • $TIC (ASAP Macro)
    Displays elapsed and CPU time.
  • TITLE (ASAP Command)
    Specifies default title for future graphics.
  • TORUS (ASAP Command)
    Creates a toroidal or doughnut surface.
  • TOWARDS (ASAP Command)
    Specifies importance area or direction sampling for scattered rays. Required when specifying a SCATTER MODEL.
  • TRACE (ASAP Command)
    Traces all currently selected rays in storage.
  • TRANSPOSE (ASAP Command)
    Rotates the current distribution data by 90 degrees.
  • TREE (ASAP Command)
    Displays object name tree.
  • TUBE (ASAP Command)
    Creates a tube with rectangular/elliptical cross-sections.
