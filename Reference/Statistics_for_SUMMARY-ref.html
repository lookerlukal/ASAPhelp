Statistics for SUMMARY

The SUMMARY command, a subcommand of HISTORY, is described in this topic.

SUMMARY produces output which looks like the following example: 

 Path-Set        :  ACTIVE
 Paths            :  300
 Total Flux      :  1.000000
            AVG                     RMS                      MIN                        MAX
 Flux :   0.3333335E-02   0.6798726E-03   0.2400000E-02    0.4000000E-02
 X_f  :   -.1986822E-09   0.2872279           -.4500000            0.4500000
 Y_f  :   1.540351             0.9933338          -.2500000             2.684211
 Z_f  :   2.807016             0.8789477          1.736842               4.000000
 X_i  :   -.1986822E-09   0.2872279           -.4500000             0.4500000
 Y_i  :   0.1092752E-08   0.2872280           -.4500000             0.4500000

 Z_i  :   0.000000            0.000000             0.000000               0.000000

  • Path-Set is the setname given to SUMMARY, or the ACTIVE pathset if no setname is given. Paths is the number of paths in the Path-Set. Total Flux is the total flux delivered by all paths in the Path-Set.
  • Then we see average (AVG), root-mean-square (RMS) and extreme (MIN and MAX) values for several parameters:
  • Flux: the flux delivered by the paths in the Path-Set.
  • X_f, Y_f, Z_f : The final (f) coordinates (X,Y,Z) of the paths in the Path-Set.
  • The final location of each path is the location at which ray-tracing stopped.
  • Typically this is the last intersection, but for missed rays is extended by the vector defined by the MISSED command.
  • X_i, Y_i, Z_i : The initial (i) coordinates (X,Y,Z) of the paths in the Path-Set.
  • The initial location of each path is the location of it's originating source ray.
