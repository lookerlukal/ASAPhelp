Create Rays/Beams

This topic lists commands that are used when creating rays or beams.


  • A slash between entries indicates that only one of the entries is allowed at a time.
  • Commas indicate that any combination of entries is acceptable.
  • A semi-colon indicates that the commands must be grouped together.

RAYSETBegin defining a table of rays/beams
GRID RECT/POLAR/ELLIP/OBJECT/HEX/WIN/DATDefine a spatial grid of rays/beams
SOURCE POSIT/DIREC/FOCUS/LINE/WAVESpecify source(s) for the rays/beams
EMITTINGComposite of random emitters
EMIT DISK/RECTSimulate a random emitting surface
EMIT CONE PYRAMID/BOX/SPHEROIDSimulate a random emitting volume
EMIT FILAMENT/HELIXSimulate a random emitting curve (wire)
EMIT RAYSArbitrary collection of rays
EMIT IESEmitter specified by IES file data
EMIT ENTITY/OBJECTRandom emission from defined surface(s)
EMIT DATAEmit according to distribution data file
GAUSSIANRays for coherent astigmatic Gaussian model
DECOMPOSE POSITION/DIRECTIONDecompose an existing coherent field
