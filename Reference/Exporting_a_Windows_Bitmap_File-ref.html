Exporting a Windows Bitmap File

The DIS2BMP.exe utility reads an ASAP DIS (display) file and creates a Windows BMP (bitmap) graphic file. It is stored in the ASAP installation folder, \bin. You can use this utility to create and export bitmap files for presentations or reports.

The resolution (pixels) are the same as the PIXELS (ASAP command) resolution from within ASAP. For example, if the user has specified a PIXEL of 200, the BMP should be 200 pixels wide.

DIS2BMP converts one or three DIStribution data files to a monochrome or color BMP file respectively: 

DIS2BMP disfile [ bmpfile ]
DIS2BMP redfile greenfile bluefile [bmpfile ]

RGB component files should be of the same PIXEL size. The DIStribution files may have been saved previously with a WRITE command.

Note: Run the Project example, "BMPSourceTrace" in your ASAP installation area in the folder, projects\samples
