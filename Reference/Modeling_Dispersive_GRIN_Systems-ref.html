GRIN Systems - Modeling Dispersive

The basic Gradient Index (GRIN) function in ASAP

does not have a functional variation in wavelength, only in position. The technique to modeling dispersive gradients is to define a separate GRIN function for each wavelength and to update the object data between individual ray traces. In effect you do an individual monochromatic analysis without a WAVELENGTHS command directly associated with the Media database. A WAVELENGTHS command may still be associated with individual sources.
