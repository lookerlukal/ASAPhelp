GRIN Overview

Gradient Index (GRIN) materials have indices of refraction that vary as a function of position within the material. Consequently, light does not necessarily follow a straight line while propagating through the material.

The more common types of GRINs are axial, radial, and spherical:


  • Axial GRINs have an index variation that is a function of material depth.
  • Radial GRINs have an index variation that is a function of radial position from the axis.
  • Spherical GRINs have an index variation that is a function of radial position within a spherical coordinate system.
