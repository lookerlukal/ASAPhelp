Defining an Apodization of Emitting Volumes

This topic lists the input arguments and return for defining an apodization of emitting volumes.

FUNCTION USERBOTH( P, D, C, ERRMSG )

Input arguments:


P(1:3) = (X,Y,Z) = random ray position coordinates
D(1:3) = (A,B,C) = random ray direction coordinates
C (1:15) = array of coefficients passed by ASAP from the USERAPOD commad
ERRMSG = error message string (80 characters)

Return:


  • ray flux multiplier
  • error message in ERRMSG, otherwise ERRMSG is left blank. If ERRMSG is returned non-blank, ASAP will stop running.
