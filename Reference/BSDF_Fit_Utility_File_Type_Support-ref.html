BSDF Fit Utility: File Type Support

This topic lists the files types supported by the BSDF Fit utility and the ASCII file format.

The file types supported by the BSDF Fit utility include:
  • ASTM (American Society for Testing and Materials), as outlined in the document, “Standard Practice for Angle Resolved Optical Scatter Measurements on Specular or Diffuse Surfaces”.
  • ASCII-formatted output, generated by some Schmitt Industries' BSDF measuring equipment.
  • Harvey Fit files (*.hs) that were originally saved under ASAP 6.6.
  • Any ASCII file created with the following format:

LineASCII file formatComments
Line 1:<Specular angle measured from normal in degrees. (Angle of Incidence)>
All other lines:<Angle from specular in degrees><BSDF value>The angle from specular must be positive for forward scatter and negative for backward scatter.

The following example of an input file for BSDF data was taken at an angle of incidence of 15 degrees:


15
-10.233
-5.345
0.463
5.323
10.264
Note:
  • Zero corresponds to the measurement taken at the specular angle and should be the maximum value for specular or partially specular materials.
  • In all cases, you can correctly import only in-plane data with a constant angle of incidence.
  • Once a file is imported, the data is displayed in the graphs entitled Measured Forward BSDF and Measured Backward BSDF. The Harvey or Polynomial model can now be fit to the measured data. Saving the file in the BSDF Fit utility (File > Save) creates a binary file with the extension *.fit, which contains all measured data, parameters, and any other settings (such as view size). Your original data files remain unchanged.
  • File extensions for BSDF data can vary. Typically, the wavelength of the laser that was used in the measurement becomes the extension (for example, #.633 for a wavelength of 633). If the file is renamed with the extension, *.atm, you do not need to set the "Files of type" in the Open File dialog box to "BSDF Fit Utility Files". When importing files from within the BSDF Fit utility, the utility ignores file extensions.
