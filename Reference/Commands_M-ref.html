Commands and Macros (M)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “M”.

  • MANGIN (ASAP Command)
    Creates a Mangin mirror.
  • MAP (ASAP Command)
    Produces a depth map of the currently defined object surfaces.
  • MATRIX (ASAP Command)
    Specifies the transformation matrix that operates on the surface general transformation matrix.
  • MEDIA (ASAP Command)
    Creates a single refractive material.
  • MEDIA ABSORB (ASAP Command)
    Creates an absorbing refractive material.
  • MEDIA CRYSTAL (ASAP Command)
    Defines birefringent media.
  • MEDIA GRIN (ASAP Command)
    Creates a GRIN refractive material.
  • MEDIA SCATTER (ASAP Command)
    Assigns to medium a GENERAL polynomial or USERFUNC function in the global coordinates X, Y, Z.
  • $MENU (ASAP Macro)
    Displays a menu of currently defined macros for selection.
  • MESH (ASAP Command)
    Creates a 3D representation of the distribution in the system vector (*.VCR) file.
  • MINIMIZE (ASAP Command)
    Minimizes the RMS spot size or the average of the previously specified fields, weighted by the w's.
  • MINMAX (ASAP Command Argument)
    Sets BSDF bounding.
  • MIRROR (ASAP Command)
    Creates a simple mirror.
  • MISSED (ASAP Command)
    Sets how a missed ray is plotted.
  • MODEL (ASAP Command Argument)
    Scattering from anisotropic surfaces (for example, brushed, diamond-turned) is not rotationally symmetric at normal incidence, and not necessarily symmetric about the plane of incidence otherwise.
  • MODELS (ASAP Command)
    Creates a scattering model.
  • MODIFY (ASAP Command)
    Modifies current distribution data.
  • MOVE (ASAP Command)
    Moves rays along their propagation direction to a new location.
  • MOVE PARABASALS (ASAP Command)
    Moves the parabasal rays for each ray/beam to its base ray plane.
  • MOVE TO FOCI (ASAP Command)
    Moves rays or beams along the propagation direction to a focus with the given parabasal rays and the base ray.
  • MOVE TO PLANE (ASAP Command)
    Moves rays along their propagation direction to a specified reference plane.
  • MOVE TO POINT (ASAP Command)
    Moves rays along their trajectories to their closest approach to the given point x y z.
  • MOVE TO SPHERE (ASAP Command)
    Moves rays along their propagation direction to a specified different sphere.
  • $MSGS (ASAP Macro)
    Switch on or off the display of warning and informative messages other than error messages.
  • MULTIPLE (ASAP Command)
    Converts the surface into multiple sheets.
