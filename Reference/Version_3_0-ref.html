Version 3.0
Version 3.0

May 1991

Recent GALOP enhancements: 
  Chromatic aberration correction
  Complete Schott catalog for glass selection
  Much improved hybrid optimization algorithm (SVD with PSD damping)
  New variables
    Stop position
    Edge (marginal) ray height (power redistribution)
    Tilted/decentered aspherics that maintain 1st order layout
    SINGLET and DOUBLET thicknesses
  Generation of 2D plot file in ANALYZE for later hardcopy
New RABET commands: 
  LENSES are a sequence of up to 10 arbitrary conicoids (one object)
    SEQUENCE entered directly
    MIRROR surface
    SINGLET lens
    MANGIN mirror
    DOUBLET cemented (achromat)
    RIGHT angle prism
    PENTA prism
    WEDGE of glass
    AFOCAL beam expander or reducer (mirrors and/or lenses)
      ALTER any lens parameter
    IDEAL lens module from ABCD matrix
  MODELS for surface scatter in extended version
    LAMBERTIAN diffuse
    HARVEY shift-invariant polished surface
    POLYNOMIAL with actual BSDF data FIT option
    VANES angled with rounded tips
    USERBSDF programmed by user
  EMITTING surfaces or volumes from a randomly created ray set
  TITLE and identifier for plots
  DISPLAY
    COMBINE two data sets
    REDUCE current data to smaller subset
    ... COLOR override option on all plot commands
    MESH write to 3D vector file
    ENCLOSED energy within a rectangle
    RANGE reset for minimum and maximum functional value
  LSQFIT variable normalization and singular value threshold
  HISTORY of ray trajectories from SAVE file
  SCATTER with MODEL assignment
    TOWARDS an important area bounded by an edge
  DOMACROS determines priority of user macro commands
  SEGMENTS sets number used to draw elliptical arcs
  SHOW current commands settings
Enhanced RABET Commands: 
  SEARCH LIST option added
  EDGES can now have smooth elliptical arc segments
    Dashed LINE option
    OVAL is shape factored polygonal edge
    RTRACK is racetrack
  SURFACES
    MULTIPLE linear displacements
  HORN FIT to point/derivative pairs option added
  BOUNDS checking with EDGES added
  SPREAD automatically sums multiple wavelengths incoherently
  FIELD sums all coherent contributions, just not current WAVELENGTH
  FIELD ... DELTA option allows varying "time"
  Relative indexing for ray and source numbers added to GET/PUT and SELECT
  GET totals and averages for all the current rays
  RAY,REPLOT graphical inputs
    "?" displays current cursor coordinate
    Equivalent commands written to USR file
  DISPLAY
    FFT any part not just modulus
    RADIAL averages and/or plots function and/or integral
    GRAPH has new options
    CONTOUR level values written on plot
  OPDMAP ... PSF option
  TRACE GRAPH option displays progress of ray storage
  USERAPOD input apodization both in position and directional space
  SOURCE LINE and WAVEFUNC for arbitrary input wavefronts
Code changes: 
  RABET default array sizes increased:
  Pixel sampling         128x128 to 256x256
  Objects                    160 to 200
  Surfaces+edges+lenses      480 to 600
  Media/coatings              40 to 50
  Interpolation wavelengths    5 to 9
  Simultaneous HOE orders      3 to 5
  Important areas per object   3 to 10
  SPREAD and FIELD from 2x to 20x faster
  Faster SELECT command avoids ray paging
  Glass MEDIA referred by name can be automatically created from SCHOTT.CAT
  USERFUNC,USERSURF coefficients passed in double precision
  Concave facets/polygons now allowed
  BSI is now a set of subcommands in STRAP
Macro Language: 
  SGN math function added
  $TIC internal macro for elapsed CPU time
  $ARG controls macro argument prompting
  $SCR displays and/or edits register fields
  $CASE allows upper, lower, or both cases for commands
  $ECHO controls/overrides echo of input
Macro libraries: 
  UTIL contains many useful macros for lenses, prisms, etc.
  ACCS2RBT permits reading or converting AccosV input files
  OSLO2RBT                  "            OSLO       "
2D graphics drivers: 
  TERMSP now supports UNIX X-Windows through XTERM emulator.
  HPGLNP can output separate plotter files
  PSCRIP creates encapsulated PostScript files, portrait or landscape
  EPSONP now handles 24-pin printers as well as 9-pin
Documentation: 
  New format and packaging
  TUTORIAL is now two volumes totalling almost 700 pages with an index
