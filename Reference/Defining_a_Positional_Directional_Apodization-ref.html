Defining a Positional or Directional Apodization

This topic lists the input arguments and return for defining an positional or directional apodization.

FUNCTION USERAPOD( X, Y, K, C, ERRMSG )

Input arguments: 

(X,Y) =	positional coordinates if K = 
           directional coordinates if K = 1

C(1:15) = array of coefficients passed by ASAP from the USERAPOD command

ERRMSG = error message string (80 characters)

Return:


  • input intensity as a function of coordinates
  • error message in ERRMSG, otherwise ERRMSG is left blank. If ERRMSG is returned non-blank, ASAP stops running.
