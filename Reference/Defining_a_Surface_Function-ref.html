Defining a Surface or Function

This topic lists the input arguments and return for defining an surface/function.

REAL FUNCTION USERFUNC( P, PFCN, CFCN, MXNC, RMAT, KWAN, GRAD, ERRMSG)

Input arguments:


P(1:3) = point in global coordinates
PFCN(1:3) = global coordinates of function reference point
CFCN(1:MXNC) = array of function coefficients passed by ASAP from the USERFUNC command
RMAT(1:3, 1:3) = rotation matrix for transforming from global to local coordinates
KWAN < 0 return
gradient only = 0 return function only; > 0 return both
ERRMSG = error message string (80 characters)

Return:


  • USERFUNC = value of function if KWAN > 0
  • USERFUNC = average surface curvature (1/radius) at point of KWAN = 0
  • GRAD(1:3) = gradient of function if KWAN != 0
  • error message in ERRMSG, otherwise ERRMSG remains blank. If ERRMSG is returned non-blank, ASAP stops running.
