Commands and Macros (E)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “E”.

  • $ECHO (ASAP Macro)
    Controls display of input commands.
  • EDGES/CURVES/ENTITIES (ASAP Command)
    Signals ASAP that edge definition commands follow.
  • $EDIT (ASAP Macro)
    Calls user-specified editor from within ASAP.
  • ELLIPSE (ASAP Command)
    Creates an elliptical edge/curve.
  • ELLIPSOID (ASAP Command)
    Creates an ellipsoidal surface.
  • EMITTING (ASAP Command)
    Creates a composite single-source emitter from several EMITTING volume commands.
  • EMITTING BOX/SPHEROID (ASAP Command)
    Creates a ray set uniformly distributed within a box or spheroid of an emitting volume.
  • EMITTING CONE/PYRAMID (ASAP Command)
    Creates a rayset uniformly distributed within a cone or pyramid.
  • EMITTING DATA (ASAP Command)
    Creates rays from data in given binary distribution file.
  • EMITTING DISK/RECTANGLE (ASAP Command)
    Creates random emitting surfaces.
  • EMITTING ENTITY or OBJECT (ASAP Command)
    Randomly distributes rays over the faceted surface(s) of SURFACE/EDGE entities or SURFACE/EDGE OBJECTS (entity number k or object k (number or name)).
  • EMITTING FILAMENT (ASAP Command)
    Creates a randomly generated ray set that is uniformly distributed along a predefined or arbitrary curve.
  • EMITTING HELIX (ASAP Command)
    Creates a volume emitter with randomly generated rays, uniformly distributed along a helical curve.
  • EMITTING IES (ASAP Command)
    Creates a ray set that emits according to Illuminating Engineering Society (IES) data.
  • EMITTING RAYS (ASAP Command)
    Creates an arbitrary set of rays.
  • ENCLOSED (ASAP Command)
    Calculates an encircled (ensquared) energy using a square array of points.
  • END (ASAP Command)
    Terminates program processing.
  • ENTITIES (ASAP Command)
    Defines mixed geometrical entities without using the EDGE, LENS, or SURFACE commands.
  • $ERR (ASAP Macro)
    Branches on an input error.
  • $EVAL (ASAP Macro)
    Evaluates a given $FCN function.
  • $EXIT (ASAP Macro)
    Starts immediate program termination.
  • $EXP (ASAP Macro)
    Switches the expression precedence.
  • EXPLICIT (ASAP Command)
    Converts the current function to explicit form.
  • EXPLODE (ASAP Command)
    Explodes lens conicoids into separate surface objects.
  • EXTEND (ASAP Command)
    Linearly extends one or both ends of a curve edge.
  • EXTREMES (ASAP Command)
    Lists the minimum and maximum information regarding position, direction, flux, or optical path length in currently selected ray data.
