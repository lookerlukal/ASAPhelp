Polarization Effects of a Sapphire Dome

A birefringent material can change the polarization apodization and therefore the irradiance across the beam. This change is sometimes referred to as a polarization aberration. In the example of this topic, see how the ordinary and extraordinary indices are defined.

PLOT POLAR plots the two orthogonal linear polarization components. Since SPLIT 1 is on, ASAP creates ordinary and extraordinary rays, and it is the combination of the two that leads to the actual field behavior. The FIELD...DELTA option is used to plot the polarization ellipses and electric field as a function of time for the combined effect of the ordinary and extraordinary rays.
Example of PRINT MEDIA, COATING Command Output 
--- UNITS MM
 --- WAVELENGTHS 10.2 UM
 --- MEDIA
 ---   4.03 'GERMANIUM'
 --- COATING PROPERTIES
 ---   0 1 'TRANS'
 --- PRINT MEDIA
 
 MEDIUM    Index/Absorb     FUNC:     exponent        steplength            maxnum
                  1.000000            0         1.00000     0.100000E+10        1000          VACUUM_AIR
    1            4.030000            0         1.00000     0.100000E+10        1000          GERMANIUM
 
 
 --- PRINT COATING
 
                                    10.20
 COATING Name           R  1  T

  1 TRANS               0.000  1.000

Key to Output


MEDIUM= Media data base location
Index/Absorb = Refractive index/absorption coefficient at the first wavelength
FUNC:= Data base reference to FUNCTION used as a GRIN function
exponent= Power to which GRIN function is raised
steplength= Current step length for differential GRIN ray trace
maxnum = Maximum number of steps for GRIN ray trace
COATING Name= Coating data base location and name
10.2 = Wavelength
R= Reflection coefficient
T = Transmission coefficient
