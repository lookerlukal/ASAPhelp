Converting BMP Bitmap Files to/from Distribution Data

Two utilities are available in ASAP for converting bitmap files.

BMP2DIS and DIS2BMP. BMP2DIS converts one color in a BMP bitmap image to a DIStribution data file and sets the full "width" (in system length units) of the distribution. 

BMP2DIS file [ Red ] [ width ]
               Green
               Blue
               Mono

The Mono version converts the three colors into a grey-scale, using the standard formula, .30*Red+.59*Green+.11*Blue.

DIS2BMP converts one or three DIStribution data files to a monochrome or color BMP bitmap image file, respectively:

DIS2BMP disfile [ bmpfile ]

DIS2BMP redfile greenfile bluefile [ bmpfile ]

Note: Run the Project example, "BMPSourceTrace" in your ASAP installation area in the folder under projects\samples.
  • Importing CCD Images
    The BMP2DIS.exe utility is for importing CCD images into ASAP, and is stored in your ASAP installation area under \bin. It reads a Windows BMP (bitmap) file and creates an ASAP DIS file. In the following syntax, the width option specifies system units length in ASAP.
  • Exporting a Windows Bitmap File
    The DIS2BMP.exe utility reads an ASAP DIS (display) file and creates a Windows BMP (bitmap) graphic file. It is stored in the ASAP installation folder, \bin. You can use this utility to create and export bitmap files for presentations or reports.
