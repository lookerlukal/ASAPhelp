LIST Command Overview

While a graphical representation is generally the only way to look at large amounts of data (such as a spot diagram), numerical data is frequently needed for requirements verification/compliance, and comparisons.

The LIST command is a low-level data access command for generating tables of various types of ray data, including:


  • Positional coordinates, current objects, ray fluxes, optical path lengths for base rays and selected parabasal rays
  • Directional coordinates, current objects, ray fluxes, optical path lengths for base rays and selected parabasal rays
  • Position, direction, flux, size, and divergence of each ray/beam in a format compatible with the input to the EMITTING RAYS command
  • Wavelength and ray numbers for each source
  • Current medium, split level, source number, number of objects, current object, and previous starting object for each ray
  • Polarization ray tracing information
