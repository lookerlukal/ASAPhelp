RENDER Dialog

This topic describes the options in the RENDER dialog on the System Menu.


Show PlotIf checked, plots the raytrace prior to the rendering.
Use Object Scatter ModelsAny reflective SCATTER MODELS assigned to objects are used in the rendering.
Override ColorOverrides the background color of the image. The default is black.
Optional Light Source
Relative IrradianceIlluminates the scene by a light source of unit irradiance emanating from this point (for example, a camera mounted flash bulb). Unless a second LIGHT source is specified, no shadowing is visible.
Direction A, B, CDirection of propagation of the collimated beams.
