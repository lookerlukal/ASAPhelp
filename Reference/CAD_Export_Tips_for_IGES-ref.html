CAD Export Tips for IGES

This topic gives you tips for exporting CAD files to IGES.


  • When bounding with an edge, the edge is swept an infinite distance along a direction perpendicular to the plane containing the edge. For edge bounding to work predictably, these two situations should exist:
    1. Edge should be closed - If a bounding edge is open, it will be closed with a line that passes through the start and end points of the edge.
    2. Edge should lie on a plane
  • For surface bounding to work predictably, the surface should be G1 continuous, that is, no sharp corners.
  • Trimming an object using the BOUNDS modifier requires that the bounding edge or surface, intersects the bounded surface. The resulting intersection curve must either be closed or form a closed curve when trimmed with existing boundaries on the bounded surface. Avoid creating closed intersection curves that intersect the outer boundary of a surface at a single point.
  • A surface used as a bounding entity is not extended to infinity. The surface is constrained to lie within its LOCAL box.
  • All ARRAY command qualifiers, except BOUNDS, are ignored.
  • A bounding region as defined by the LIMITS modifier must be rectangular; non-rectangular regions are ignored. An example of a non-rectangular region is when the LIMITS modifier is used with a coordinate direction. This situation defines a cylindrical bounding region, with constant elliptical cross-section in planes perpendicular to the specified coordinate direction. If a non-rectangular region is needed, BOUNDing to surfaces or edges should be used.
  • There will be no ‘round trip’, that is, the sequence below will not, in general, generate the original INR file defined in the first situation:
    1. Run an INR file
    2. Export the geometry in the INR file in the first situation (edge should be closed) to an IGES file.
    3. Import the IGES file in the second situation (edge should lie on a plane).
  • If units are not specified for IGES export, they are assumed to be MM.

  • The conversion of an ASAP bounded surface to an IGES trimmed surface is dependent on tolerances for 3D position and angle:


    3D Position ToleranceControls the accuracy of the intersection of an ASAP object with a bounding surface or edge. If it is too small, the intersection procedure may fail. If the tolerance is too large the intersection procedure may give an incorrect result.
    Angle ToleranceControls the intersection curve that results from the intersection of an ASAP object with a bounding surface or edge. A small angle may prohibit the construction of an intersection curve. A large angle could result in an incorrect intersection curve.
  • A surface REDEFINEd with the THICKNESS option cannot be bound with another surface or edge-based entity. Also, the LIMITS modifier has no effect.
  • A surface cannot be bounded with surfaces REDEFINEd with the THICKNESS option.
