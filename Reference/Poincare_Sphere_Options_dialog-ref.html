Poincare Sphere Options Dialog

This dialog is accessible from the 3D Viewer context menu in the Tree View pane. It is enabled when polarization data is present in the current DIS file, an includes options for displaying the Poincaré sphere. Its fields are described.

Property Type Field Property Description
Grid Settings Default=selected Enables display of grid lines to display the coordinates of the Poincaré sphere.
Thickness Default=1 Specifies thickness for the longitudinal and latitudinal grid lines.
Color button Default=light gray Opens the Color dialog to change color of the longitudinal and latitudinal grid lines.
Sphere Options:    
Transparency Default=selected Displays points located inside and at back of Poincaré sphere.
Display Equator Lines Default=selected Displays the equator line on the Poincaré sphere.
Display Poles Default=selected Displays the poles on the Poincaré sphere.
Display Meridian Default=selected Displays the meridian on the Poincaré sphere.
Display Parallel Lines Default=selected Displays the lattitudinal and longitudinal grid lines on the Poincaré sphere.
