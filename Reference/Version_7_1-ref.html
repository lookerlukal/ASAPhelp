Version 7.1
Version 7.1
Release Notes for ASAP 7.1 Kernel as of February 28, 2002
New Commands
 
MODELS
		RAWDATA version of BSDFDATA that does not interpolate in logarithmic space.
OBJECT
		RBOUNDS version of BOUNDS allows virtually unlimited entity ranges.
Enhanced Commands
 
MOVE
		BY/TO a specified optical path length.
		TO a best-fit SPHERical wavefront.
DECOMPOSE
		POSITION CONVERGE/DIVERGE accuracy improved by using non-planar beamlets.
		No longer creates "left-handed" beams that have anomalous pi/2 phase shift.
MODELS
		VOLUME uses (and MIE displays) absorption efficiency if non-zero.
SURFACES
		SOLID faster and now set automatically by TUBE, semi-ELLIPSOID, and CONDUIT.
		...HEXAGONAL aperture option in addition to RECTANGLE and ELLIPSE.
		SAMPLED distribution files are now fully buffered for a tremendous speedup.
		OPTICAL allows new parabolic term to be subtracted from base definition.
DISPLAY
		REDUCE now has a shorter form for picking out every nth pixel.
		GRAPH DISCRETE option drawsd vertical lines from axi to each data point. 
SELECT on number of times ray has HIT objects and/or AXIS angular range.
EMIT DATA
		Now allows control of which hemisphere from a 2D distribution.
		Can read directly new compressed DUMP format.
EDGE
		POINTS now more thoroughly checks consistency of input data.
		SPLINE 3D version optionally allows specification of CURVatures at points.
		LINE/ELLIP/RECT/OVAL/ROUND redefining default sudivision can be suppressed.
		CONIC now allows a parabola to be subtracted from base curve.
FIELD
		More accurate (and sometimes faster) version of the FD-BPM algorithm.
		Validity extended by using non-paraxial invariants for Gaussian beamlets.
		Restored applicability and improved speed of the "fast" method.
MEDIA SCATTER rays are now uniquely tagged for breakdown by PATHS.
SCATTER
		TOWARDS temporary limitation to 32767 scattered rays per incident removed.
OBJECT now issues warning for duplicate names.
HISTORY can process only every nth ray instead of all rays.
CAD IGES 
		Now exports ARRAYs of surfaces in the form of 308/408 Subfigures.
		Displays residual angular errors for SURFACE entity spline approximations.
LENS
		ABERRATIONS
	   	STORE can now write ZEMAX and OSLO lens files (in addition to CodeV).
VUFACETS defaults tessallation of unBOUNDed planar EDGE objects to 3D viewer.
Macro Language

Macro argument "#" substitution no longer occurs beyond "!!" comment start.

Macro, function, and variable name lengths increased from 16 to 32 characters.

$MOVE same as $COPY except the source file is deleted afterwards.

$EVAL can now take a $FCN that returns a complex (real,imaginary) result.

Other Changes
 
*.SYS (as usual) and *.REG files are incompatible with previous versions.
IGES import
		Added option to compress (COMPOSITE) multiple trimming curves into one.
		ALL geometry option now includes unbounded Faces (510) of solid Shell (514)
		Unnested Subfigures (308/408) are now processed.
Total geometry storage doubled in both Basic and Pro versions.
Most coherent SPREAD/FIELD Gaussian beamlet calculations 3 to 5 times faster.
Maximum allowed FFT size set to 2^16 (65536) in both Basic and Pro versions.
