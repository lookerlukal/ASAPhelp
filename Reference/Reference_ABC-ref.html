Reference (A-C)

Reference topics in this section begin with the letters "A" to "C".

  • Abel Transform Overview
    The Abel transform and its inverse are used in image reconstruction problems to recover a 3D distribution from its projections; that is, the tomography problem.
  • Anisotropic Polarizing Elements Overview
    Retardance (sometimes called birefringence) is the property of an interface, where the Optical Path Length (OPL) or phase change between the orthonormal components, which mathematically describe the polarization state, is a function of the incident polarization state. Many polarizing optical elements, including waveplates and some polarizers, work on this basis.
  • Assigning Specular Properties to an Object
    In conventional optical codes, an optical element is either reflecting or refracting. In ASAP, an object may be both simultaneously.
  • Axial GRIN Singlet
    The example of Axial Gradient Index (GRIN) shows an axial GRIN singlet lens. This lens was actually designed in ASAP but only its GRIN properties are examined. The gradient function is simply a linear function of z coordinate in the singlet.
  • Babinet Principle
    Babinetâ€™s principle is for modeling difficult diffraction problems. Set the amplitudes of certain beams to be of opposite sign from other beams, so beam amplitudes are in effect subtracted from one another.
  • Batch Mode
    If you are an experienced ASAP user, you may prefer to use ASAP in batch mode. You may also find it more efficient to perform long, computationally intensive runs in batch mode. Alternatively, you can develop an input file (*.inr or *.enx) using the Builder, and then perform the full run in batch mode. From the File menu in the Builder, you can convert the *.enx file to an *.inr file by selecting Save As and then ASAP INR Files .
  • Best Focus Position
    The ASAP best focus position is the point in 3D space where the root mean square (RMS) distance to all the considered and selected rays is a minimum.
  • Bounding and Slicing of Objects
  • Calculating Geometrical MTFs
    In optical systems where the residual aberrations are large, the diffraction MTF can be difficult to compute. Consequently, an analogous measure of performance, the MTF is defined. Whereas, the diffraction Modulation Transfer Function (MTF) is the Fourier transform of the Point Spread Function (PSF), the geometrical MTF is a Fourier transform of the spot diagram.
  • Calculating Modulation Transfer Functions
    The Modulation Transfer Function (MTF) characterizes the contrast performance of an optical system as a function of spatial frequencies. It is the modulus of the Optical Transfer Function (OTF).
  • Calculating Phase Transfer Functions
    While Modulation Transfer Function (MTF) calculations are by far the more common of the two, there are instances when the Phase Transfer function (PTF) is also needed to completely characterize the imaging properties of an optical system.
  • Cartesian Coordinate System in ASAP
    ASAP uses a single, global, right-handed, Cartesian coordinate system with components x, y, and z. "Edges", "lenses", and "surfaces" are the basic geometrical entities of the program.
  • Coordinate Transformations in ASAP
    The coordinate transformations used in ASAP are applicable both to explicit transformations (for example, SHIFT, ROTATE) and implicit transformations (for example, AXIS LOCAL in combination with FIELD or CVF).
  • Circular Diffraction Grating
    A circular diffraction grating has equally spaced grating rings that extend radially outward.
  • Coating Storage
    COATINGS are stored in a complex array. As COATINGS are defined, they fill the array at the next available location.
  • Combinations of INTERFACE and FRESNEL Settings
    Combinations of the INTERFACE and FRESNEL command settings are listed in this topic.
  • Creating Rays to Simulate Random Emitter
    The EMITTING commands simultaneously set the ray positions and directions in a random manner to simulate natural emitters; for example, blackbody radiators.
