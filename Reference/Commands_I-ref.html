Commands and Macros (I)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “I”.

  • IDEAL (ASAP Command)
    Creates an idealized optical element (ABCD and Jones matrices).
  • IESFILE (ASAP Command)
    Writes out Illuminating Engineering Society (IES)-format file.
  • $IF (ASAP Macro)
    Creates conditional processing of a block of records or block structures.
  • $IF THEN (ASAP Macro)
    Use the THEN keyword at the end of a $IF statement to trigger processing of conditional block structures.
  • IMAGE Curve/Edge (ASAP Command)
    Images a curve/edge through a specified LENS entity.
  • IMAGE (Global Point) (ASAP Command)
    Images a global point through a specified LENS entity.
  • IMAGE (Ray Positions) (ASAP Command)
    A ray modifier that finds the conjugates of ray coordinates through a LENS (images the current ray positions through a specified LENS entity).
  • IMMERSE (ASAP Command)
    Creates rays in MEDIA in other than zero.
  • IMPORT (ASAP Command)
    Import a lens from a ZEMAX file.
  • INTERFACE (ASAP Command)
    Assigns a reflective, refractive or diffractive interface to objects.
  • INTERFACE POLARIZATION (Polarization Modifiers) (ASAP Command)
    Assigns a polarization modification interface to objects.
  • INTERFACE REPEAT (ASAP Command)
    Assigns the interface characteristics of either a specified object or the previous object to the current object.
  • INVERT (ASAP Command)
    Reverses the parametric direction of a curve.
  • $IO (ASAP Macro)
    Controls I/O redirection.
  • IRRADIANCE (ASAP Command)
    Selects the axis that energy or peak flux density is computed relative to a given fixed direction for all sample points.
  • ISOMETRIC (ASAP Command)
    Creates an isometric view of the current distribution data file.
  • $ITER (ASAP Macro)
    Iterates input records and a set of variables.
