Display Menu Dialogs

Topics in this section of the Contents tab include dialogs accessible from the Display menu.

  • CONTOUR Plot Dialog
    Creates a contour plot of the current distribution file.
  • DIRECTIONAL Dialog
    Produces a polar plot of the current angular distribution data file. Optionally, the plot can be a Cartesian plot.
  • ENCLOSED Dialog
    Calculates an encircled (ensquared) energy, using a square array of points. Plots the percent enclosed as a function of the size of the square.
  • GRAPH Dialog
    Creates 1D profile(s) of the current distribution data file.
  • ISOMETRIC Dialog
    Creates an isometric view of the current distribution data file.
  • PLOT 3D Dialog
    Creates a 3D representation of the current distribution data file with 1D cross-sections.
  • RADIAL Dialog
    Rotationally averages the current distribution data, and optionally replaces the current distribution data.
  • MESH Dialog
    Writes a 3D representation of the distribution into the 3D system vector file so that it can be replotted with other geometrical entities.
  • HISTOGRAM Dialog
    Displays a plot of the number of data values.
  • AVERAGE Dialog
    Smoothes current distribution data by averaging adjacent pixels. Averaging is equivalent to convolving the distribution with a finite window.
  • COMBINE Dialog
    Combines the current distribution data file with a previously calculated distribution data file. Used for analysis of general wavefronts.
  • FFT Dialog
    Calculates the spatial frequency spectrum of the current distribution data.
  • FOLD Dialog
    Averages the current distribution data about the given pixel number of the specified coordinates.
  • FORM Dialog
    Controls the form of the current distribution data to be plotted.
  • MODIFY Data Dialog
    Alters the current distribution data.
  • NORMALIZE Dialog
    Re-normalizes the current distribution data.
  • OFFSET Dialog
    Shifts the origin of the current distribution data.
  • RANGE Dialog
    Adjusts the vertical plotting scale of current distribution data. Affects only the GRAPH, PLOT3D, and ISOMETRIC plots.
  • REDUCE Dialog
    Selects a subset of the current distribution data and rejects the remaining data.
  • SECTION Dialog
    Prints or transfers the current distribution data to/from variables.
  • TABLE Dialog
    Creates a table of the current data. Optionally overlays values onto a contour plot.
  • TRANSPOSE Dialog
    Rotates the current distribution data by 90 degrees. Optionally, creates a different view of the function when used before calling PLOT3D.
  • VALUES Dialog
    Lists the functional values of the current distribution data file, corresponding to the exact pair of specified coordinates.
