Assigning Specular Properties to an Object

In conventional optical codes, an optical element is either reflecting or refracting. In ASAP, an object may be both simultaneously.

Since ASAP calculates the power in each ray/beam, the reflection and transmission coefficients must be given at each interface in addition to the interface media.

The INTERFACE command is used to assign specular optical properties (transmission, reflection, and diffraction) to objects. INTERFACE assigns specular optical properties (transmission, reflection, and diffraction) to objects.

INTERFACE draws upon previously defined databases:


  • MEDIA for refractive index data
  • COATING for multilayer stack prescription/properties
  • SURFACES/FUNCTIONS for gradient index functions or for diffraction grating/HOE spacing
