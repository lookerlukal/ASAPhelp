Late-breaking information about this release of ASAP.

SQAWCS worked:

KERNEL

06PRD00001348 - $STAMP

This new macro causes ASAP to print a timestamp based on the current system time to the console. The components of the timestamp are also assigned to named registers for convenience, as illustrated by the following example: 

$STAMP
2007-01-31 17:32:19
&REG _YEAR _MONTH _DAY _HOUR _MINUTE _SECOND
_YEAR=2007
_MONTH=1
_DAY=31
_HOUR=17
_MINUTE=32

06PRD00001372 - PXP Active Set File Open Error

Fixed a rare failure in which HISTORY PX failed to create an ACTIVE pathset file.

GUI

06PRD00000024, 06PRD00001266 - Output Window Scroll

ASAP now handles proper scrolling of lengthy text fields within the ASAP Command Output Window

06PRD00000082 - Builder 'Bend Factor'

ASAP Builder default value for 'Bend Factor' has been changed from -20 to -1.
