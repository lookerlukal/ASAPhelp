Macro Formatting - Quick Reference

Macro format for both library (filename.LIB) and on-the-fly macro definitions: use the $IO LIBRARY filename to associate that macro library with an ASAP session.

n is the number of prompts following the macro : prompt 1 corresponds to macro argument #1, macro prompt 2 corresponds to macro argument #2.

$macroname echoes contents of macro.

&macroname does not echo contents of macro. 

macroname { n	
:
:
}
n prompts

SYSRAY { 1
RAYS 0; EMITTING DISK Z 0 2@1 #1
}
ENTER NUMBER OF RAYS>
