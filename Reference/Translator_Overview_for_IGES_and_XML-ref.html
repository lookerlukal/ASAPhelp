Translator Overview for IGES and GTX

ASAP CAD+ includes IGES and GTX (XML) import capability, by translating IGES (Initial Graphics Exchange Specification) files and XML (eXtensible Markup Language) in GTX format to ASAP format. You can perform optical analyses on geometries initially created in a CAD program.

In the case of IGES, the translator convert top-level parametric surface definitions (entities 100 through 144) into ASAP CURVE-based objects, and writes them to I2A and V2A files of the same name. You can rename the I2A file to an INR file for direct reading by ASAP.

These are the general steps in the translation process:


  • In the CAD program, export the system geometry in IGES or GTX file format.
  • Translate the data from IGES or GTX to ASAP format: select Open on the ASAP File menu and then select *.igs or *.gtx as the file type to translate.
  • Assign optical properties to the ASAP OBJECTS created from the IGES or GTX file.
  • Export IGES files from the File menu or by right-clicking the Objects tab on the ASAP Workspace window.
