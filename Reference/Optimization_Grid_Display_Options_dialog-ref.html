Optimization Results - Grid Display Options Dialog

The Grid Display Options dialog is displayed when you right-click the table in the Optimization Results window.

Property Type Field Property Description
Display all Trial Solutions User-specified Displays the total number of trial solutions specified on the Exit Criteria tab of the Optimization Setup Summary window. This option is selected as the default.
Display every _ Trial Solutions User-specified If the check box is selected, only a sub-set of the total number of trial solutions is displayed. Default is 5 trial solutions.
Do not display Trial Solutions User-specified If the check box is selected, displaying the tabular output for the graph is by-passed.
