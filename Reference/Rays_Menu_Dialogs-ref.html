Rays Menu Dialogs

Topics in this section of the Contents tab include dialogs accessible from the Rays menu.

Select one of the commands on this menu that is followed by an ellipse (...) .

  • BRO Light Source Library Wizard - Welcome Dialog
    Describes the fields on the Light Source Library Wizard dialog.
  • BRO Light Source Server Dialog
    Describes the fields on the Light Source Server dialog.
  • Consider Objects/Rays Dialog
    Select the objects to which the CONSIDER command should be applied.
  • SELECT Rays Dialog
    Designates current set of rays that ASAP is to consider in calculations and output. This dialog box is accessible when you right-click in the Display Viewer window.
  • Discard Rays Dialog
    ASAP can discard inactive rays when several rays are traced and only a small subset is analyzed extensively.
  • MOVE Rays Dialog
    The MOVE Rays dialog, which is accessible from the Rays menu, moves rays along their propagation direction to a new location. This topic describes the fields in the dialog.
  • REVERSE Rays Dialog
    Reverses the propagation direction of all currently defined rays.
  • View Positions 3D Dialog
    Designate source or object color.
  • View Rays 3D Dialog
    Plots a line from the current position of each ray along its direction vector.
  • List Ray Data Dialog
    The List Ray Data dialog is accessible from the Rays> Ray Information menu. The dialog generates a list of ray data.
  • Show Ray Details Dialog
    Retrieves (GETs) ray data and copies it into variables.
  • Ray Data Extremes Dialog
    Lists the minimum and maximum information regarding position, direction, flux, or optical path length in the currently selected ray data.
