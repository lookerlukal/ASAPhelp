Version 3.3

ASAP 3.3 release, November 1993

Approximately two dozen minor bugs have been fixed

Macros files now supplied for components from the following manufacturers:

Edmund, Newport, Melles-Griot, and Spindler-Hoya

Maximum number of geometrical entities increased from 600 to 900

Compatible with new 9.3 APART release for DOS and Unix systems

Access to user-programmable Fortran routines available again on some platforms

AutoCAD translator significantly improved

PC version: 

  Now uses Pharlap 5.1 extender which eliminates slow load under DOS 6.0
  ASAP.BAT temporarily remaps arrow and escape keys for use during $SCR input

Bulletin board system now in place for technical support (bug reports, patches)
