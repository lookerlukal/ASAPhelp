Defining a BSDF Model

This topic illustrates how to define a BSDF model in ASAP.

FUNCTION USERBSDF( A2B2, BB0, B02, C, N, ERRMSG )

Input arguments:

A2B2 = A^2+B^2 where (A,B) are the direction cosines of the scatter direction

BB0 = B * B0 where (0, B0) is the direction vector of the specular direction

B02 = B0^2

C(1:N) = array of coefficients passed by ASAP from the USERBSDF command

ERRMSG = error message string (80 characters)

Return:


  • isotropic BSDF
  • error message in ERRMSG, otherwise ERRMSG is left blank. If ERRMSG is returned non-blank, ASAP stops running.
