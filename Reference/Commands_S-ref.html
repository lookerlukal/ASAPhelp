Commands and Macros (S)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “S”.

  • SAG (ASAP Command)
    Deforms or sags an edge onto a surface.
  • SAMPLED (ASAP Command)
    Creates an explicit surface interpolated from sampled data.
  • SAMPLED (Cylindrical) (ASAP Command)
    Creates a deformed cylindrical surface interpolated from sampled data.
  • SAVE (ASAP Command)
    Writes future ray trace data to a file for post-processing.
  • SAWTOOTH (ASAP Command)
    Creates a sawtooth pattern edge in the plane.
  • SCALE (ASAP Command)
    Specifies arbitrary scaling of an entity.
  • SCALE FROM (ASAP Command)
    Scales from the given length units to the current system UNITS.
  • SCATTER (ASAP Command Argument)
    This command argument for inhomogeneous Monte-Carlo scattering.
  • SCATTER RANDOM/MODEL (ASAP Command)
    Assigns a scattering interface to an object.
  • SCATTER REPEAT (ASAP Command)
    Assigns the scatter characteristics of a given object to the current object.
  • SCATTER RMS/BSDF (ASAP Command)
    Assigns a near-specular scattering interface to an object.
  • $SCR (ASAP Macro)
    Defines a user-programmable screen template.
  • SEARCH (ASAP Command)
    Sets the local and global object intersection strategy.
  • $SEARCH (ASAP Macro)
    Provides user control of which paths are searched, and also the order in which paths are searched. These paths are defined and associated with symbolic names. $SEARCH specifies up to 20 paths to be searched for screen (*.scr), macro (*.mac), and macro library (*.lib) files.
  • SECTION (ASAP Command)
    Prints or transfers the current distribution data to/from variables.
  • SEED (ASAP Command)
    Initializes the seed for the random number generator.
  • SEGMENTS (ASAP Command)
    Controls number of segments plotted per arc.
  • SELECT (ASAP Command)
    Isolates a ray set for further analysis.
  • SEQUENCE (ASAP Command)
    Creates a lens composed of a sequence of conicoid surfaces.
  • SHAPE (ASAP Command)
    Sets the beam shape of the individual rays of the currently selected rayset.
  • SHIFT (ASAP Command)
    Specifies a relative shift of an entity along the coordinate axes.
  • SHOW (ASAP Command)
    Displays current settings.
  • $SHOW (ASAP Macro)
    Displays the status and states of several internally defined macros.
  • SINGLET (ASAP Command)
    Creates a simple singlet lens.
  • SKEW (ASAP Command)
    Specifies an arbitrary skewing of an entity.
  • SMOOTH (ASAP Command)
    Quadratically (or cubically) smoothes the current curve.
  • SOLID (ASAP Command)
    Makes the previous surface a solid.
  • SOURCE (ASAP Command)
    Specifies sources for the rays/beams.
  • SOURCE WAVEFUNC (ASAP Command)
    Specifies the ray propagation direction using a wavefront function.
  • SPECTRUM (ASAP Command)
    Simplifies flux weighting of rays as a function of wavelength - for spectral apodization.
  • SPLINE, EXPLICIT 2D (ASAP Command)
    Explicit 2D curvature (G2) continuous, piecewise cubic spline curve in the given plane (default 0).
  • SPLINE, GENERAL 3D (ASAP Command)
    General 3D piecewise, cubic, spline curve with given reference point (default origin), and defined in terms of subsequent relative points and tangent directions (and optional inverse radii of CURVatures).
  • SPLIT (ASAP Command)
    Controls the number of times a ray may be split into specular components.
  • SPOTS (ASAP Command)
    Creates a geometric spot diagram for the currently selected ray data.
  • SPREAD (ASAP Command)
    Calculates the exact coherent/incoherent energy distribution.
  • STATS (ASAP Command)
    Lists statistics of the currently selected ray data.
  • STATS TRACE (ASAP Command)
    Prints statistics of a previous TRACE.
  • $STO/$RCL/&STO (ASAP Macro)
    Stores or recalls variable data to or from a file.
  • STORE (ASAP Command)
    Stores the current state of the lens in the ASAP internal database, or optionally in the following file types: ASAP input , macro, CODE V SEQuence, Zemax, or OSLO.
  • SUBSET (ASAP Command)
    Selects a subset of the current ray set and rejects remaining ray data.
  • SUM (ASAP Command)
    Creates a composite scatter model.
  • SUPERCONIC (ASAP Command)
    Superconic asphere used in optical design.
  • SURFACES/FUNCTIONS/ENTITIES (ASAP Command)
    Signals ASAP that surface definition commands follow.
  • SWEEP (ASAP Command)
    Specifies the optional sweeping of a curve into a surface.
  • $SYS (ASAP Macro)
    Runs a system command or opens a Windows command window.
  • SYSTEM (ASAP Command)
    Stores and retrieves system information.
