ASAP Prompts

Describes ASAP prompts (ASAP>) that precede the Command Input window, and lists the prompt variations.

Top-Level Prompt

The top level prompt in the program is ASAP>. If you enter a key word in the Command Input window, the prompt changes to that key word prompt. Typing RETURN and clicking the Run button on the toolbar restores the ASAP prompt. You can enter commands for constructing system geometry and source definitions at the prompt. Alternatively, you can create ASCII files with a system editor, and store system geometry and sources you create in separate files.
