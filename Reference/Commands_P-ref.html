Commands and Macros (P)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “P”.

  • $PAGE (ASAP Macro)
    Pauses text output.
  • PARABASAL (ASAP Command)
    Sets the number of parabasal rays.
  • PARAMETERIZE (ASAP Command)
    Sets the parameterization axis for meshing the current surface.
  • PARTICLES (ASAP Command)
    Creates a particulate scatter model.
  • PATCHES (ASAP Command)
    Points represent Bezier surface patches.
  • $PATH (ASAP Macro)
    Sets the default directory "path" for future file openings.
  • PATHS (ASAP Command)
    Produces a table of summarizing propagation paths of currently defined rays.
  • PERFECT (ASAP Command)
    A "perfect" (but realistic) lens.
  • PHYSICAL (ASAP Command)
    A comprehensive, physical reflection model that is "good" even for rough surfaces at grazing incidence.
  • PICTURE (ASAP Command)
    Produces a picture of the current distribution data.
  • PIXELS (ASAP Command Argument)
    Sets the number of pixels for the current plot only.
  • PIXELS (ASAP Command)
    Sets the resolution for graphics and calculations.
  • PLACE (CURve/edge) (ASAP Command)
    Places the entity at the global position that corresponds to parametric coordinate u of CURve/edge k
  • PLACE (Global Coordinate) (ASAP Command)
    Specifies the absolute placement of an entity in global coordinates.
  • PLACE (Relative to Entity) (ASAP Command)
    Places the current entity relative to previously defined entities.
  • PLANE (ASAP Command)
    Creates a plane normal to the axis of symmetry, at the value given by the third entry.
  • PLANE NORMAL (ASAP Command)
    Creates a plane specified by its surface normal and a point.
  • PLANE POINTS (ASAP Command)
    Creates a plane specified by three points.
  • $PLAY (ASAP Macro)
    Replays current output buffer.
  • Plotting Commands
    The PLOT command is a general command for plotting virtually any aspect of system geometry and selected ray data.
  • PLOT3D (ASAP Command)
    Creates a 3D representation of the current distribution data file with 1D cross-sections.
  • PLOT BEAMS (ASAP Command)
    Plots extent of each Gaussian beam within the currently specified graphics window.
  • PLOT CURVES (ASAP Command)
    Plots a wire-frame outline of curves, with the control points displayed, in the currently specified graphics window.
  • PLOT EDGES (ASAP Command)
    Plots a wireframe outline of edges in the currently specified graphics window.
  • PLOT ENTITIES (ASAP Command)
    Plots surface, edge, and lens entities in the currently specified graphics window.
  • PLOT FACETS (ASAP Command)
    Plots parametric mesh representation of objects in the currently specified graphics window.
  • PLOT LENSES (ASAP Command)
    Plots lenses in the currently specified graphics window.
  • PLOT LIMITS (ASAP Command)
    Plots a wireframe box denoting the extent of the specified objects limits box in the currently specified graphics window.
  • PLOT LOCAL (ASAP Command)
    Plots local boxes of surface entities.
  • PLOT MESHES (ASAP Command)
    Plots all currently specified objects using a wireframe mesh.
  • PLOT POLARIZATION (ASAP Command)
    Plots polarization state ellipses.
  • PLOT RAYS (ASAP Command)
    Plots vectors along the direction of each ray contained within the currently specified graphics window.
  • PLOT SURFACES (ASAP Command)
    Plots surfaces in the currently specified graphics window.
  • PLOT (ASAP Command Argument)
    Plots current model.
  • $PLOT (ASAP Macro)
    Used to control screen graphics and plots.
  • POINTS (2D) (ASAP Command)
    Creates a generalized planar edge.
  • POINTS (3D) (ASAP Command)
    Creates a generalized edge.
  • POLARIZ (ASAP Command)
    Sets the polarization properties for future ray creation.
  • POLARIZ K (Reference Ray Direction) (ASAP Command)
    Sets the direction of the reference ray.
  • POLARIZ (Polarization Vector) (ASAP Command)
    Sets the polarization vector in a ray local polarization coordinate system.
  • POLARIZ RANDOM (ASAP Command)
    Randomize the polarization state vector of the rays.
  • POLARIZ TREF (Transverse Reference Direction) (ASAP Command)
    Sets polarization reference coordinate system.
  • POLYNOMIAL/TRINOMIAL/BINOMIAL (ASAP Command)
    Creates a scatter model based on user-supplied data or polynomial coefficients.
  • PostScript File Utility (PSCRIP) (ASAP Command)
    The PSCRIP.exe utility reads ASAP PLR (plot) files and creates PostScript files for printing. The utility is stored in your ASAP installation area under the \bin folder.
  • PRINT (ASAP Command)
    Prints out specific database information.
  • PRINT (Polarization Models) (ASAP Command)
    Print defined surface polarization modifiers.
  • PROFILES (ASAP Command)
    Plots slices through objects contained within the specified graphics window.
  • PUT (ASAP Command)
    Copies variable data into the current ray data.
