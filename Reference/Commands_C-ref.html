Commands and Macros (C)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “C”.

  • CADEXPORT (ASAP Command)
    Writes out CAD-compatible system geometry file.
  • CARTOVAL (ASAP Command)
    Creates a Cartesian oval surface.
  • $CASE (ASAP Macro)
    Specifies case sensitivity.
  • CHARACTER (ASAP Command)
    Creates an edge of a specified string.
  • $CLEAR (ASAP Macro)
    Clears the output buffer so that the scope of future $GRAB commands is limited to only the output that follows.
  • CLIP (ASAP Command Argument)
    Clips the distribution data according to an object or edge.
  • CLIP (ASAP Command)
    Specifies positional or directional ray limiting or clipping boxes.
  • COARSEN (ASAP Command)
    Converts the current edge to a piecewise linear curve.
  • COATINGS LAYERS (ASAP Command)
    Creates a coating specified by its multilayer prescription.
  • COATINGS MODELS (ASAP Command)
    Specify angular properties for reflection and transmission.
  • COATINGS PROPERTIES (ASAP Command)
    Creates a coating specified by its transmissive and reflective properties.
  • COLLECTION (ASAP Command)
    Calculates the collection PERCENT efficiency or FLUX of the current rayset.
  • COLORS (ASAP Command Argument)
    Sets color for current graphics commands.
  • COLORS (ASAP Command)
    Reassigns the color used to display each object.
  • COMBINE (ASAP Command)
    Combines the current distribution data file with a previously calculated distribution data file.
  • COMPOSITE (Edge Modifier) (ASAP Command)
    Combines several edge entities into a single edge entity.
  • COMPOSITE (Lens Modifier) (ASAP Command)
    Combines several lens entities into a single lens entity.
  • CONDUIT (ASAP Command)
    Sweeps a circle along a planar explicit cubic.
  • CONIC (ASAP Command)
    Creates a quadratic Bezier edge/curve in the plane given at the 2nd and 3rd entries.
  • CONSIDER (ASAP Command)
    Selects a set of objects for ray tracing and plotting.
  • CONTOUR (ASAP Command)
    Creates a contour plot of the current distribution data file.
  • $COPY (ASAP Macro)
    Performs a raw, byte-for-byte copy of the source file to a destination file.
  • CORNER (ASAP Command)
    Creates an axis-aligned corner of a cube (three mutually perpendicular planes), with the apex at the location given by the third entry.
  • CPE (ASAP Command)
    CPE (Cascaded Polarization Element) is used to define a compound polarization device. Every polarization device cascaded in a CPE element is specified by its polarization device type or type index and its device name or device list index in its defined list.
  • CRYSTAL (ASAP Command Argument)
    Identifies the medium as a birefringent uniaxial crystal.
  • CUTOFF (ASAP Command)
    Sets the conditions controlling ray termination.
  • CVF (ASAP Command)
    The CVF (Complex Vector Field) command provides a bidirectional translation facility between ASAP and FDTD Solutions™ by Lumerical.
