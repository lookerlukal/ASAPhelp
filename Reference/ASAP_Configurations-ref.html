ASAP Versions

BRO offers several ASAP versions and toolkits to match your needs, which are listed in this topic, and described in detail on the BRO Web site. This topic also provides tips on memory-resident ray tracing, array dimensions, total geometry storage, and maximum Fast Fourier Transform size for ASAP versions.


  • ASAP lite
  • ASAP CAD+
  • ASAP OPTICAL+
  • ASAP ADVANCED
  • BIO Toolkit™ (for ASAP)
  • CATIA Module (for ASAP)
  • ELTM Module (for ASAP)
Note:

BRO supports ASAP only on these Windows operating systems: Windows 2000 and XP.

The Registration dialog box, accessible from the ASAP Help menu, indicates your ASAP configurations, Customer Identification (ID) number, software and hardware keys, and other information.

Memory-resident Ray Tracing

When sufficient RAM is available on your computer, ray traces can be sped up significantly by making the ray files memory resident. This is most effective on systems with one gigabyte of memory, or greater. For ASAP OPTICAL+ and ASAP+ versions, see the topic, XMEMORY (ASAP Command).

  • Software Updates
    Breault periodically provides product releases to customers with current Maintenance Agreements. We notify all ASAP customers by email when a release is available for download.
  • ASAP Demonstration Program
    BRO has packaged a number of complete sample projects to give you an idea of the many capabilities in ASAP (Advanced System Analysis Program). You can run any sample project as many times as you want for as long as you want. If you are currently using the ASAP Demonstration Program, and not the full ASAP version, this topic provides a few tips.
  • Sample Project Files
    Sample Projects are complete working examples of optical systems built in ASAP, along with brief documentation in the form of text files. Sample project files can be opened in all licensed and demonstration versions of ASAP.
  • ReflectorCAD Segmented Reflector Design Software
    ReflectorCAD from BRO is a time-saving program for designing your segmented reflectors. ReflectorCAD provides a graphical means of laying out the segments of a reflector, convenient methods for adjusting segment aim and checking edge shadowing, and the ability to quickly calculate the reflector's approximate output.
