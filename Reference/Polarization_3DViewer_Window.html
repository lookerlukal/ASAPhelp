Polarization - Poincare Sphere Visualization Tool in 3D Viewer

The Poincaré Sphere Visualization Tool (PSVT) displays the polarization state of rays in the 3D Viewer. The PSVT includes three components: graphics pane to view the Poincaré sphere, flux legend, and wavelength legend in the left pane, and the toolbar at the top. This topic describes each of these areas.

Displaying Polarization Data in the 3D Viewer Window

When you open a display file with polarization data, the 3D Viewer window changes. A Poincaré sphere is displayed in the graphics window, the left pane includes polarization legends for wavelength markers and flux, and the toolbar expands to enable polarization-specific buttons and slider.

PSVT Graphics Window - Poincaré Sphere

PSVT Left Pane and Toolbar

The left pane of the PSVT contains the wavelength legend and flux legend.


FeatureDescription
Wavelength data marker displays a legend of markers for varying wavelength values. Select the initial default marker in the 3D Viewer File Preferences dialog. Remaining markers are assigned by ASAP. Select marker size in the 3D Viewer File Preferences dialog.
Wavelength legend displays colors of flux values. An active Poincaré sphere must be displayed and you must have chosen to render flux data.
Select either Linear (default) or Logarithmic. Default min/max values for the mapping range are the smallest and largest flux values found in the ray polarization data. Click Apply to initiate rendering flux information.
Turns on or off the rendering of flux data. If no active Poincaré sphere is present, the button is disabled.
Opens the dialog for Poincaré Sphere Options. If no active Poincaré sphere is present, the button is disabled.
Degree of polarization (DOP) slider specifies a value for viewing the DOPs of the rays within the current ray set displayed in the graphics pane. Range of values is 0 (completely unpolarized light) to 1 (completely polarized light). Default is set to 1. If no active Poincaré sphere is present, the slider is disabled.
PSVT Context Menu

The PSVT context menu is accessible by right-clicking the left pane of the 3D Viewer window. The lower section includes the polarization commands, which are available only when a Poincaré sphere is actively displayed. These commands are described below in the table.




Menu CommandDescription
Poincare Sphere OptionsOpens Poincaré Sphere Options dialog.
Display Polarization InformationDisplays the S1, S2, and S3 polarization state data, polarization reference vector, ray flux, and ray direction for a selected point in a pop-up box to the right of the selected point. The menu is disabled if no point is selected on the Poincaré sphere. If multiple points are selected, data is displayed in the Command Output window.
Save Polarization InformationSaves ray data for multiple selected points to a specified text file. Selecting this command opens the Save As dialog. The command is disabled if no point is selected on the Poincaré sphere.
Compute EllipseDisplays the computed polarization data in the Command Output window. The command is disabled if the currently loaded DIS file contains no polarization data.
Render Flux Turns on or off the rendering of flux data.
