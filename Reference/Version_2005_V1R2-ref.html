Late-breaking information about this release of ASAP™.

KERNEL

POLARIZ Settings

ASAP now generates a warning at source-creation time, if it is necessary to change the polarization to avoid nonphysical, axially polarized rays. This warning should be taken seriously.

SHOW: Polarize

The SHOW command reports the complete complex reference-polarization vector. Previously, only the real parts were displayed.

FIELD command causes Kernel crash

ASAP now prints a diagnostic message (instead of crashing) upon a FIELD… MULTIPLY|COUPLE|ADD request for distributions of non-matching dimensions.

WRITE command causes Kernel crash

Eliminated obscure Kernel crashes associated with WRITE.

CRYSTAL waveplate in an IDEAL analyzer

Corrected the a polarization error with birefringent CRYSTAL media.

$FAST

The use of $FAST in $DO and user-defined macros now correctly parses all records following the loop/macro-structure.

$ITER

Introduced the NOMIN option to suppress the minimizing iteration of $ITER.

DISPLAY: Integral value restored.

Output of the “Integral” value was restored for ill-conditioned distributions having no variation of data or incompletely defined data.

TIR POLARIZATION Difference

Fixed an error in the phase of the p-polarized component light subject to total internal reflection.

STATS: TOTAL

The STATS command now prints TOTAL data, even if no rays exist, to facilitate processing with $GRAB.

3D Viewer now has additional help for the Polar Plot functionality

LOCAL

Calculate Flux Distribution Dialog

Calculate Flux Distribution outputted the generated '3D Radiant Sphere' option incorrectly.

ASAP now generates the correct command string based upon user input in the Calculate Flux Distribution Dialog.

CIE

CIE Autoscale

ASAP now supports correct Autoscale functionality on within the Display Viewer.

Display Pull-down

Display Graphics pull-down was missing MESH option.

ASAP now has the MESH option added to the Display Graphics pull-down.

ELTM

GEOMETRY tab functionality

ASAP now supports correct functionality upon error condition of the source having no rays.

ENX file support

ASAP no longer supports direct loading of ENX files into ELTM.

Light Source Library (LSL) Support

When creating a source with the Light Source Library Wizard in conjunction with ELTM, ASAP will automatically load the newly created source.

ELTM Source Path

ASAP now checks for empty source paths within ELTM.

ELTM Geometry and Rays loading

ASAP now correctly makes single copies of geometry and raysets within ELTM.

Incoherent Radiant Dialog

Option Radiant dialog under Analysis menu: Incoherent Radiant Dialog did not recognize Rescaling Factor.

ASAP now generates correct form of the Radiant Dialog commands and recognizes user input values for Rescaling Factor.

Plot Viewer

Plot Viewer Annotation Tool Bar did not render lines correctly

ASAP now correctly renders lines in the Plot Viewer Annotation Toolbar when User changes focus.

Registration Dialog

Registration Dialog Crash

ASAP now correctly implements functionality of the Registration Dialog.

Remote

File Open Failure

ASAP Remote now supports file open functionality for files resident on a REMOTE machine.

RENDER

RENDER Dialog Formatting

ASAP now correctly formats commands PIXELS and RENDER within the RENDER dialog.

User Preferences

ENT support

The User Preferences dialog correctly reflects the deprecation of ENT file support within ASAP.

Translators

IGES/GTX Import Dialog

IGES/GTX Import Dialog has been renamed within ASAP to CAD Import Dialog

IGES Import

Optional Color Name read failure on IGES object type 314.

ASAP now employs a delimiter ‘,’ versus ‘;’ when reading an optional color name on an IGES object of type 314.

Imported Surface shift

ASAP no longer alters revolved surface-end angles when the end angle exceeds 360 degrees.

Degenerate Edge Tolerance

ASAP now employs a more relaxed test for detecting a degenerate edge.
