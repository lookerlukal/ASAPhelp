Commands and Macros (F)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “F”.

  • FACETS (ASAP Command)
    Controls the subdivision of mesh patches.
  • $FAST (ASAP Macro)
    Performs fast reading of numeric arrays.
  • FCN (ASAP Command)
    Specifies a function to wrap around the function of the previous surface.
  • $FCN (ASAP Macro)
    Defines new in-line math functions.
  • $FF (ASAP Macro)
    Inserts a form feed into the output stream.
  • FFAD (ASAP Command)
    Formats currently selected ray data and plots full-field aberration displays (FFAD).
  • FFT (ASAP Command)
    Calculates the spatial frequency spectrum of current distribution data.
  • FIELD (ASAP Command)
    Calculates the exact complex field distribution.
  • FIELDBPM (ASAP Command)
    Field propagation through small geometries and inhomogeneous media.
  • FIELDSUM (ASAP Command)
    Sums coherent beams.
  • FITTED (ASAP Command)
    Creates a surface specified by curve fitting an arbitrary set of points.
  • FLUX (ASAP Command)
    Modifies ray fluxes of currently selected ray data.
  • FMAP (ASAP Command)
    Creates a contour slice map of a 3D surface.
  • FOCUS (ASAP Command)
    Determines the best 3D focal point of currently selected ray data.
  • FOLD (ASAP Command)
    Averages the current distribution data about the pixel number of the specified coordinates.
  • FORM (ASAP Command)
    Controls the form of the current distribution data.
  • FRESNEL (ASAP Command)
    Adjusts the reflection/transmission coefficients specified on an interface using Fresnel's formulae during a ray trace.
  • FTSIZE (ASAP Command)
    Sets the Fourier Transform size to the default maximum.
