Commands and Macros (G)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “G”.

  • GAUSSIAN (ASAP Command)
    Creates a beam set simulating a coherent astigmatic Gaussian mode.
  • GENERAL (ASAP Command)
    Creates a general surface defined by polynomial coefficients.
  • GET (ASAP Command)
    Retrieves ray data and copies it into variables.
  • $GO (ASAP Macro)
    Branches to other records or skips over input records.
  • $GRAB (ASAP Macro)
    Grabs isolated numbers or literals from the output buffer.
  • GRAPH (ASAP Command)
    Creates 1-D profile(s) of the current distribution data file.
  • GRID DATA (ASAP Command)
    Creates a rectangular grid of rays.
  • GRID ELLIPTIC (ASAP Command)
    Generates a rectangular grid of rays inside an elliptical aperture.
  • GRID HEX (ASAP Command)
    Generates a hexagonal grid of rays.
  • GRID OBJECT (ASAP Command)
    Generates a uniform grid of rays on an object.
  • GRID POLAR (ASAP Command)
    Generates a circular grid of rays.
  • GRID RECT (ASAP Command)
    Generates a rectangular grid of rays inside a rectangular aperture.
  • GRID WINDOW (ASAP Command)
    Creates a rectangular raster-type ray grid.
  • GRIN (ASAP Command Argument)
    Similar to the command argument, ABSORB, GRIN handles the gradient index (GRIN) materials by assigning to the medium a GENERAL polynomial or USERFUNC function.
  • GROUP (ASAP Command)
    Groups a collection of objects or sources as a single unit.
  • $GUI (ASAP Macro)
    Sends the given command strings to the graphical user interface (GUI).
