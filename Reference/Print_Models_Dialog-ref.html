PRINT Models Dialog

The PRINT Models dialog is accessible from the System menu command, List Model.

The dialog includes the following polarization options.
Property Type Property Value Property Description
Select List Option:    
All Disables the Polarization Options selection group. Functions the same as the PRINT (Polarization Models) command.
  Polarization Only Enables Polarization Options.
Model Index Optional Specifies the list index of the model to be printed. The default is blank.
Polarization Options:    
Model Type Cascading Element, Jones Matrix Element, or Realistic Polarizer Select one of these types. Jones Matrix Element is the default.
