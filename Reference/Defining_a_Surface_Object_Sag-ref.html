Defining a Surface or Object Sag

This topic lists the input arguments and return for defining an surface intersection.

" \t " "FUNCTION USERSAG( R2, ASPH, N, ERRMSG )

Input arguments:

R2 = X^2 + Y^2 where (X,Y) are coordinates of point of the surface

ASPH(1:N) = array of coefficients passed by ASAP from the DEFORM command

ERRMSG = error message string (80 characters)

Return:


  • additional sag to be added to OBJECT
  • error message in ERRMSG, otherwise ERRMSG is left blank. If ERRMSG is returned non-blank, ASAP stops running.
