Release Notes

Topics in this section include all known release notes for ASAP up to the last version. Release notes for the current version can be viewed in the ASAP Command Output window, or opened from your ASAP installation root folder.

Beginning with ASAP 2005 V1R1, release notes include both kernel and interface (GUI) changes. Prior release notes include only kernel changes.
