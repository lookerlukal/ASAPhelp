Build 05V1R3

KERNEL

VERSION

ASAP now provides the VERSION command which writes the ASAP Kernel version message to the console.

ARRAY

The ARRAY command now recognizes common variants of the option RAN, such as RANDOM.

WIDTHS

WIDTHS now defaults to a value of 1.0 for BEAMS INCOHERENT and a value of 1.6 for BEAMS COHERENT. Non-default values of WIDTHS therefore must be explicitly set after the BEAMS command.

SAMPLED

ASAP no longer abnormally terminates during direct creation of SAMPLED data, as happened in all earlier versions if the distribution filename were already in use.

Resource Release

ASAP no longer abnormally terminates during a rare scenario of Kernel resources release.

GUI

CATIA

ASAP now supports import of CATIA Catpart and Catproduct files created within CATIA releases V5R13, V5R14 and V5R15.

BUILDER

ASAP limits the inclusion of Builder (*.enz and *.enx) file extensions from the ASAP projects.

ASAP no longer abnormally terminates when using Ctrl+C and Ctrl+V on blank lines

User interface fixes include updating default values, clarification and re-positioning of column for the following commands:

WINDOW

TUBE

GRID POLAR

COATINGS LAYERS

COATING MODELS

COATINGS PROPERTIES

COATING

MEDIA

Calculate Flux Distribution

ASAP now generates correct script for the Calculate Flux Distribution dialog.

Light Source Library Wizard

The Wizard page, Source Geometry Setup - Create Raysets, now correctly accepts user input and creates raysets.
