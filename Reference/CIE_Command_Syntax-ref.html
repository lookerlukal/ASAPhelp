CIE Command Syntax

CIE analysis features that are accessible from command syntax are listed below.

Note: Note: Some CIE features supported in the user interface are not supported by command syntax. The last selected Display Viewer profile is automatically used.

You can run the CIE analysis from the Command Input window by entering the macro,

$GUI CalculateCIE

This $GUI Calculate CIE macro performs color analyses to quantify the chromaticity and lightness of photometric values using CIE standard observer models and CIELAB 1976 color appearance models. This is intended to test the uniformity of the projected light on an analysis object, using CIE standards.

The following options are available:

$GUI CalculateCIE -

Note: leave a space after CIE and before the hyphen ( - ).


OptionArgumentsDescription
-ND

Sets the CIE Analysis dialog to not display. Example: $GUI CalculateCIE -ND

See Notes below.

-SOMvalue

Sets the Standard Observer Model to either 2 or 10 degrees. See the CIE Glossary of Terms.

Example: $GUI CalculateCIE -SOM

-CAMvalueSets the option for Color Appearance Model to one of the following text values: 1931 (default), 1964, or 1976
-CSvalue

Sets the Color System to one of the following values:

0 = display monitor (default)

1 = National Television System Committee (NTSC)

2 = Society of Motion Picture and Television Engineers (SMPTE)

3 = European Broadcasting Union (UBE)

4 = Short Persistence

5 = Long Persistence

-WPtext

This sets the observerâ€™s White Point to Illuminant A, B, C, D55, D65, or E.

Example: $GUI CalculateCIE -WP

See the CIE Glossary of Terms for a description of Observer White Point.

Notes

When the ND option is selected, the values from four ASAP commands take effect and control several qualities of the analysis. The values for SELECT and CONSIDER have an impact on the rays and the objects contained in the PGS file that is used, or not used, for this analysis, respectively. The attributes of WINDOW and PIXELS control the window and pixelization used in this analysis.

When the ND option is not selected, the values from just the two ASAP commands take effect and control several qualities of the analysis. The values for SELECT and CONSIDER will have an impact on the rays and the objects contained in the PGS that will be used, or not used, for this analysis. As a result, the objects in the dialog box are only those CONSIDERed.
