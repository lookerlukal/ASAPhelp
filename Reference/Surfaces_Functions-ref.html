Reference Point Properties
  • The surface/function is defined about the reference point, which is located on the surface only if the first coefficient, the constant term, is zero.

  • The reference point locates the surface/function in space.

    Tip: for most classical optical surfaces, the reference point is the intersection of a coordinate axis with the surface.

  • Simple translations/shifts of the surface/function are performed on the reference point.

Typically, polynomial equations describe surfaces infinite in size. A plane is a perfect example of this. Some polynomials, such as ellipsoids, are not infinite in size and are autolimited. Since optical elements have finite size and an aperture semi-diameter, polynomials describing them must also model this behavior. Therefore, the user restricts the size of the optical element in three ways:


  • Within the syntax of some SURFACE commands
  • With a LOCAL box around the surface or a LIMIT box around the object
  • Bounding the surface with other mathematical entities

ASAP stores the name of the command used to generate the base SURFACE/FUNCTION shape, but does not store either the arguments of the command or the individual transformation commands.
