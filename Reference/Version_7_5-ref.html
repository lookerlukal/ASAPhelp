Version 7.5
Version 7.5
Release Notes for ASAP 7.5 Kernel as of January 06, 2002.
Changed commands (affects backward compatibility)

EMITTING OBJECT default is now total ray number instead of rays per facet.

Renamed Commands (but with backward compatibility)

CAD extended to more descriptive CADEXPORT (abbreviation will still work).

FIELD split into FIELDSUM and FIELDBPM (abbreviation still works for both).

New Commands
 
  DISPLAY
    HISTOGRAM plot (number of values in each of specified number of intervals).
  SURFACE
    SUPERCONIC rotationally symmetric asphere used in optical design.
  MEDIA scatter USERprog routine can even change wavelength (e.g. fluorescence).
Enhanced Geometry Commands
 
  "CADEXPORT_ASAP-cmdref.html">CADEXPORT
    Improved ACCURACY of IGES rotationally symmetric SURFACES.
    Can call VECT2DXF utility to create DXF version of current VCR file.
  SURFACE
    ARRAY
      Now has simpler alternative syntax for orthogonal case.
      New SEARCH option can dramatically speedup tracing of "tiled" surfaces.
    CORNER cube can now have a HEXAGONAL aperture.
  CURVE
    USERCURVE can produce smooth rational or non-rational cubic SPLINE curve.
    SPLINE with CURVatures will use rational quadratic segments when possible.
  LENS
    SINGLET MANGIN aberration elimination for any bending and conjugate factors.
    ABERRATIONS solve to maintain back focal distance (last conicoid to focus).
  VUFACETS can also LIST the number of facets created for each object.
  PLOT FACETS can limit, if possible, the total number of facets on any object.
Enhanced Properties Commands
 
  OBJECT
    ROUGHNESS has additional probability distributions to select from.
    SCATTER cone can now be back TOWARDS incidence direction (retro-reflection).
  FUNCTION
    USERFUNC passed current WAVELENGTH (or associated ray/beam's wavelength).
  MODELS
    RAWDATA default collapse threshold changed from .001 to .000001.
Enhanced Analysis Commands
 
  DISPLAY
    Also prints out RMS variance and widths of data.
    GRAPH DISCRETE option displays data as vertical bars of controllable width.
    MESH now defaults to monochrome but also accepts standard COLOR option.
    Automatically calls BMP2DIS utility if given file has a .BMP extension.
  SELECT by previous OBJECT now accepts wildcards in specified name.
 AXIS LOCAL can refer to a specific object not just a ray/beam's current one.
  SPOTS/OPDMAP/SPREAD/FIELDSUM/RADIANT can be in local coordinates of an entity.
  SAMPLED/MESH/DECOMPOSE/EMIT DATA can process local coordinate distributions.
  FIELDBPM Finite Difference Beam Propagation Method (FD-BPM).
    BC option
      Expanded to asymmetric computational window boundary conditions.
      Easy option for (and defaulted to) physically absorbing region at edges.
      Size of absorbing regions can be specified as fraction of total samples.
    Ordering INTERFACE media allows "mismatch" errors to be fixed automatically.
    Semi-vectorial version of wide-angle algorithm added for polarized fields.
  GROUP/CONSIDER special case of ".?" wildcard added.
  CLIP POS followed by a BOUNDS\RBOUNDS command allows complex ray bounding.
  DECOMPOSE accepts any complex distribution file not just BRO029.DAT.
  STATS ALL option also lists objects with no rays and zero flux on them.
  FIELD COUPLE more easily to fundamental mode of step or gradient index fiber.
Macro Language
 
  &GUI parses any expression results into their string equivalents.
  Added ability to access a macro in a library other than current $IO LIB one.
  Higher-order Bessel functions added (BJn and BKn for n up to 9).
  RAN function generates random numbers with various probability distributions.
  $RAN option for default DEC generator or new BEST one.
  $TIC option allows user to set the time units for this and all future output.
  $ITER/$EVAL new form that generates random perturbations of given variables.
  &GRAB strings instead of numbers from output buffer.
  $GRAB can now also find numbers that immediately follow an equal sign (=).
Other Changes
 
  As usual, *.SYS files are incompatible with previous versions.
  Spatial distribution files can now have an arbitrary global orientation.
  IGES Import
    When possible will use wildcard on GROUPs instead of number.
    Can now handle (very rare) case of nested COMPOSITE curves.
  Ray selection flags increased (SELECT still works on any number of rays!)
    Pro from 32 to 100 million, Basic from 9.6 to 10 million.
  Faster tracing of semi-quadratic patches (e.g. IGES imported surfaces).
  Experimental code for "intelligently" setting the default FACETing on objects.
