ELTM - Tasks Supported

The Exterior Lighting Test Module (ELTM) supports the following main functions:


Geometry

Importing reflector and lens geometries from ASAP, IGES, OSLO, and ZEMAX (see Importing/Exporting)

Reflector surface characteristics

Lens materials and transmission characteristics

Light Sources

Using the BRO Source Wizard to create raysets, or write scripts to a template and run the script, or from other sources

Orienting

Beam selection (high, low...)

Ray data or ray sets from ASAP

Supported Test Standards

ECE (Economic Commission for Europe)

FMVSS (Federal Motor Vehicle Safety Standards

SAE (Society for Automotive Engineering)

Results

Color-coded pass/fail data; report generation; graphical presentation

Lumens in angular space (graph)

Lux in position space

  • ELTM Geometry Tab
    Add, remove, and edit your geometry files to the Exterior Lighting Test Module (ELTM). You can also import a source model from the BRO Light Source Library.
  • ELTM Options Tab
    On the ELTM Options tab, select the settings for the type of output you want to view (traced rays, contoured plot, or meshed plot) in the Results area, and check the other settings.
  • ELTM Tests Tab
    Select the test on the ELTM Test tab that you want to apply to your geometry. The list includes over 75 supported test standards (SAE, FMVSS, and ECE) and a brief description of each.
  • ELTM Contour Setup Dialog
    Select the type of ELTM plot contours in the results table, and set parameters for scaling, range, and number of contours.
