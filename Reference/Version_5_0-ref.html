Version 5.0

ASAP 5.0 Release Notes as of May 24, 1996

General 
  New graphical user interface and on-line documentation on most platforms.
Macro Language 
  $PLAY can now turn ON or OFF the capturing of output into the replay buffer.
  $PATH added for setting default directory for files.
ASAP Enhanced Commands
 
  SHOW can now display ALL settings and not just a select set.
  REPLOT allows object number interrogation by pointing and hitting # key.
  CONSIDER can now produce a list of the currently considered objects.
  INTERFACE DIFFRACT efficiency can reference a polychromatic COATING PROPERTY.
  GAUSSIAN command has shorter form for symmetric fundamental mode.
  SELECT ray by current MEDIA has been added.
  MOVE can now specify the closest approach TO a POINT.
  ...PIXELS option will temporarily override number on any pertinent command.
ASAP New Commands 
  DRAWING produces 4 views of 3D VECTOR file data (like old ENDRAW program).
  SPRINT and QPRINT are shorter versions of PRINT.
  DISPLAY;IES writes out a Standard Photometric file of angular distribution.
ASAP Changes 
  Forward referencing of undefined entities is now strictly forbidden.
  Maximum number of sources/wavelengths has been increased from 127 to 1024.
  Maximum BOUNDS entities per object increased from 20 to 30 (100 in Plus).
  VIRTUAL.PGS and *.SYS files are incompatible with previous releases.
ASAP Known Problems That Will Be Fixed in 5.1
 
  Rays can leak through a parametric CURVE-based object in rare circumstances.
  Fluxes in rays split at a CRYSTAL interface are off (especially at non-normal)
Utility Programs 
  DXF2ASAP will now read Release 13 files but will not process any new entities.
