Graphics Files

The graphics archive contains a complete record of all plots produced in a given ASAP session. These files may be examined by opening the file from a plot window.

ASAP produces graphics files with the following extension types:

filename.plr - files run by selecting the lightning button or typing $IO command

asaptemp.plr - files run from the ASAP editor

bro020.dat - all commands run from the command line

Vector files are three-dimensional records of all lines, facets, and points that are plotted during a given ASAP session. Information is appended to this file every time a plot is produced. You can view this file in the ASAP 3D Viewer. ASAP produces vector files with the following extension types:

filename.vcr - files run by selecting the lightning button or typing the $IO command

asaptemp.vcr - files run from the ASAP editor

bro030.dat - all commands are run from the command line
