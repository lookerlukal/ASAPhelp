Modify or Use Internal Ray/Beam Data as Input

This topic lists commands that are used when modifying or using internal ray/beam data as input.


  • A slash between entries indicates that only one of the entries is allowed at a time.
  • Commas indicate that any combination of entries is acceptable.
  • A semi-colon indicates that the commands must be grouped together.

GETLoad ray data into input registers
PUTMove values in registers back
