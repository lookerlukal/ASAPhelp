Radiant Sources

Radiant sources are 3D source characterization files created by Radiant Imaging, Inc. Radiant sources are generally obtained for one-of-a-kind sources, specialty bulbs, or other sources, which are not in the BRO Light Source Library.

Note:

The distribution file should be obtained from Radiant Imaging in the ASAP format (*.dis) for distribution files.

To obtain a full list of the Radiant Source Library contents, please visit the Radiant Web site at www.radiantimaging.com.

  • Radiant Imaging Source Coordinate Convention
    Radiant Imaging, Inc. uses a left-handed coordinate system. When viewing a Radiant Source in ProSource, it is shown using this convention.
