Late-breaking information about this release of ASAP.

Software Quality Work Certificates:


Feature Description
New: ASAP Online Help Converted to HTML Help, with improved appearance and usability. Some help, particularly for the Light Source Library, remains to be converted.
New: Multivariate nonlinear optimization methods Facilitated in a new and easy-to-use graphical interface. Existing ASAP scripts can be used with little or no modification, with continuous graphical and tabular progress reporting.
New: Polarization raytracing and analysis Expanded in scope and accuracy to include physically realistic polarizers, powerful tools for specifying polarization (including randomization) of sources, and visual polarization analysis with an interactive Poincare sphere.
New: SolidWorks LayeringA layering capability has been implemented as a SolidWorks(TM) plug-in to streamline the organization and import of CAD geometry into ASAP.
Fixed: Display Viewer 3D Function now launches correctly.
Fixed: FRESNEL TIR The behavior of FRESNEL TIR, which changed in ASAP 2006 V2R1, has been restored for non-absorbing media. For absorbing media, FRESNEL TIR remains fundamentally non-physcial.
Fixed: Memory access A memory-access issue, which could have resulted in data loss related to the definition of COATING LAYERS, has been fixed. BRO is not aware of any instance of this occurring in actual use.
Fixed: 22 errors resolved in Builder:These are described below briefly; detailed questions should be directed to BRO.

The RADIAL command now works correctly in the Builder.

The CONSIDER EXCEPT now works correctly in the Builder.

"Force Entity" now works correctly in the Builder.

The Builder now supports USERAPOD { DIR | POS | ANGLES }.

The Builder now supports ARRAY EXP.

Handling of BOUNDS has been improved in the Builder.

The EDGE and SWEEP commands now work correctly in the Builder.

Forgetting to save changes sometimes resulted in an empty file.

The PARABOLIC command now works correctly in the Builder.

The Builder now supports $GO.

Eliminated a crash which could occur when a Builder file was invoked via the "Run" button in the main ASAP interface.

The Builder can now load templates generated by the Light Source Library Wizard.

"System…Object Control…Object Modifiers" has been fixed.

A crash could occur when calling a macro from the Builder. This has been fixed.

A crash could by induced by clicking the leftmost "*" column. This has been fixed.

The TUBE command now works correctly in the Builder.

The Builder now supports all valid SEARCH syntaxes.

The Builder now supports all valid SELECT syntaxes.

Using the arrow-keys in the Mini-Builder no longer locks up ASAP.

Omitting WAVELENGTH units in a Builder file could crash ASAP. This has been fixed.

The CVF command now works in the Builder.

The column was erroneously incremented upon saving a file.
