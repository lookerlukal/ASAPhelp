Defining a Surface Intersection

This topic lists the input arguments and return for defining an surface intersection.

SUBROUTINE USERSURF( PRAY, DRAY, PSRF, CSRF, MXNC, RMAT, DIST, * )

Input arguments:


PRAY(1:3) = previous ray point in global coordinates
DRAY(1:3) = global direction cosines of ray
PSRF(1:3) = global coordinates of surface reference point
CSRF(1:MXNC) = array of surface coefficients passed by ASAP from the USERSURF command
RMAT(1:3, 1:3) = rotation matrix for transforming from global to local coordinates

Return:


PRAY(1:3) = intersection point in global coordinates
DRAY(1:3) = direction of normal to surface at intersection point
DIST = distance along ray to intersection with surface

use alternate return if no intersection or error
