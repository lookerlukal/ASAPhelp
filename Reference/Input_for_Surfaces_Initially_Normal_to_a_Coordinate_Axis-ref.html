Input for Surfaces Initially Normal to a Coordinate Axis

The second entry on the following commands designates the axis of symmetry (either X, Y, or Z) for the surface. Many of the surfaces created by these commands can extend to infinity unless a LOCAL command follows or where permitted, a trailing aperture option of the following form is specified: 

    ... X ... ELLIPSE   a [ a' [ o [ s [ s' ] ] ] ]
        Y     RECTANGLE
        Z     HEXAGONAL a [ o [ s [ s' ] ] ]

For the HEXAGONAL form, a is the center-to-vertex distance (maximum height).
