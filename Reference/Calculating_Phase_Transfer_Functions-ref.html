Calculating Phase Transfer Functions

While Modulation Transfer Function (MTF) calculations are by far the more common of the two, there are instances when the Phase Transfer function (PTF) is also needed to completely characterize the imaging properties of an optical system.

By definition, the OTF is defined as the Fourier transform of the point spread function. The OTF is in general a complex function that may be written in the form:

OTF(v_x,v_y) = MTF(v_x, v_y)EXP[iPTF(v_x,v_y)]

The FFT command by default extracts the MTF from the OTF and places it into the distribution data file. The FFT WAVEFRONT and FFT PHASE commands extract the PTF from the OTF and place it into the distribution data file. The FFT WAVEFRONT command attempts to remove the 2Ï€ phase steps (or jumps) by integrating in the across direction as set by the WINDOW command. When doing both tangential and radial PTF calculations, it may be necessary to re-enter DISPLAY and TRANSPOSE the data prior to calling FFT WAVEFRONT a second time.

To select the center of the PTF curve for plotting, it is usually necessary to use GRAPH 0.5 rather than the default since the maximum value of the PTF is not likely to be coincident with the maximum of the data.
