Linear Transformations for Lenses

Linear transformations change the shape, scaling, or orientation of a geometrical entity (surface, edge, lens, object, group, or rays) by applying a general 4x4 transformation matrix (linear transformations change only the orientation of LENSES). Linear transformations that change the location of a geometrical entity are applied to the reference point (see table) of the entity.


LENSESReference Point
SEQUENCEat vertex of first conicoid
MIRROR at intersection of the mirror and coordinate axis
SINGLETat intersection of first conicoid vertex and coordinate axis
MANGINat intersection of first conicoid vertex and coordinate axis
RIGHTat intersection of first face and coordinate axis
AFOCALat intersection of first conicoid vertex and coordinate axis
IDEALat intersection of first conicoid vertex and coordinate axis
Tips
  • Any number of the following linear transformation commands may be applied after any entity definition to build up the final matrix. The order in which these transformation commands are entered is exactly the order in which they are to be applied to the entity. These commands must be grouped together following an entity definition with no other commands between them (except for a comment).
  • The LIST option causes the resulting 4-by-4 transformation matrix to be printed and decoded into simple operations if possible.
  • For the transformation of a direction vector, the dummy row contains all zeros.
  • Ray data may also be transformed by these commands.
  • For more information, see Coordinate Transformations in ASAP.
