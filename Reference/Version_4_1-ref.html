Version 4.1

ASAP 4.1 Release Notes as of July 25, 1995

IMPORTANT CHANGE (requires modifying any affected user files)
  OBJECT; signs of solid "q" factors have been reversed to conform with BOUNDS.
Other changed commands 
  PLOT ...
    LOCALS/LIMITS plots only boxes, i.e. no longer INTERSECTS surface.
    OBJECTS has been deleted since MESHES is now the equivalent (and default).
  SEGMENTS now sets number of line segments in a 45 (instead of 90) degree arc.
Undocumented 4.0 Commands
 
  EMITTING HELIX creates a straight helical filament source directly.
New Commands 
  SURFACE
    PARAMETERIZE given local coordinate axis for meshing surface.
  PLOT ...
    ENTITIES plots all entity types (SURFACES, LENSES, and EDGES).
    SURFACES directly plots surface entities which have local boxes.
  DISPLAY
    IMAGE produces "gray"-scale picture of distribution using SHADED process.
  MODEL
    VCAVITY is a geometrical model for a rough (random v-cavities) surface.
Enhanced commands 
  $EDIT can now edit arbitrary files (not just inputs and macros).
  $HELP system is now menu-driven.
  DISPLAY
    WRITE can now optionally produce only the DIN or DIS version of file.
    DIRECTION now has a legend to help distinguish the two curves.
    AVERAGE can optionally smooth data using a median window technique.
  PLOT ...
    MESHES/FACETS work not only for EDGE objects but also LENS and SURFACE ones.
    FACETS...HIDE option produces hidden-line perspective plot.
  USERAPOD BOTH "usap3d.#" file can now accept data every degree in both angles.
  XMEMORY can specify any name for page file (not just VIRTUAL.PGS).
  LEVEL can generate scatter rays for ALL specular child rays (not just parent).
  SCATTER can be ABSolute or RELative with separate reflect/transmit scaling.
  SELECT now works on any number of rays (not just ones with bit flags).
  FIELD can now ADD its results to a previous field calculation.
  HISTORY can show ray SPOTS on objects instead of PLOTting trajectories.
Code Changes
 
  As usual *.SYS files are incompatible with previous versions.
  All known bugs have been fixed.
New Platforms 
  AIX 4.1 on IBM RISC/6000 and PowerPC.
  Microsoft Windows 3.1 on Intel CPUs.
