Surface or Function Entities

ASAP is capable of handling any "implicit" surface that can be represented by an arbitrary function of a general N-th order polynomial in the global coordinates X,Y,Z with arbitrary reference point x,y,z and real coefficients c.



ASAP can go to tenth order (N=10); that is, 286 coefficients. Therefore, anything from a plane to an arbitrarily oriented elliptical toroid can be modeled. If the reference point lies on the surface, then the first coefficient (the constant term) will always be zero. "fcn" is an optional macro function (intrinsic; for example, SIN, or user-defined $FCN).

An "explicit" surface function (sag equation) can also be represented by an arbitrary function of a general N-th order polynomial in the local coordinates X,Y with arbitrary reference point x,y,z and real coefficients c.
