Commands and Macros (J-L)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letters "J" to "L".

  • JONES (ASAP Command)
    Defines a Jones matrix element.
  • LAMBERTIAN (ASAP Command)
    Creates a Lambertian scatter model.
  • $LEAVE (ASAP Macro)
    Causes premature exit from a loop, macro, or input file.
  • LENSES ENTITIES (ASAP Command)
    Signals ASAP that lens definition commands follow.
  • LEVEL (ASAP Command)
    Controls the number of times a scattered ray may be rescattered.
  • LIGHTS (ASAP Command)
    Sets up optional light sources for the RENDER command.
  • LIMITS (ASAP Command)
    Assigns a global bounding box to the object.
  • LINE (ASAP Command)
    Creates a line edge.
  • LIST (ASAP Command)
    Lists currently selected ray data by the second literal entry for the current ray set.
  • LIST ELLIPSE (ASAP Command)
    Lists polarization parameters of currently selected ray data.
  • LIST INTEGER (ASAP Command)
    Lists ray database information.
  • LIST (Parabasal Ray Data) (ASAP Command)
    Lists currently selected base and parabasal ray data.
  • LIST RAYS (ASAP Command)
    Lists ray data in EMITTING RAYS format.
  • $LOC (ASAP Macro)
    Makes the given registers/variables names local to the current macro.
  • LOCAL (ASAP Command)
    Assigns a local bounding box to the surface/function.
  • LSQFIT (ASAP Command)
    Controls the singular value decomposition least squares fitting algorithm.
