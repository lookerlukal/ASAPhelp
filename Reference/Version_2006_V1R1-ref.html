Features:

70301 -- CIE Custom Display

Added the ability to customize the data columns displayed in the CIE data output window.

70301 -- CIE Render Intensity

Added the ability to view RGB rendered with respect to intensity.

70302 -- Tolerancing phases II and III

Added sensitivity and compensator analysis functionality.

Enhanced Reporting capabilities.

70305 -- CAD Export

Enhanced IGES CAD export, now exports BOUNDS and LIMITS trimming information.

70306 -- IES Export -- 06PRD00000327

Improved management of length units.

Added support for the source flux.

Made output consistent with more IES file viewers.

70310 -- Path eXplorer

The Path eXplorer is a fast, flexible and powerful tool for discovering and

analyzing patterns in the behavior of light passing through an optical system.

It is fully scriptable with over a dozen new commands and is also accessible

through the user interface.

70315 -- Quick Start Toolbar enhancements

Added quick access to several example interactive scripts such as:


  • Bio-Optical Models - Dialog-driven skin modeling technique and a dialog-driven

    script which can be easily used for the creation of slab tissue models.

  • Array Enhancers
  • Lens Creation

06PRD00001071 -- New command $SEARCH for managing macro (MAC), library (LIB) and

screen (SCR) files.

(Rollup of 1052 & 1063 and more) Search semantics can now be tailored at will,

with complete backward compatibility.

SQAWCS Worked:

06PRD00000017 -- CIE Unfunctional buttons

Toolbar and pull-down menus are now fully functional.

06PRD00000023 -- CIE Autoscale

Negative Autscale WINDOW Y -X now works.

06PRD00000120 -- CIE models 64 and 76 showing the wrong colors in the quadrant view

CIE models 1964 and 1976 now show correct colors in the quadrant view.

06PRD00000122 -- CIE toolbar buttons

The save, maximize chroma, maximize display, maximize text buttons on the

toolbar now function correctly.

06PRD00000123 -- CIE Saving CV-X, CV-Y and CV-Z values

The CV-X, CV-Y, and CV-Z values are now being saved directly and unnormalized to

the text file as they should.

06PRD00000124 -- CIE Results table columns

The results table shows columns labeled CV-X, CV-Y, and CV-Z have been labeled more clearly.

06PRD00000125 -- CIE Combo box for display selections.

CIE display selections can now be customized. See CIE Custom Feature above.

06PRD00000126 -- CIE Copy to the clipboard

Ability to copy the display viewer/chromaticity diagram to the clipboard is functional.

06PRD00000197 -- LSL Wizard

Provided clearer instructions for creating raysets with LSL Wizard.

06PRD00000217 -- Intuitive interface for SAVE/HISTORY data

See Path eXplorer, above.

06PRD00000402 -- Crash on inappropriate nesting of POINTS

ASAP now reports a syntax error instead of terminating.

06PRD00000461 -- Scotopic Function EYE approximation has large errors.

The EYE[] Photopic luminous efficacy is now linearly interpolated with CIE 1924

Spectral Luminous Efficiency Function for Photopic vision with 1nm resolution.

Also the EYE[] Scotopic luminous efficacy is linearly interpolated with CIE 1951

Spectral Luminous Efficiency Function Scotopic vision with 1nm resolution.

06PRD00000491 -- VALUES command gives incorrect values

Fixed

06PRD00001049 -- Zombie Hang

Implemented a fix to lessen the effects of ASAP instances that were not closing properly.

06PRD00000350 -- LSL Wizard

ASAP now displays units for flux value on Source Geometry-Create Rayset dialog.

06PRD00000378 -- LSL Wizard

Removed ability to change primary and secondary axis during a rayset creation.

06PRD00000383 -- LSL Wizard

Removed the ability to choose 'Create and Save Rayset' option when LSL Wizard is initiated from within ELTM.

06PRD00001043 -- LSL Wizard

Added ability to add wavelengths and edit existing wavelengths by double-clicking on a column in wavelength list.

06PRD00001052 -- $SCR files are now more flexibly relocatable.

See $SEARCH command above.

06PRD00001060 -- DISPLAY has potential to crash Kernel

Fixed rare application-termination caused by DISPLAYing malformed distribution files.

06PRD00001062 -- LSL Wizard

Eliminated crash that occurs when selecting points on spectrum display.

06PRD00001063 -- $IO LIBRARY libraries are now more flexibly relocatable.

See $SEARCH command above.

06PRD00001117 -- Fixed script-termination manifest when library$macro was

invoked twice in succession.

Now works as documented.

06PRD00000423 -- Removal construction surfaces from CATIA translation

Fixed. User must check the hidden status of all nodes in the product tree.
