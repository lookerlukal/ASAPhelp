ELTM - Importing Files

Using ELTM, you can import your geometry files or BRO light sources from the BRO Light Source Library.


File TypeExtension
ASAP Builder*.enx
ASAP command mode*.inr
IGES*.igs
OSLO*.len
ZEMAX*.zmx
Caution

ASAP (*.inr) files may disrupt the capabilities of ELTM. Use only commands that are compatible with the Builder, so that ELTM can modify parameters it expects to be static. Avoid commands such as DISPLAY and TRACE.
