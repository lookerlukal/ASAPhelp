Coating Storage

COATINGS are stored in a complex array. As COATINGS are defined, they fill the array at the next available location.

Tip

For the following sequence, the INTERFACE command, which is an object modifier, can reference these coatings for assignment as an object's specular optical properties:


BARE_SUBSTRATECOATING 0
0.25 MGF2 ... 'SCOAT'COATING 1
0 1 ... 'TRANS'COATING 2

BARE_SUBSTRATE is the coating name for modeling the polychromatic transmission properties of a bare substrate.
