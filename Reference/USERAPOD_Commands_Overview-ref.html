USERAPOD Commands Overview

The USERAPOD command imposes either a spatial or directional apodization (both options can be used together) on a distribution of rays.

ASAP has three options under the USERAPOD command:


  1. POSITION creates a spatial apodization specified by a table of data or a function
  2. DIRECTION creates a directional apodization specified by a table of data or a function
  3. ANGLES creates a spatial and/or directional apodization specified by a function or data stored in a file
Note:

Since flux is assigned to each ray during its creation, the USERAPOD command must be entered before the GRID or EMITTING command that specifies the initial ray coordinates.
