Batch Mode

If you are an experienced ASAP user, you may prefer to use ASAP in batch mode. You may also find it more efficient to perform long, computationally intensive runs in batch mode. Alternatively, you can develop an input file (*.inr or *.enx) using the Builder, and then perform the full run in batch mode. From the File menu in the Builder, you can convert the *.enx file to an *.inr file by selecting Save As and then ASAP INR Files .

Starting Batch Mode

At the command prompt, type:


RABET [filename.inr] /bif you are using ASAP/Basic, or
RABETP [filename.inr] /bif you are using ASAP/PRO

The /b switch is for running the designated file in batch mode. If /b is not entered, an ASAP prompt displays with a list of options. You can enter input directly to ASAP from the command prompt, as you would in the Command Output window.

Note:

Before you can use the batch mode for the first time, you must have entered your ASAP keys in the Registration dialog box in the ASAP user interface. This dialog box is on the Help main menu.
