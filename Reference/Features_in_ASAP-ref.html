Features in ASAP

Topics in this section of the Contents tab include new and existing features in the ASAP user interface.

  • BSDF Fit Utility Overview
    In most instances where it is appropriate to apply the Harvey model or the Polynomial Fit model, you can use the BSDF Fitting utility, which is integrated with ASAP.
  • CIE Color Analysis Overview
    International Commission on Illumination (CIE) color analysis tests for uniformity of projected light on a surface. CIE numerical and color analysis in ASAP is ideally suited for projection and display customers. Although ASAP has been able to track multiple wavelengths for light rays, CIE visualization mechanism makes it possible to determine their positions in the CIE color coordinate space.
  • ELTM - Exterior Lighting Test Module Overview
    ASAP Exterior Lighting Test Module (ELTM Module) facilitates the process of testing automotive exterior lighting.
  • Importing/Exporting
    You can import or export several types of files, either those generated by ASAP or those used as inputs to ASAP.
  • BRO Light Source Library Overview
    The BRO Light Source Library is a growing collection of U.S. and European source models that can be imported directly into ASAP projects.
  • BRO Light Source Library Wizard - Overview
    The Source Wizard is a step-by-step guide for implementing pre-defined source models in ASAP 2005 and later versions. Users control the source's coordinate positioning and axis alignment with the BRO Light Source Library Wizard.
  • Optimization - Overview
    The Optimization feature in ASAP addresses your need to optimize optical systems under evaluation that were designed or analyzed with ASAP. This topic gives a frame of reference, focusing on criteria you need to satisfy, and the process of designating an optimization method before you can initiate an optimization run in ASAP via the user interface. Related topics are listed at the end of this and other topics for this feature.
  • Path Explorer Overview
    The Path Explorer renders a three-dimensional, interactive visualization of specifically selected ray trajectories, or paths, computed in the course of ASAP simulations. This feature is useful for isolating rays that are doing particular things, such as scattering or ending on a selected object(s).
  • Polarization - Poincare Sphere Visualization Overview
    The purpose of the ASAP polarization feature enhancement is to provide a Poincaré Sphere Visualization Tool (PSVT) via the 3D Viewer for visualizing the polarization state of rays.
  • ASAP Remote Overview
    Remote is a feature of all ASAP products except ASAP Lite. To use it on a client machine, you must have a valid license.
  • Tolerancing in the Builder
    The Builder in ASAP allows for tolerance data entry and automatic Monte Carlo analysis.
  • Tolerancing in the Editor
    Tolerancing in the ASAP Editor is a two-step process: First, you create a tolerance data file, and then perform the Monte Carlo or Sensitivity analysis.
  • SCR (Screen) Input Editor
    The SCR input editor defines a programmable screen template— a user interface, such as a dialog box—for editing input/output (I/O) dialog boxes. The SCR Editor supports the I/O of any ASAP macro.
  • Script Templates Overview
    Script templates assist in the definition and description of user-specified optical systems in ASAP. ASAP provides a collection of pre-defined script templates, and users may create custom script templates.
