CAD Library

The BRO CAD library (cad.lib) contains macros for turning common ASAP surfaces into parameterized (EDGE/CURVE-based objects).

The library is located in your ASAP installation folder under projects\samples\tutorial folder. It contains macros for overriding some of the common ASAP surface commands so that you can build an equivalent edge-based object. The object can then be exported via the CAD command.

For example, the following surface, 

				SURFACE 
				PLANE Z 5 RECT 3 3 0 
				OBJECT = 'RECT'

can easily be redefined as an equivalent edge base object: 

				$IO LIB CAD 
				   &PLANE Z 5 RECT 3 3 0 
				   OBJECT = 'RECT'
