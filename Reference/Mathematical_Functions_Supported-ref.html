The fractional part of the argument to RAN is the relative amount (total probability) of an additional uniform variate with the same RMS or maximum; for example, RAN[2.6] has a probability distribution that looks like a house.

Tip

The ability to perform arithmetic operations during input decoding is a powerful feature when used with the macro facility.
