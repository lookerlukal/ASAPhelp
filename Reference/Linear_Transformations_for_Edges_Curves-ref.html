Linear Transformations for Edges and Curves

Linear transformations change the shape, scaling, or orientation of a geometrical entity (surface, edge, lens, object, group, or rays) by applying a general 4x4 transformation matrix (linear transformations change only the orientation of lenses). Linear transformations that change the location of a geometrical entity are applied to the reference point (see table) of the entity.


EDGES/CURVESReference Point
ELLIPSEat intersection of edge plane and coordinate axis
RECTANGLEat intersection of edge plane and coordinate axis
ROUNDEDat intersection of edge plane and coordinate axis
Tips
  • Any number of linear transformation commands can be used to build up the complete transformation. The order in which these transformation commands are entered is exactly the order in which they will be applied to the entity.
  • The transformation does not have to be orthonormal; this setup permits scaling and skewing of the entity.
  • For the transformation of a direction vector, the dummy row contains all zeros. Ray data may also be transformed by these commands.
  • For more information, see Coordinate Transformations in ASAP.
