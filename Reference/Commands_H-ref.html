Commands and Macros (H)

Topics in this section of the Contents tab include ASAP commands and macros that begin with the letter “H”.

  • HALT (ASAP Command)
    Sets the conditions controlling ray termination.
  • HARVEY (ASAP Command)
    Creates a Harvey (linear shift invariant) scatter model or simple specific model.
  • HEADER (ASAP Command)
    Redefines the file header for the current data in the display file.
  • HELIX (ASAP Command)
    Creates a general helical curve.
  • $HELP (ASAP Macro)
    Displays information on all or one command or a listing of the MACROS in the current library.
  • HISTOGRAM (ASAP Command)
    Displays a plot of the number of data values.
  • HISTORY (ASAP Command)
    Recalls ray data from a previously saved binary file for plotting or listing. The first syntax is for compatibility. The second syntax is for the Path Explorer.
  • HORN (ASAP Command)
    Creates a generalized tube with polynomial profiles and elliptical/rectangular cross-sections.
