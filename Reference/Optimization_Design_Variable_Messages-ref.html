Optimization Informational Messages
Optimization Informational Messages

This topic describes informational messages issued by ASAP when you are defining design variables, design objectives, and objective constrains for optimization. It recommends corrective action.

ASAP Messages for Defining Design Variables
Subject Description Action
Count Total number of design variables has surpassed the maximum, and defining a design variable is disabled on the Optimization menu. Remove some of the design variables and add new ones.
Text Selected text in the script is not a valid design variable. Select text in the script that is a valid design variable.
Name Selected design variable name is already in use. Select a unique design variable.
Alias Alias name is already in use. Assign a unique alias.
Minimum or Maximum value Entered value is not valid. Enter a valid minimum and maximum value.
Step Size Size is not valid. Enter a valid size.
No name or nominal value The design variable name and nominal value must not be blank. Enter a value for these fields.
ASAP Messages for Design Objectives
Subject Description Action
Text Selected script is not a valid design objective. Select valid script in the Editor window.
Name Selected design objective name is already in use. Select a unique design objective name.
ASAP Messages for Objective Constraints
Subject Description Action
Text Selected script is not a valid objective constraint. Select valid script in the Editor window.
Name Selected name for the objective constraint is already in use. Select a unique constraint name.
