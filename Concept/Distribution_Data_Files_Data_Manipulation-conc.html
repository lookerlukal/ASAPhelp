Distribution Data Files - Data Manipulation

ASAP supports a rich set of data processing commands that act on the distribution data files.

These include:


  • Normalization, averaging, rotational averaging, and folding
  • FFT
  • Modification of specific pieces (pixels) or regions of data
  • Combining (adding, subtracting, multiplying) the current distribution file in ASAP with a distribution data file on disk
  • Data set reduction
  • Transfer of distribution data into and out of variables
Tip

Distribution data files are stored on disk and are copied into ASAP when a DISPLAY command is entered. The original distribution data file is unchanged unless a WRITE command replaces it.
