CIE Window

The CIE window shows the results of a CIE analysis in ASAP.

Only CIE files (*.cie) can be opened in the CIE window. The window is composed of three panes: Chromaticity Diagram, Display Viewer, and Results Table. During CIE calculations, the status bar, at the bottom of the CIE window, indicates that calculations are occurring.



Chromaticity Diagram

The View Sample button on the CIE Color Analysis dialog box displays a CIE Chromaticity Diagram. The diagram is based on your selected Observer White Point, which you select from the drop-down box on the CIE Color Analysis dialog.

The diagram identifies information about the currently selected white point, black body locus associated with the current color space, current point of interest from the Display Viewer, current point of interest from the Results Table, and the gamut triangle for the current color system when the Display Gamut box is selected in the CIE Color Analysis dialog.

CIE Display Viewer

The Display Viewer is a graphical view of selected colorimetric data in the current window for the currently considered object. This view depends on the selected View Selection on the CIE Window toolbar.

CIE Results Table

The Results Table displays results of the CIE calculations. The data is dependent on the Display/Calculations profile that you selected prior to the CIE Color calculations. A highlighted row corresponds to a selected point in the Chromaticity Diagram and CIE Display Viewer.

Right-click in the table to access the following commands:


Copy AllCopies entire contents of the table to the Clipboard
Copy LineCopies only contents of currently selected line in the table to the Clipboard
