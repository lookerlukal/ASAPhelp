Parabasal Rays - Scaling the Width

The integrity of any superimposed optical field is a function of the spatial interaction of the individual beams with respect to each other. The number of beams and their overlap controls this interaction.

The total number of beams is determined by a RAYSET or GRID command. The WIDTH command is used to scale the width of each beam to change their overlap.
