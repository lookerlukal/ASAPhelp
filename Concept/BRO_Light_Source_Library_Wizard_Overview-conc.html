BRO Light Source Library Wizard - Overview

The Source Wizard is a step-by-step guide for implementing pre-defined source models in ASAP 2005 and later versions. Users control the source's coordinate positioning and axis alignment with the BRO Light Source Library Wizard.

The Light Source Library Wizard is encountered in three distinct settings:


  • Creation of user-specified raysets for later use
  • Implementation of source models and pre-defined raysets in script files
  • Use of the ELTM module with ASAP

In addition to using the wizard from within the ASAP ELTM module, two options are available in the Light Source Library wizard: Create and save rayset(s), and Writing ASAP script commands to a template.

Note:

ASAP customers must be current on their maintenance agreement to import sources.
