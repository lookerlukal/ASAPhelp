Display Viewer Window Overview

The Display Viewer window depicts distribution data files in either 2D or 3D. It creates plots of ray spots and controls contours, palettes, pointers, and labeling. Options described in this topic enhance the data viewing.



The 3D feature lets you view portions (or slices of data from an oblique angle. It gives visual height information of pixels, rather than only the 2D, flat view. The 2D view must be active before you can select the slice, value, and view from the toolbar.

  • Display Viewer Startup
    This topic describes how to open existing display files for viewing in the Display Viewer window.
  • Display Viewer Menu
    The Display menu is accessible on the ASAP men bar when the Display Viewer is open. This topic describes the commands on this menu.
  • Display Viewer Window Toolbar
    The Display Viewer Window toolbar includes commands for copy, paste, and other functions, eliminating the necessity of your accessing the main menu.
  • Display Viewer Background Settings
    This topic describes commands on the context menu in the Display Viewer.
  • Display Viewer Graphic Display
    This topic describes commands on the context menu in the graphic display area of the Display Viewer window.
  • Display Viewer Palette Display
    This topic describes commands on the context menu in the color palette area of the Display Viewer window. The menu is accessed by right-clicking the palette area.
  • Display Viewer Plot Display
    This topic describes commands that are accessisble from the plot display area in the Display Viewer window.
  • Display Viewer Select Rays Menu
    The Select Rays menu opens from the Display Viewer when you have selected an area by holding down the Shift key, drawing a rectangle in the display area, and then right-clicking the rectangle.
  • Display Viewer - Selecting an Area or Rays
    You can select an area of the display by holding down the Shift key on your keyboard and dragging the mouse to draw a rectangle around the desired area.
  • Display Viewer Tile Dialog
    The Tile dialog box opens from ASAP Workspace by right-clicking Display Viewer and selecting Tile from the context menu.
  • Load Configuration Dialog
    When you apply any of the four Display Viewer window user-defined view, use the Load Configuration dialog box.
  • Save Configuration As Dialog
    When you make changes to any of the four Display Viewer window user-defined view settings, you can save the changes to file for future use in the Save Configuration As dialog box.
  • SELECT Rays Dialog
    Designates current set of rays that ASAP is to consider in calculations and output. This dialog box is accessible when you right-click in the Display Viewer window.
