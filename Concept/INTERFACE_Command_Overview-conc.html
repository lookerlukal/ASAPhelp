INTERFACE Command Overview

The INTERFACE command is used to assign specular optical properties (transmission, reflection, and diffraction) to objects. The command draws upon previously defined databases:


  • MEDIA for refractive index data
  • COATING for multilayer stack prescription/properties
  • SURFACE/FUNCTION for gradient index functions
  • SURFACE/FUNCTION for diffraction grating/HOE spacing
