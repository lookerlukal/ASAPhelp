ASAP Demonstration Program

BRO has packaged a number of complete sample projects to give you an idea of the many capabilities in ASAP (Advanced System Analysis Program). You can run any sample project as many times as you want for as long as you want. If you are currently using the ASAP Demonstration Program, and not the full ASAP version, this topic provides a few tips.

With any of the supplied sample projects, you can perform analysis and make use of many ASAP features (for example, viewing, plotting, and printing). Please keep in mind, though, that you cannot modify any of the sample projects or create your own projects with the demo version.

For information on ASAP dialogs or commands, please use online Help. This is the complete Help available in the full version of ASAP.
