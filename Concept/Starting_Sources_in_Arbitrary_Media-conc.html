Starting Sources in Arbitrary Media

ASAP initializes MEDIA 0 to represent vacuum/air, so you do not need to define a medium.

MEDIA 0 defines a medium whose refractive index is unity for all wavelengths and calls it VACUUM or AIR.

MEDIA 0 also initializes the ray medium to something other than vacuum/air, but this approach is not recommended. Use the IMMERSE command in place of MEDIA 0.
