Media Database Overview

ASAP uses refractive indices to calculate optical transformations at an object. The refractive indices are stored in a MEDIA database. You must specify any refractive media data for a given analysis by entering that data in a MEDIA table.

Unless the calculation has some wavelength dependence, or dispersive media are used, you need only enter a single refractive index/medium.

ASAP performs only a simple linear interpolation to compute the refractive indices of dispersive media at wavelengths other than those defined in the MEDIA table. There is no private catalog utility, although one could be implemented with the macro feature.

Glass catalogs are available but their data is not entered directly in the MEDIA table. That data is accessed directly by an object on the object's INTERFACE command.
