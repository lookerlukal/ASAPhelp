Optimization - Setup Summary Window

The Optimization Setup Summary window presents a multi-tabbed user interface for viewing and entering information that ASAP will use to optimize a system under evaluation (SUE).



Note: To make Optimization Setup Summary window the active window, select on the taskbar in Windows.
Tabs on this window include Design Variables, Design Objectives, Objective Constraints, Exit Criteria, and Optimization Method. Move among the tabs in any order to enter or change settings. Select the Help button to open the help topic related to the active tab. If you select the Close button, the optimization session ends.
Note: You do not need to follow a specified linear process for completing the Optimization Setup Summary window. ASAP validates your entries or notifies you via informational messages about invalid entries.
