ELTM Results

The results table in ELTM reports the data processed in ASAP. You can view the results in the Display Viewer window in ELTM.

Columns:


AreaLists test areas. Select an area to view the results in the display window.
ResultPass/fail status (see Color Coding)
ValueAverage value
RMSRoot Mean Square
MinActual minimum value
Spec MinTest specification's minimum value
MaxActual maximum value
Spec MaxTest specification's maximum value
Color Coding
  • Green indicates passing and red indicates failure.
  • Teal (blue) with an asterisk indicates when a test area is not fully within the display window. Even though the test may have seemed to have passed, it is not clear what would have happened if the window had been large enough to encompass the test area in question.
  • Yellow occurs when a test area is completely outside the display window, so no points can be tested.
Remarks

The intensity beam pattern shows a slice of data on a pixel-by-pixel basis. ASAP determines the steepest point (-0.4 degrees +/- 0.2 degrees).
