DECOMPOSE Command Overview

Aperture diffraction calculations made by ASAP are partly a function of the beam sampling of the aperture. The question is whether a suitable number of Gaussian beams clipped by the aperture so that the aperture apodization is represented in the subsequent diffraction pattern.


  • For example a circular, collimated beam of light incident on a rectangular aperture that is brought to focus has an irradiance pattern that is a sinc-squared function. The fidelity of the ASAP diffraction pattern to the real pattern is a function of how well the Gaussian beams passing through the aperture represent the pattern of the aperture.
  • When using ASAP to filter a set of beams spatially at focus, the above sampling condition would not be fulfilled at all. For example, a set of Gaussian beams that describe a perfect optical field at focus overlays each beam. A spatial filter placed in the focal plane of the ASAP model passes all of the beams, since the base ray would typically pass through the aperture.
Tip

The parabasal rays are not clipped. This is also true for beams near, but not at-focus, for which aperture sampling becomes an issue.

Spatial Filter Calculations

With the DECOMPOSE command, you can perform spatial filter calculations where aperture sampling is an issue. The DECOMPOSE command transforms a distribution file, created by FIELD ENERGY, into a new set of Gaussian beams representing the optical field at the point of the FIELD ENERGY calculation. Since FIELD ENERGY has a CLIP option, you may clip the distribution; that is, spatially filter it. The DECOMPOSED set of beams will simulate the spatial filtering. Also note that the window size for FIELD acts like another spatial filter clipping radiation outside the window.

Source Creation

Consider DECOMPOSE as a command for source creation. DECOMPOSE creates the spatial and angular properties of the beams, but you need to define the physical properties of these beams. Since DECOMPOSE essentially creates a new source, use the appropriate commands to verify the source fidelity.

  • Decompose Coherent Field Dialog
    Decomposes an existing field into a new set of Gaussian beams.
