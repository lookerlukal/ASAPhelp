Edge and Curve Modifiers

Edge/curve modifiers fall into four categories, which are listed in this topic.


  • Options that distort or deform the edge points including SAG, ASCALE, and ALTER
  • SWEEP command that sweeps an edge/curve into a surface
  • COMPOSITE command that combines several edges/curves into a single edge
  • Linear transformations that shift, rotate, or scale the edge points
