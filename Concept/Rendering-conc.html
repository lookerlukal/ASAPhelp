Rendering

The two types of rendering supported in ASAP are described in this topic.


  1. Static rendering using the RENDER command.
  2. Dynamic rendering using the OpenGL libraries.
Tip

If you want to change the color of an object, you must use one of the color options in ASAP. You cannot change the coloring after the rendering has been generated.

  • Dynamic Rendering
    The ASAP 3D Viewer is linked to the OpenGL libraries for real-time, dynamic rendering of systems and ray traces.
  • Static Rendering
    You can create a rendering of the system geometry by setting the appropriate WINDOW and PIXEL values (and optionally, LIGHT values), and then entering the RENDER command. The RENDER command traces a grid of rays whose density is determined by the PIXEL command through the viewing window.
