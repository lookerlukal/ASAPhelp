Converting 3D Vector Files to AutoCAD DXF Format

The VECT2DXF utility takes a BRO 3D vector file and converts it to POINT, 3DLINE, 3DFACE, CIRCLE and POLYLINE/VERTEX entities in a standard DXF file format.

The DXF file can be read by software compatible with AutoCAD version 11 or above. Each object (color) in the file is assigned to a different LAYER. You have the option of specifying a BLOCK name that will contain all the entities. Otherwise, they are written out as separate entities. Any plotted ray information in the vector file is also exported.
