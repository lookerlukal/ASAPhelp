Scatter Analysis - Common Approaches

This topic addresses common approaches to scatter analysis.


  • Random perturbation of incident ray optical path length and direction:

    - Performed with the ROUGHNESS command

    - One ray in, one ray out

  • Random Lambertian Scatter:

    - Performed with the SCATTER RANDOM command

    - One ray in, one to many rays out

  • Scatter towards important edges or directions:

    - Performed with the SCATTER MODEL and TOWARDS commands

    - One ray in one, to many rays out

  • Lorentzian beam shape assignment:

    - Performed with the SCATTER BSDF command

    - One ray in, two rays out
