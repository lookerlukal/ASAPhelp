BSDF Fit Utility Overview

In most instances where it is appropriate to apply the Harvey model or the Polynomial Fit model, you can use the BSDF Fitting utility, which is integrated with ASAP.

The Harvey model is a mathematical function that describes BSDF as a function of the angle of detection. It also describes the angle of incidence of the beam used for the measurements.

The Polynomial model creates a scatter model based on user-supplied data or polynomial coefficients.
