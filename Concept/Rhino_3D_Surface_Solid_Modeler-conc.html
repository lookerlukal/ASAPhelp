Rhino 3D Surface/Solid Modeler

The Rhinoceros® 3D Surface/Solid Modeler ("Rhino"), from Robert McNeel & Associates, is supported by ASAP CAD+. Using the integrated IGES/ASAP file translator, you can import IGES files you create with Rhino.

For information on Rhino and for minor updates, See the Rhino Web site at www.rhino3d.com.

Technical Support

If you have an inquiry related to the following types of issues, contact BRO Technical Support:


  • Importing an IGES file to ASAP
  • Exporting a Rhino file to an IGES file for use in ASAP
  • Using the IGES/ASAP translator
  • Converting an *.inr file in ASAP
Note:

You do not need to use Rhino as part of the IGES/ ASAP import process. Several CAD packages export IGES files directly into ASAP.

To make trimmed Rhino surfaces in ASAP more efficient; that is, faster raytracing:

Select All your surfaces and apply the ShrinkTrimmedSurface command to them (Surfaces > Edit Tools)
