Dynamic Rendering

The ASAP 3D Viewer is linked to the OpenGL libraries for real-time, dynamic rendering of systems and ray traces.

Typically, this is done with the PLOT FACETS command, which is capable of generating polygon facets for most types of objects. (PLOT EDGES is effective for edge-based objects.) This information, like all 3D data, is stored in the BRO030.dat or *.vcr file. Any ray trace data generated with the TRACE PLOT command is also stored in these files. By issuing a series of PLOT and TRACE commands, you can generate enough 3D information for sophisticated renderings.

Once 3D data are generated, the preferred way to view the rendering is to enter the $VIEW macro command from the Command Input window, or select 3D View from the Graphics menu. This flushes the output buffer, closes the *.vcr file, opens a 3D Viewer (render) window, and opens the *.vcr file as input to the 3D Viewer window. Alternatively, you can open a 3D Viewer window from View > 3D View.

With a series of PLOT and TRACE commands, ASAP can generate enough 3-D information for sophisticated renderings.

Tips
  • When issuing a series of TRACE PLOT commands, ASAP cannot always generate polygon information for objects based upon surfaces/functions, and therefore some objects may be missing from the rendering. For example, nearly planar objects constructed with the OPTICAL command will be rendered properly; a steeply curved one may not.
  • The TORUS surface/function can present problems for the rendering. Depending upon the local and global limits boxes and any bounds that might be present, a torus object may not be rendered either. If objects are missing from your rendering, use the static rendering technique to visualize your system.
  • Shaded renderings often require the display of many colors at once. If the X-windows palette does not have sufficient colors, image quality may suffer. To get the best results with the 3D Viewer window, use a display mode with 65556 colors or true color (24-bit).
Note: The 3D Viewer window may not work properly if the display is in 256-color mode.
