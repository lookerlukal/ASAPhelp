Software Updates

Breault periodically provides product releases to customers with current Maintenance Agreements. We notify all ASAP customers by email when a release is available for download.

In addition to a current Maintenance Agreement, you need an existing hardware key to use the release.

If you need the current ASAP release, please contact Customer Service.

You can determine your support status by opening the Registration dialog on the ASAP Help menu. If the date next to License information has passed, your maintenance plan has expired. Please contact BRO for information on the software maintenance plan.

Release notes for previous ASAP releases are included in Help.
