Sample Project Files

Sample Projects are complete working examples of optical systems built in ASAP, along with brief documentation in the form of text files. Sample project files can be opened in all licensed and demonstration versions of ASAP.

In Windows, you can load a project file from the File menu by selecting Project > Load. To open the folder that contains samples, double-click the Samples folder, and then double-click a specific sample project folder. Locate the project file (*.apf) you want to open and double-click it. A text file (*.txt) containing more information on the project opens with the sample project.
