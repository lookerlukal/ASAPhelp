BRO Light Source Library Overview

The BRO Light Source Library is a growing collection of U.S. and European source models that can be imported directly into ASAP projects.

The library, available to ASAP customers with current software maintenance agreements, includes filament, light-emitting diode (LED), arc, and cold cathode fluorescent (CCF) sources. These sources include detailed geometry and produce near- and far-field patterns that are in agreement with measurements.

Source model geometry is specified by calling the complete list of sub-assembly macros within the library file. The library file also contains source-specific information such as: EMITTING OBJECT assignments, MEDIA, COATINGS, system UNITS, applicable WAVELENGTHS, spectral information (in the case of multi-spectral sources), object count and individual object scatter properties.

Light Source Library files are located in your ASAP installation area under the folder, Projects\LightSource. You can also download the entire source library, including updates and new source files, from the secure BRO server if you have a current maintenance agreement.

The information contained in a BRO source model is extensive. A detailed reconstruction of the light source’s physical geometry is created in a NURBS-based CAD program and then imported into ASAP. Within ASAP, accurate optical properties are applied to the various components of the source model.

For filament based sources, the filament is turned into an emitting object. The characterization of the emissive behavior of the filament is carefully reproduced in ASAP using a flux weighting technique that uses real, measured data to reproduce the power “ramping” that occurs with filaments.

For LED sources, native ASAP emitters are used to reproduce the emission of the LED die. Our LED sources add another level of detail by including the power spectrum data of source. This data can be used to perform detailed color analysis using the colorimetry feature in ASAP (8.0 and later).

Arc sources use a different approach for characterizing the arc emitter. We must first capture an image of the arc using a CCD camera and a zoom lens. This image can be converted into a display file in ASAP and then into a 3D volume emitter by using the Inverse Abel method. The individual ray fluxes are determined by the relative pixel intensities of the CCD image.
