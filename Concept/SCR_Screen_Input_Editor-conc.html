SCR (Screen) Input Editor

The SCR input editor defines a programmable screen template— a user interface, such as a dialog box—for editing input/output (I/O) dialog boxes. The SCR Editor supports the I/O of any ASAP macro.

SCR dialogs can be used to create and assign values to ASAP variables. For example, use an SCR-created dialog to test a scenario of the same script multiple times with different values.

The $SCR macro in ASAP gives rules for designing a dialog box. The SCR Editor automatically applies these rules for the user.

As you create the layout of the dialog box in the left pane of the ASAP - Scr document window, the corresponding ASAP script is automatically created in the right pane.



  • SCR Editor - How to Create a Screen
    You can create the layout of a new dialog box in the left pane of the Scr document window.
  • SCR Checkbox Control Properties Dialog
    Defines the label associated with a checkbox, and the name of a variable associated with the checkbox.
  • SCR Edit Control Properties Dialog
    Dialog box applies to the Screen (SCR) Editor.
  • SCR Static Text Control Properties
    In the Text box, type the text you want to appear in the screen template you are creating.
