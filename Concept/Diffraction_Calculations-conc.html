Diffraction Calculations

The technique ASAP uses for diffraction analysis is named beam superposition. While superposition theory is not an entirely new concept, the combination and implementation of it with real ray tracing is new. The ASAP technique is substantially enhanced to include: effects of higher-order Gaussian beam modes, polarization, and general electromagnetic wavefront propagation, including mixed coherent/incoherent spread functions.

ASAP beam superposition algorithm

The ASAP beam superposition algorithm models an arbitrary optical field with fundamental beams. The fundamental beams are each represented by an apodization function, such as a Gaussian, and a set of real rays describing the apodization function, which may be traced through the opto-mechanical system. This set of rays characterizes the near-field and far-field properties of the beam. It is possible to use the ray trace information from the individual beams to recreate the energy distribution at any position in space. How these fundamental beams interact with each other determines the coherence of the source.

Traditionally, diffraction analyses are done using two very different methods, each with limitations. However, the diffraction analysis option in ASAP does not use either of these techniques and therefore does not suffer from these limitations.

You can derive the pupil function, if enough rays are traced through a system and the optical path differences (OPD) between the rays are calculated. By using Fourier transform techniques on this function, you can get the point spread function (PSF), which in turn may be Fourier-transformed to obtain the system transfer function. This method is probably the most common technique implemented in current optical design software. Consider the following:


  • Sampling of the pupil function and grid spacing requirements in order to perform the fast Fourier transform (FFT)
  • Calculation speed if a discrete Fourier transform is performed
  • Aliasing problems associated with nonperiodic functions
  • Determination of diffraction image on a plane close to focus and nearly perpendicular to the axis of the imaging cone
  • Systems that have a well-defined exit pupil with minimal pupil aberrations
  • Difficulty in doing diffraction calculations in a skew direction

When modeling an optical field by a complex matrix, you can propagate this field using FFT s. Consider the following:


  • Resolution of the sampling process and the need to add guard bands
  • Aliasing of the FFT
  • Inability to accurately model the interaction of the field with complex optical surfaces
  • Different propagation techniques depending upon whether the beam is in the near field or far field
