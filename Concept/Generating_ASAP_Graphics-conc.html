Generating ASAP Graphics

ASAP has the ability to write graphical data to several files simultaneously. These files contain information from which printed copies are made.

Character Graphics

Character graphics are incorporated into the FOR006.DAT (or filename.OTR) file as ASAP is generating them.

Screen Graphics

Screen graphics (or what you see on the screen) are written into the FOR020.DAT (or filename.PLR) file. This file is an ASCII text file containing the (x,y) coordinates (in screen coordinates) of everything written to the screen. End-of-plot record statements separate the individual screen graphics.

3D Vector File Graphics

Almost every 3D data point generated by the graphics commands is written into the FOR030.DAT (or filename.VCR) file. This file is also an ASCII text file but is open ended; as ASAP executes graphics commands, new data is continually being added to this file. Therefore ASAP does not know which graphics command generated which piece of data.

FOR030.DAT/FILENAME.VCR

3D coordinate information

The graphics in the file may be replotted as illustrated in REPLOT so that you may see concatenations of everything in this file, including system models, ray traces, and some analysis outputs.
