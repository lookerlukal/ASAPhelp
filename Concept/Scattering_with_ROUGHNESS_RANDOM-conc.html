Scattering with ROUGHNESS RANDOM

This topic gives several tips for using ROUGHNESS RANDOM, which simulates scatter from a rough surface by randomly perturbing the direction of a reflected or refracted ray. The actual surface of the object is not altered.

Tips
  • Do NOT use this command for coherent calculations. It does not conserve energy in the SPREAD or FIELD commands.
  • One ray in, one ray out.
  • No need to set the number of scatter levels or number of scattered rays.
  • Reasonable model for many transmissive diffusers or diffuse reflectors.
  • Combine with other scatter commands to form a composite scatter model. For example, a bright halo around the specular direction, combined with a Lambertian floor.
  • Modifies objects; therefore, follow with an OBJECT command.
  • Total scatter is controlled with the INTERFACE command.
