Optical Properties

Using ASAP, you can build optical system models by first creating optical property and system geometry databases. The databases have additional commands for direct data entry and commands called modifiers that allow you to alter the databases. As you build an object, you associate data from these databases to the object for ray tracing.

ASAP supports the modeling of four types of optical properties on any object:


  • Transmission according to Snell's law (this includes extraordinary ray propagation within a uniaxial birefringent material).
  • Reflection according to the law of reflection.
  • Grating diffraction according to the grating equation.
  • Scatter according to the BSDF assigned to the object.

Any number of these optical properties may be assigned to an object simultaneously using the IINTERFACE command for transmission, reflection, and grating diffraction, and the SCATTER command for scatter.
