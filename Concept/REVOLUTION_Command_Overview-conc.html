REVOLUTION Command Overview

ASAP has two SURFACE/FUNCTION commands for defining a 2D curve rotated into a 3D surface.


  1. REVOLUTION, when followed by a coordinate axis, a location along the coordinate axis, and a set of polynomial coefficients, specifies a mathematical curve directly; it is spun around the designated axis to create a 3D surface in space.
  2. REVOLUTION (Fitted) when followed by an axis, an order number, and a series of pairs of coordinates, calls the ASAP singular value decomposition (SVD) routine that fits the pairs of coordinates to curve of the specified order and then spins the curve to create a 3D surface.
