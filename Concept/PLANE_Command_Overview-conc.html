PLANE Command Overview

This topic describes the three SURFACE/FUNCTION commands in ASAP that directly define a planar surface.


  1. PLANE specifies the coordinate axis and the location along the axis.
  2. PLANE NORMAL specifies the normal vector to the plane (in global coordinates) and a point on the plane. The point becomes the reference point for the surface.
  3. PLANE POINTS specifies the global coordinates of three points on the surface of the plane. The second point becomes the reference point for the surface.
