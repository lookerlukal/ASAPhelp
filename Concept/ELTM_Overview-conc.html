ELTM - Exterior Lighting Test Module Overview

ASAP Exterior Lighting Test Module (ELTM Module) facilitates the process of testing automotive exterior lighting.

You can use ELTM within ASAP to analyze and test illumination patterns of reflector/bulb geometries for determining compliance with government requirements.

To open the ELTM window in ASAP, click File> New, or click on the ASAP toolbar and select ELTM. Alternatively, in ASAP Workspace, right-click ELTM and select New or Open. ELTM provides an intuitive, three-pane interface:



The test criteria were based on several standards:


  • SAE Ground Vehicle Lighting Standards Manual, SAE HS-34, from the Society for Automotive Engineering,
  • Federal Motor Vehicle Safety Standards (FMVSS), and
  • Economic Commission for Europe (UN/ECE).

The ELTM interface includes a series of dialog boxes that walk you through the test process in the following sequence:


  1. Importing geometry or light source file.
  2. Editing the file, if necessary, in its native editor.
  3. Selecting the applicable test and running it.
  4. Processing data (trace rays and plot mesh, contour plot) automatically in ASAP.
  5. Viewing results in ASAP plot windows.
  6. Saving the test information (including files and settings).

NOTE: A sample project (*.apf file) for ELTM is in the ASAP directory asap install folder\projects\Samples\ELTMheadlamp

  • ELTM - Importing Files
    Using ELTM, you can import your geometry files or BRO light sources from the BRO Light Source Library.
  • ELTM Designer
    To customize printed reports in ELTM (Exterior Lighting Test Module), use the ELTM Designer window to create a template file or edit an existing one.
  • ELTM Results
    The results table in ELTM reports the data processed in ASAP. You can view the results in the Display Viewer window in ELTM.
