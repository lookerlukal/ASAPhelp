Quick Start Toolbar Overview

The Quick Start toolbar in the ASAP Window gives you quick access to catalogs in ASAP.


  1. Sources—light sources, organized by type
  2. Glass Catalog—Glass media that you can apply to ASAP objects
  3. Scatter includes scatter and roughness models that you can apply to objects
  4. Example Files—INR files of isolated and simple commands that show how the commands can be used
  5. Interactive Scripts—a place to access interactive and user-defined scripts
  6. Lenses that you can apply to objects


The toolbar can dock to the left, right, or bottom of the ASAP main window. It can be shown or hidden by selecting the Quick Start Bar on the View menu. When you start ASAP, the bar opens in the button-only view, and expands to a tree pane when you click one of the toolbar buttons. To close the tree pane, click the associated button. You can change preferences for this toolbar on the Quick Start tab of the User Interface Preferences.

  • Quick Start Toolbar - Sources
    The Sources button on the Quick Start toolbar opens the Sources page--a catalog of all the sources available on your system.
  • Quick Start Toolbar - Glass Catalog
    The Glass Catalog button on the Quick Start toolbar opens the Glass page—a list of all the available glass catalogs: French, Hikari, Hoya, Hoya New, Ohara, Schott, Schott New, Sumita, and Unusual. You can also add your own glass definitions to the User catalog.
  • Quick Start Toolbar - Scatter
    The Scatter button on the Quick Start toolbar opens the Scatter page—a list of available scatter and roughness models that you can apply to objects.
  • Quick Start Toolbar - Example Files
    The Example Files button on the Quick Start toolbar opens the Example Files page—a tree list of examples for ASAP commands. Example files are stored in your ASAP installation folder under projects\Examples.
  • Quick Start Toolbar - Interactive Scripts
    The Interactive Scripts button (formerly named the User INR Scripts) on the Quick Start toolbar, opens the tree view and gives you quick access to interactive and user-defined script files. You can add, remove, open, and run example interactive script files and user-defined script files.
  • Quick Start Toolbar - Lenses
    The Lenses button on the Quick Start toolbar opens the Lenses page—a catalog of pre-defined lenses available in ASAP. Hovering the mouse pointer over a lens displays a brief description of the lens, such as focal length, shape, and diameter.
