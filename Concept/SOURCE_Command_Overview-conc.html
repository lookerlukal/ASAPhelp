SOURCE Command Overview

The SOURCE command establishes the ray propagation direction and its initial optical path length. Use it in conjunction with GRID commands, which establish the initial ray coordinates. You can also use this command to create grids of sources.

ASAP has five options under the SOURCE command:


  1. POSITION specifies the coordinate to which the ray trajectories are to diverge; it sets up a diverging spherical wavefront
  2. FOCUS specifies the coordinate to which the ray trajectories are to converge; it sets up a converging spherical wavefront
  3. DIRECTION specifies a propagation direction vector; it sets up a planar wavefront
  4. LINE specifies the foci of an ellipsoidal wavefront, and is useful for creating a cylindrical wavefront
  5. WAVEFUNC specifies a wavefront function or Eikonal based upon a previously defined surface or function
