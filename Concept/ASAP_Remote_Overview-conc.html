ASAP Remote Overview

Remote is a feature of all ASAP products except ASAP Lite. To use it on a client machine, you must have a valid license.

If you are installing Remote on a remote server machine, only the client machine needs to have ASAP installed. However, to allow your machine to be used as a remote server, you must have selected "Remote Setup" during the installation on the Setup dialog box.

You can extend the power of ASAP in real time by spawning and running the Remote in the computational kernel on one or more computers on the same local area network (LAN). Access to LAN computers depends on an established protocol, which includes the client (local) user receiving permission from remote ASAP users to access their computers as servers. The client collects data computed on remote computers. Files reside on remote computers for the duration of the session, and can be copied to the local computer.
