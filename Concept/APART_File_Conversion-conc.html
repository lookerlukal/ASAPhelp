APART File Conversion

APART® (from Breault Research Organization) includes two conversion programs for converting system data stored by the first APART program on its BASICA (TAPE10) file into ASAP input.

The AP2RBS program translates all objects into the smooth, continuous SURFACE input to ASAP. AP2RBE, on the other hand, generates faceted objects described by EDGES. However, you can combine the outputs from these two programs to form a single input to ASAP.

Both programs prompt you for two file specifications. The first is the APART BASICA file. This specification assumes that APART is also licensed on the same computer system. The second is the desired name for the ASAP input file that will be generated. The programs then ask if you want to keep the original object numbers as assigned in APART, or simply renumber the objects sequentially.
