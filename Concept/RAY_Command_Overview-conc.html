RAY Command Overview

This topic describes functions of the RAY, RAYS, or RAYSET command.


  • RAYS n resets the number of rays currently defined in the virtual.pgs file to n . If n is 0, this command essentially resets the ray data.
  • RAY x y z. . . specifies and trace an individual ray through the system geometry.
  • RAYS { X, Y, Z }. . . creates a user-specified distribution of rays; you specify the plane in which the rays are to be created, and then the individual ray coordinate pairs.
Note: The RAY command can be used in place of the GRID command.
