ASAP Workspace

ASAP Workspace includes a list of all available document windows in ASAP. ASAP Workspace reflects whichever window you have in focus. Right-click any of the listed windows to open a pop-up menu (Open, New, and Close All).



The ASAP Workspace window initially displays status lists on two tabs, Views and Objects. If it does not open, click on the toolbar.

Viewing ASAP Windows

The Views tab lists all the available ASAP windows. As you open files associated with any of these windows, ASAP Workspace lists the open file(s) below the window name with which the file type is associated. You can see which files are open, and bring a hidden window into focus by clicking the file name on the Views tab.

Right-click a file name to show the system menu for that document; for example, Restore, Size, or Minimize. If your objects are named with a hierarchical schema, you can set the selected child objects for ray tracing and plotting (called the considered state) by clicking a parent item on the Views tab.

Right-click a main group item (bolded). Select Open to open an existing file, New to open a new document window, and Close All to close document windows of the type highlighted.

If you have more than one Display Viewer window open, right-click Display Viewer in ASAP Workspace and select Tile to view all open Display Viewer windows in one dialog.

Including Objects

The Objects tab lists all the objects associated with an open file. The check box next to each object is for designating which objects to include (CONSIDER command) in an operation. The status of these boxes determines if ASAP should use an object or ignore it in subsequent analyses. Objects are considered when they are created, but can be turned off when a detector or specific object's properties are concerned. You can also view the current considered state in the Command Output window by entering CONSIDER in the Command Input window. Highlight one of the objects on the list and right-click anywhere on the tab to open a pop-up menu. Select Export DXF or Export IGES.

Displaying a Project File

If you choose to define an ASAP project, you will be able to manage multiple files of many types more easily and efficiently. Frequently used files can be organized in a simple index that becomes part of the ASAP Workspace window. Any file can be opened for editing from this file list, or run without opening once you are finished developing the file. You have the option to link multiple files together, allowing them to run in a prescribed order.


  • The Files tab is displayed in ASAP Workspace only upon opening a project.
  • Click a file name in the list to open it in its associated document window (for example, Builder, Editor, or Plot).
  • To add an existing file to the project, select the project path at the top of the file list, right-click, select Add Files to Project. Browse to the location of the file you want to add and double-click its name. Move files up and down the list by dragging them to the location you want them.
  • Right-click in the background area of the pane to add files to the list of active files, or to initiate Autorun.
  • Right-click over a specific file to open for editing, running, or removing the file from the list, or setting up an Autorun chain.
  • The right-click menu includes commands for Open, New, and Close All. The Display Viewer window type has a right-click menu that includes the Tile command for viewing all or selected Display Viewer windows in tile mode. When Tile is selected, the Tile dialog box opens.
Dragging from the Quick Start Toolbar

Media and models can be dragged onto objects in ASAP Workspace from the Quick Start toolbar after creating geometry with the Builder or a script file.

  • Projects in ASAP
    ASAP uses a file management concept called projects. For easy viewing and quick access, you can organize each project by linking all the files associated with it into a project file. The ASAP project file (*.apf) tracks which files are associated with a project, and the directory in which they are stored.
  • Project Properties Dialog
    This dialog box is available from either File> Project> Properties, or by right-clicking a project name in ASAP Workspace.
  • Sample Project Files
    Sample Projects are complete working examples of optical systems built in ASAP, along with brief documentation in the form of text files. Sample project files can be opened in all licensed and demonstration versions of ASAP.
