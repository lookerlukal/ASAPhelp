Importing/Exporting
Importing/Exporting

You can import or export several types of files, either those generated by ASAP or those used as inputs to ASAP.

Importing and exporting capability is determined by the applicable ASAP configuration.

  • BRO Light Source Library Overview
    The BRO Light Source Library is a growing collection of U.S. and European source models that can be imported directly into ASAP projects.
  • APART File Conversion
    APART® (from Breault Research Organization) includes two conversion programs for converting system data stored by the first APART program on its BASICA (TAPE10) file into ASAP input.
  • CAD Library
    The BRO CAD library (cad.lib) contains macros for turning common ASAP surfaces into parameterized (EDGE/CURVE-based objects).
  • Converting BMP Bitmap Files to/from Distribution Data
    Two utilities are available in ASAP for converting bitmap files.
  • Hewlett-Packard File Utility (HPGLNP)
    The HPGLNP.exe utility reads ASAP PLR (plot) files and creates Hewlett-Packard files for plotting.
  • Importing a ReflectorCAD File
    You can use ASAP to check the output of a reflector you created in ReflectorCAD. Follow instructions in ReflectorCAD Help for changing appearance-related settings in ReflectorCAD for output appearance and contouring.
  • PostScript File Utility (PSCRIP) (ASAP Command)
    The PSCRIP.exe utility reads ASAP PLR (plot) files and creates PostScript files for printing. The utility is stored in your ASAP installation area under the \bin folder.
  • Radiant Sources
    Radiant sources are 3D source characterization files created by Radiant Imaging, Inc. Radiant sources are generally obtained for one-of-a-kind sources, specialty bulbs, or other sources, which are not in the BRO Light Source Library.
  • SolidWorks Parts Only 3D Modeling Engine
    ASAP CAD+ and ASAP ADVANCED include a license of the SolidWorks® Parts Only 3D Modeling Engine. With these versions of ASAP, users may write ASAP GTX files from within SolidWorks, import XML files, or use BRO's proprietary smartIGES™ system to import system models from any CAD package.
  • SolidWorks Files
    If you have SolidWorks (from SolidWorks Corporation) installed and you have ASAP CAD+ or ASAP ADVANCED, you can export geometry files from SolidWorks to ASAP.
  • Export to CAD Dialog
    Writes a CAD-compatible system geometry file. Export includes surface representation of all lenses, surface/sampled, and continuous curve- and edge-based objects.
  • Exporting Files to DXF
    The three methods in ASAP for exporting geometry (LENS, SURFACE/SAMPLED, and continuous CURVE/EDGE-based objects) to DXF are described in this topic.
  • CATIA Files, Importing in ASAP
    You can import CATIA files via the CAA V5 plug-in included with ASAP CAD+ if you have CATIA (a product of Dassault Systemes) installed and you have ASAP CAD+.
  • CODE V File Conversion
    ASAP OPTICAL+ and ASAP ADVANCED include the fully integrated capability of importing CODE V® files.
  • Translator Overview for IGES and GTX
    ASAP CAD+ includes IGES and GTX (XML) import capability, by translating IGES (Initial Graphics Exchange Specification) files and XML (eXtensible Markup Language) in GTX format to ASAP format. You can perform optical analyses on geometries initially created in a CAD program.
  • OSLO to ASAP Translation
    ASAP OPTICAL+ and ASAP ADVANCED include the fully integrated capability of importing OSLO® files. You can open this file type (*.LEN) in ASAP from the Open dialog box on the File menu.
  • ZEMAX Translator Overview
    ASAP OPTICAL+ and ASAP ADVANCED include a ZEMAX-to-ASAP translator for opening ZEMAX files in ASAP.
