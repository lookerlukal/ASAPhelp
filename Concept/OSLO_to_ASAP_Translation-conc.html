OSLO to ASAP Translation

ASAP OPTICAL+ and ASAP ADVANCED include the fully integrated capability of importing OSLO® files. You can open this file type (*.LEN) in ASAP from the Open dialog box on the File menu.

  • OSLO to ASAP Translation Step 1
    The first step in the OSLO to ASAP translation process involves selecting a *.LEN file produced by OSLO.
  • OSLO to ASAP Translation Step 2
    The second step in the OSLO to ASAP translation process is to select the ASAP filename, and set several translation options.
  • OSLO to ASAP Translation Step 3
    The final step in the translation process is to choose how the translator deal with refractive indices. (In the following discussion, the term “glass” refers to any refractive material including glass, plastics, crystals, and other refractive materials.)
  • OSLO Restrictions
    This topic describes specific features and limitations of the OSLO to ASAP translator.
