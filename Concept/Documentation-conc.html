Documentation for ASAP Users

The BRO Knowledge Base provides Adobe Acrobat PDF files for on-line display, saving, or printing of ASAP technical publications and user guides. You can install documentation directly from the BRO ASAP CD or download the most up-to-date versions of documentation from the BRO Knowledge Base.

ASAP Getting Started Guide (BRO Knowledge Base)

The ASAP Getting Started Guide includes a quick tour of ASAP, an introduction to the user interface, installation information, and importing/exporting tools.

ASAP Primer (installable from BRO Knowledge Base)

The ASAP Primer is an introductory book that instructs new users on the philosophy and use of ASAP, and includes working examples.

ASAP Reference Guide

The ASAP Reference Guide includes descriptions of all the ASAP commands and macros, including overview information. The content was derived from the ASAP online Help.

ASAP Documentation (BRO Knowledge Base)

ASAP technical publications and white papers are available on the BRO Knowledge Base.

Tip

To open a PDF file, you need the Adobe Acrobat Reader installed on your machine. If you do not already have it, you can download the free Adobe Acrobat Reader from the Adobe Web site, http://www.adobe.com.
