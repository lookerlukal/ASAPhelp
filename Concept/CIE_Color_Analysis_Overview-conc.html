CIE Color Analysis Overview

International Commission on Illumination (CIE) color analysis tests for uniformity of projected light on a surface. CIE numerical and color analysis in ASAP is ideally suited for projection and display customers. Although ASAP has been able to track multiple wavelengths for light rays, CIE visualization mechanism makes it possible to determine their positions in the CIE color coordinate space.

CIE color analysis in ASAP includes these main features:


Analysis of CIE Color OutputTest for uniformity of projected light on a surface. Output is displayed in the CIE window and includes CIE chromaticity plots.
Customized Data Display and CalculationsSelect and order individual data items in the CIE Color output display window, as well as select individual calculations for speed purposes.
Rendering Color IntensityView the color intensity of your selected ray data.
CIE color analysisTests for uniformity of projected light on a surface. The CIE Color Analysis dialog box is accessible from the Analysis menu in ASAP.
