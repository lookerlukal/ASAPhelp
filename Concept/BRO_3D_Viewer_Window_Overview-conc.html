3D Viewer Overview

The BRO 3D Viewer is a powerful, often essential visualization tool for dynamically manipulating geometry, rays, and analysis results. With the 3D Viewer, you can visualize your model and its performance.

While ASAP generates 2D plots in the various plot windows, it also stores full 3D plot data in a vector (*.vcr) file. To view the contents of the current vector file, select 3D View from the System menu. This launches the 3D Viewer window, and opens the current vector file (*.vcr) in the 3D Viewer.

  • 3D Viewer Uses
    The BRO 3D Viewer™ displays ASAP geometry data that is stored in the vector (*.vcr) file.
  • 3D Viewer Window
    The BRO 3D Viewer window consists of a navigation pane (tree view, left), graphics pane (object view, right), and a toolbar (top). This topic describes these window elements.
  • Polar Plot Window in 3D Viewer
    Right-click the file name in the Tree pane of the 3D Viewer window, and select Polar Intensity Plot to open the Polar Intensity Plot dialog.
  • Axes Preferences Dialog
    Controls the appearance and placement of axes in the 3D Viewer.
  • 3D Distribution File Optional Properties Dialog
    Select coloration, cutoff, and colors for 3D Viewer window.
  • Display File Optional Properties Dialog
    Sets the color for specific *.dis files. To access the Display File Optional Properties dialog box, right-click in left tree pane and select Optional Properties.

  • Save File Optional Properties Dialog
    Specifies the log cutoff and color scheme in 3D Viewer graphic window.
  • Sampled (MAP) Surface Optional Properties Dialog
    Specifies faceting and data display properties in 3D Viewer.
  • User Interface Preferences Dialog
    Sets the default viewing preferences in ASAP.
  • Polarization - Poincare Sphere Visualization Overview
    The purpose of the ASAP polarization feature enhancement is to provide a Poincaré Sphere Visualization Tool (PSVT) via the 3D Viewer for visualizing the polarization state of rays.
