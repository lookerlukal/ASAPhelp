MOVE Command Overview

The MOVE command operates on the currently defined ray data by translating or moving the ray data along each ray's trajectory according to some criteria. These criteria include the following commands:

MOVE TO/BY. . . moves the ray data TO a specific coordinate or BY a specific amount.

MOVE TO SPHERE/MOVE TO PLANE moves the ray data to a specified spherical or planar surface.

MOVE TO FOCII moves each ray or beam to the centroid of the intersections of the specified parabasal rays with the base ray.
