Tree View

The hierarchical tree of objects occupies the left pane of the 3D Viewer window. ASAP object names are used to separate levels in the object tree list.

At the top level, files are displayed, and below them, the objects they contain are displayed. For ASAP object names (*.vcr files only), dots indicate levels.



A context menu of commands can be accessed by right-clicking an object name.

The Polarization Marker Legend and Polarization Flux Legend are displayed in the Tree View pane when the current file has produced a Poincar√© sphere in the graphics window.

The Marker legend is selected in the User Preferences dialog for the 3D Viewer. It displays markers for varying wavelength values.

The Flux legend displays a legend of colors for varying flux values.
