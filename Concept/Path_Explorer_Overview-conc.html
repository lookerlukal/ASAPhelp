Path Explorer Overview

The Path Explorer renders a three-dimensional, interactive visualization of specifically selected ray trajectories, or paths, computed in the course of ASAP simulations. This feature is useful for isolating rays that are doing particular things, such as scattering or ending on a selected object(s).

Exploring ray history files

The Path Explorer in ASAP supports the exploration of ray history files via queries that you create. You can create queries using the Path Explorer dialogs within the ASAP User Interface or by placing HISTORY subcommands within an ASAP script editor file.

Isolating ray paths

Ray paths can be isolated that start from objects, divide at objects, split a number of times, or hit objects a number of times. Ray paths of specific scattering levels and generations of splitting also can be isolated. Ray paths can be found with intersections of objects or volumes.
