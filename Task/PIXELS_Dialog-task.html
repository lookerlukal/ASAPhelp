PIXELS Dialog

Sets the resolution for graphics and calculations.


PIXELS Resolution:Divides the window into a specified number of pixels.
Vertical Elements in Window Number of equally-spaced pixels along the vertical axis of the plotting window (default = 39)
Aspect Ratio (Horizontal / Vertical)Ratio of pixels per system unit along the horizontal axis to pixels per system unit along the vertical axis.
Plot Sample BoxIf checked, plots a pixel sampling.
