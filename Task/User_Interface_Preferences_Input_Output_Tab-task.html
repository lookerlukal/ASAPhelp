User Interface Preferences - Input/Output Tab

Describes global preferences for command input and output operations.

Input Window Settings:


Return focus to Input WindowIf selected, controls whether the Command Input window returns to the active state after other windows appear (for example, Plot windows). Returning focus to the Input window is useful when doing a lot of input from this window.
Clear Error/Warning Counters on END If selected, clears counters after an END command is issued (either via the Command Input window or via the toolbar button).
Suppress Beeps If selected, makes the error/warning beeps inaudible.
Command History Length Specifies number of lines of output to store (must be greater than 0).
Output History LengthSpecifies number of lines to store in history file.
Clear HistoryDeletes stored lines of output.

Output Window Settings:


Output FontDisplays current font for Command Output window.
Change Font Lists available fonts for the window.
Text Colors Dropdown box lists text types for the Command Output window. Text types include Command Echo, Errors, and Warnings, among others.
Palette For the selected text type, the button displays available colors, and the adjacent box displays its current color.
Show Output (Current ASAP session only) If selescted, displays output in Command Output window. Turns on automatically with each new ASAP session.
Generate Links to Plots in Command Output WindowIf selected, you can open a specified plot window from a link in the Command Output Window.
Expand Views Workspace Tab Tree Control If selected, expands the tree on the Views tab of ASAP Workspace to show all current files associated with each active window.
Update Files in Explorer Pane If selected, the Explorer pane reflects all the current files in the Working Directory on your machine.
