Quick Start Toolbar - Example Files

The Example Files button on the Quick Start toolbar opens the Example Files page—a tree list of examples for ASAP commands. Example files are stored in your ASAP installation folder under projects\Examples.

To view a description of each example file, hover your mouse over the file name.

To open an example file in the ASAP Editor window, double-click the file name on the Example Files page of the Quick Start toolbar.

The list is separated by Categories, Commands, JScript, Keywords, and VBScript.


Group Script files listed by...
JScript Examplestitle of JScript example scripts
Scripts by Category title under subject category (for example, "Analysis")
Scripts by Command command name and then by title of example script
Scripts by Keyword keywords and then title of example scripts
Scripts by Name file name of ASAP example scripts
VBScript Examples title of VBScript example scripts

Note:
  • "File name" refers to the name of the INR file; whereas "title" refers to the assigned title in the header at the top of the INR file.
  • Example script files may be added, revised, or deleted with each release of ASAP.
  • Problem scripts in the Commands group typically involve one command; whereas, problem scripts in the Categories group involve several commands.
  • JScript and VBscript examples are also included (see Scripting Language Syntax). For more information on other supported scripting languages, see Scripting Engines and PythonScript.
  • Click the example name to open the script in the Editor window, where you can view, run, or copy it.
  • When you run an example script that has the $READ macro in it, the file that $READ opens must be in the current working directory.
  • The example script files, which are located in asap install\Projects\Examples, are read-only files to prevent overwriting them. However, you can the script to a different file name and location.
