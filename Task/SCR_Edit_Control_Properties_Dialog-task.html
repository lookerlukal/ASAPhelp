SCR Edit Control Properties Dialog

Dialog box applies to the Screen (SCR) Editor.


Label Type the text of label and enter the monospacing between the label and the check box Maximum screen size is 79 x 24 characters.
Variable Name Enter the name you are assigning the variable, and specify the width of the name in monospaced characters.
Variable Type Select type from the drop-down box.
Read Only Select if the output file is read-only.
