Show Ray History Dialog
Show Ray History Dialog

Recalls ray data from a previously saved binary file for plotting or listing.

Pre-requisite:

This dialog works only when the SAVE command is issued before the TRACE command.


Save File NumberSaves the specified file number from the previous SAVE command.
List Reverse Ray Trajectories Lists the reverse trajectories of the current ray set.
Reverse Ray Trajectories Plots the reverse trajectories of the current ray set.
Spots without Trajectories Plots only the spots where the rays struck the objects.
Plot Options/Title Adds user-designated title to plot (up to 64 characters). Does not apply to List Reverse Ray Trajectories.
Override Next PlotNext plot created is overlaid with the current plot.
Override Color Applies the selected color to all plot lines, rather than default mode of multi-colored plot lines.
