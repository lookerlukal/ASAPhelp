Displaying Graphics in 2D or 3D

ASAP has complete 2D and 3D graphics capabilities that include overlays, oblique views, annotation, static and dynamic rendering, zoom in/out, interactive point-and-shoot ray tracing, and many other features.

Graphics commands are divided into two broad categories:


  1. Setting defaults, operating status, options (for example, PIXELS, WINDOW, OBLIQUE).
  2. Creating graphical output (for example, SPOTS, OPDMAP).
Note: The key to creating sophisticated graphics in ASAP is to understand the relationship between the various types of graphics commands and the data itself.

Generally, you can follow this procedure to create graphics:


  • Isolate your data (objects or ray data) using the CONSIDER and/or SELECT commands.
  • Set windows orientation, sampling and others.
  • Enter the specific graphics command: CONTOUR, DIRECTIONAL, DMAP, ENCLOSED, GRAPH, ISOMETRIC, MESH, PLOT3D, RADIAL, and/or RANGE.

If the desired graphic is an overlay (or composite) of several individual graphics, repeat this process for each one.
