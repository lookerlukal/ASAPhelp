3D Viewer faceting and display properties
Sampled (MAP) Surface Optional Properties Dialog

Specifies faceting and data display properties in 3D Viewer.


FacetingMove slider between "Fewer Facets" and "More Facets"
Data Display:
Show actual data as texture map on surfaceSelect box to display map texture.
Data Color SchemeSpecifies the colors used when drawing output as raster image. Pseudo-Spectral is the default. To reverse the order of the list from top to bottom, select Reverse order .
