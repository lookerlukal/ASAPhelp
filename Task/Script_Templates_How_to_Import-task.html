Script Templates: How to Import

You can import a pre-defined script template into your script (*.inr) file in the Editor window.

You must have an Editor window open to access the script template command.

The pre-defined scripting templates contains high-level examples of ASAP commands used for creating scripts (*.inr files). These examples include system definitions, material property definitions, geometry definitions, source definitions, trace ray operations, and analysis results.

To import a template, select a template listed on the Import Template dialog box. The content of the pre-defined template is inserted into your INR file.

The following script is an example of an ASAP pre-defined template.

BASIC Template.INT
 
SYSTEM NEW
RESET

!! Define system units and wavelengths

UNITS MILLIMETERS 'Watts'
WAVELENGTH 550 NM

!! Define coatings

COATING PROPERTIES
  0 1 'TRANSMIT'
  1 0 'REFLECT'
  0 0 'ABSORB'

!! Define Media

MEDIA  
  1.5 'GLASS'

!! GEOMETRY DEFINITIONS BEGIN HERE


!! SOURCE DEFINITIONS BEGIN HERE


!! TRACE RAYS WITH OPTIONAL PLOT

!!WINDOW Y X
!!PLOT FACETS 0 0 -1 OVERLAY
TRACE !!PLOT 

!! ANALYSIS BLOCK BEGINS HERE
