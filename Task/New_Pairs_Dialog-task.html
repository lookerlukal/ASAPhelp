New Pairs Dialog

Define a new glass using data pairs. The New Pairs dialog box opens from the Glass Editor.


Glass NameEnter the name of the new glass.
Wavelength and Refractive IndexAssign index pairs by entering the values in the corresponding boxes.
AddAdds the pairs to the grid. Delete items from the grid by highlighting the item and pressing the Delete key on your keyboard.
OK Adds the glass to the “user.cat” file and closes the dialog.
CancelCloses the dialog without creating the glass.

Restrictions for entering pairs data:


  • Up to 13 pairs can be added.
  • Values must be numeric.
  • The glass cannot be created unless wavelength/refractive index pairs are specified.
  • A wavelength must have a corresponding refractive index in order to be added to the list.
