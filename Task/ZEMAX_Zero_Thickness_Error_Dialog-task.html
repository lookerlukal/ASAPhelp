ZEMAX Zero Thickness Error Dialog

The Zero Error dialog box is displayed when a surface in the ZEMAX design has zero thickness. ASAP may have ray-tracing problems when two objects are defined to occupy the same location in space.

Note: In the real-world application, two objects cannot occupy the same space. This dialog box allows you to take one of three courses of action. In multi-configuration system this dialog may appear for each configuration.

Correct ZEMAXThis option ends the translation. BRO recommends you use this one, since it keeps the ZEMAX and ASAP models of the lens design identical to each other. The text in the dialog box explains your options, which vary depending upon the type of error.
Translator Correction 1Instructs the translator to tag the surface with zero thickness as an artificial surface. Artificial surfaces are surfaces that have the same optical medium on both the left and right sides of the surface. The translator does not translate artificial surfaces into the ASAP model. Caution: This option may cause an incorrect translation of the ZEMAX design into ASAP.

Translator Correction 2

from 0.0 to _ mm

Allows you to add a spacing change between the two ZEMAX surfaces. The default value is 0.00001 (10 nm if the units are set to mm). When you use the default value for the spacing between the two surfaces, the differences between the ray traces in ZEMAX and ASAP are insignificant.

This option activates the OK button. Click OK to continue the translation.
