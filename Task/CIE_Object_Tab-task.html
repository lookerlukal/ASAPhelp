CIE Object Tab

The CIE Object tab is part of the CIE Color Analysis dialog. Use it to designate settings for the object.


Object Identification Use the current considered object(s) in the ASAP kernel, or select a specific object for color analysis.
Use Current Object Selection Select to force the CIE calculation to use the current CONSIDERed object(s) in the ASAP kernel. Deselect when only a single object can be used for the calculation, and then select an object from the drop-down box.
