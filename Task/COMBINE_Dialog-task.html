COMBINE Dialog

Combines the current distribution data file with a previously calculated distribution data file. Used for analysis of general wavefronts.

Display: Combine Distribution Data Files
Files To Be Combined With Current Data:
File NameSpecifies the previously calculated data by its name (with extension *.dis).
Unit NumberSpecifies the previously calculated file by its logical file number (broxxx.dat).
Action:
Multiply Current and Incoming DataMultiplies the two distributions together.
New Data = Current Data + Scale x Incoming Data Adds current data to scale factor, times the incoming file data.
