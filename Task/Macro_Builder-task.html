Macro Builder

The Macro Builder is a user tool for creating macros from the Builder window.

The Macro Builder provides the following capabilities from the ASAP Builder window:


  • Defining, modifying and deleting user-defined macros

To open the Macro Builder dialog, select Macros on the Builder menu or click the button on the Builder toolbar.


Macro NameEnter a name for the new user-defined macro. If the name already exists, an error message warns you that the existing macro will be replaced.
Number of ArgumentsEnter the number of argument prompts you will include.
DescriptionEnter a brief description for the new macro.
Create Opens the Command Editor window, where you can enter the macro script. Click OK to accept the changes or Cancel to close the window without changes.
EditOpens the Command Editor window, where you can modify the macro script. Click OK to accept the changes or Cancel to close the window without accepting changes.
DeleteDeletes the selected macro name from the Macro Builder list.
CancelCloses the Macro Builder without accepting changes.

All user-defined macros are stored in XML file format, ASAPUserDefinedMacros.xml.

User-defined macros can be added to the User INR Script page of the Quick Start toolbar for quick access.

Running a User-defined Macro


  1. Click the first column of any row in the Builder window.
  2. Enter Usermacro in the cell.
  3. Tab to the next column in the row and select a macro name from the drop-down box.
  4. Enter the arguments needed for the selected macro.
  5. Click the Run button on the Builder toolbar.
