CAD Export Log File

Each time you initiate an IGES export to a CAD file, information for the export is logged to a file.

An informational message is sent to the ASAP Command Output window. A report for the IGES export in the form of a *.IgesLog file is sent to the directory that the *.igs file was written to. A new log file is generated for each export.

The message log contains the following severity types:


InfoContains information about the export process or entity being processed.
WarningA problem was encountered, but the export process for the current entity can continue.
ErrorA problem was encountered that stopped the current export processing.
