ASAP Menus

The ASAP menu toolbar includes: Command Input and Command Output windows, Quick Start toolbar, Custom toolbar, and Status bar.


MenuPurpose
File Controls creating, opening, running, and closing ASAP input (*.inr) files; working with ASAP projects; setting preferences; setting your working directory; and exiting the program.

From the Open Files dialog box, the Open Mode dropdown box specifies the ASAP document window in which to open a selected filed in ASAP. You can set a window type as the default.

View Controls display of the dockable toolbars, ASAP Workspace, and dockable windows.
System Opens the Builder; lists media, coating, and models; plots facets, profiles, and edges; opens the 3D Viewer (requires *.vcr file).
Rays Includes ray commands.
Trace Includes trace commands.
Analysis Displays ray trace data and statistical output; selects and modifies specific ray trace data.
Display Opens and saves distribution data files; displays data; processes and modifies data.
Help Opens ASAP Help. Includes About ASAP, Registration, commands, features, user interface, importing and exporting, and more.

As a new document is opened, the main menu changes dynamically to include buttons relevant to the new document window. These changes are covered in the appropriate menu Help topics.
