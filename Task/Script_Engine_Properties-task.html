Script Engine Properties

This dialog box configures the properties for a script engine that will be used in the ASAP Editor.


Script Engine Name Enter the name (for example, PerlScript)
Script Syntax File Click the browser button and locate the *.stx file in your ASAP installation area under, \bin\stx
Script File Extension Enter the file extension (for example, PL for PerlScript)
Script Case Select a case from the drop-down list.
  • Scripting Engines - Add or Remove
    To add or remove a script language engine for use in the ASAP Editor, use this dialog.
