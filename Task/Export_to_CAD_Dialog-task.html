Export to CAD Dialog

Writes a CAD-compatible system geometry file. Export includes surface representation of all lenses, surface/sampled, and continuous curve- and edge-based objects.

This dialog supports the IGES.igs (the default) and AutoCAD.dxf file standards.

The output is a CAD-compatible system geometry file.

The file name is followed by a *.igs or *.dxf extension.

Note:

Trimming information (LIMITS, BOUNDS) is translated only for IGES.

Conditions for Exporting

To export geometry from ASAP, the following conditions must be met:


  • An INR file containing ASAP geometry definitions must be open in ASAP and run.
  • At least one geometry OBJECT in the CONSIDER state must exist.
  • System UNITS for DXF must be set before exporting to CAD. (System UNITS for IGES defaults to MM (millimeters)).

If all the above conditions are met, the geometry export can proceed by selecting Export to CAD on the System menu in ASAP.
