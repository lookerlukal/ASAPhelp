CIE Add Dialog

The CIE Add dialog is part of the CIE Color Analysis. Use it to define a custom profile for display and calculation.


Profile NameEnter the name of the new calculation/display profile.
Available CalculationsLists available calculations, including CIELAB, RGB, and RGB (Intensity). When a display item is selected that requires a calculation, the calculation is automatically selected on this list.
Available Display ItemsLists data items available for viewing in the output display window, including Cartesian Coordinate X, Y, Z; Chromaticity (1930) x,y; CIELAB a*, b*, L*; Delta Eab*, RGB B, G, R; and Tristimulus X, Y, Z.
Up arrow buttonSelect to move up the selected item in the Available Display Items list. Movement changes the order of the results in the CIE Results Table.
Down arrow buttonSelect to move down the selected item in the Available Display Items list. Movement changes the order of the results in the CIE Results Table.
NoteEnter text you may want to associate with this profile.
OK buttonSelect to save changes and exit the Add dialog.
Cancel buttonSelect to exit the Add dialog without saving any settings.
