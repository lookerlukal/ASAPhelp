CATIA Files, Importing in ASAP

You can import CATIA files via the CAA V5 plug-in included with ASAP CAD+ if you have CATIA (a product of Dassault Systemes) installed and you have ASAP CAD+.

Specifically, you can import files of type *.gtx, *.catpart, and *.catproduct via the Windows Open dialog. Once you select a file of one of these types, a CATIA Setup Parameters dialog opens.


  1. Click the Geometry Filter button and then select one of the supported CATIA filters: Main Solid, All Solids (default), or All Surfaces.
  2. Click the Help button in the Setup Parameters dialog to open the associated help topic, which explains each option in this dialog.

The Setup Parameters dialog box opens on top of the Translator dialog box, unless Do not show this dialog is selected on this dialog or in ASAP Preferences on the Translators tab.
