Exporting Files to DXF

The three methods in ASAP for exporting geometry (LENS, SURFACE/SAMPLED, and continuous CURVE/EDGE-based objects) to DXF are described in this topic.


  • Click Export to CAD on the System menu to activate the Exporting to CAD dialog.
  • Use the CAD command; alternatively, you can do the following:
  • Use the utility, VECT2DXF.exe (located in your ASAP installation area in the \bin) folder to export a vector (*.vcr) file. This file type is initially created by a PLOT FACETS or VUFACETS command, and is typically viewed in the 3D Viewer. This method takes individual faceted objects and translates them to individual DXF objects. The utility, VECT2DXF.exe exports all objects in the VCR file (including rays). The faceting level defines the number of 3D Faces created for the surface in the DXF file.
Note:

Due to the 3D Faces and mesh limitation of DXF, BRO generally recommends against using DXF for optical surfaces, such as lenses or reflectors.

Since the file, asaptemp.vcr is under ASAP control, you cannot convert it until the file is renamed; for example, rename it to system1.vcr. However, you can close the file with the syntax, $IO VECTOR CLOSE.

ASAP exports all AutoCAD DXF surface definitions (and BLOCKS) - up to and including AutoCAD release 12 - to ASAP CURVE-based objects, and writes them to *.inr and *.lib files of the same name.

  • DXF/AutoCAD File Conversion
    ASAP lite includes a bidirectional DXF/CAD (digital exchange format/computer-aided design) conversion up to DXF version 12. The DXF2ASAP utility (vect2dxf.exe) reads an AutoCAD file (with a DXF extension), converts all surface definitions (and BLOCKS) into ASAP CURVE-based objects, and writes them to INR and LIB files of the same name.
  • Converting 3D Vector Files to AutoCAD DXF Format
    The VECT2DXF utility takes a BRO 3D vector file and converts it to POINT, 3DLINE, 3DFACE, CIRCLE and POLYLINE/VERTEX entities in a standard DXF file format.
  • Importing DXF Files - Basic Rules
    For importing, use the Microsoft Command Window utility, Dxf2asap.exe (located in the asap\bin folder).
