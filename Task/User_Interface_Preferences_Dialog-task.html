User Interface Preferences Dialog

Sets the default viewing preferences in ASAP.

To access this dialog, go to the ASAP File menu, select Preferences, and select the relevant tab. Click Help on the tab for information on this dialog.

  • User Interface Preferences - 3D Viewer Tab
    This topic describes preferences that apply globally to the 3D Viewer.
  • User Interface Preferences - Builder Tab
    The following preferences apply globally to the Builder.
  • User Interface Preferences - CAD Import Tab
    The user preferences described in this topic apply globally to the CAD translation process.
  • User Interface Preferences - Custom Commands Tab
    The ten customizable tool buttons provide users with an easy way to program command strings for frequently used commands. The buttons are programmed on the Custom Commands tab of this User Interface Preferences dialog.
  • User Interface Preferences - General Tab
    The user preferences described in this topic apply globally to the within ASAP.
  • User Interface Preferences - Input/Output Tab
    Describes global preferences for command input and output operations.
  • User Interface Preferences - Plot Viewer Tab
    This topic describes preferences that are applied globally to the Plot Viewer and Chart Viewer.
  • User Interface Preferences - Quick Start Tab
    Describes user interface preferences for the Quick Start toolbar.
  • User Interface Preferences - Remote Tab
    User interface preferences for ASAP Remote apply globally to ASAP Remote sessions.
  • User Interface Preferences - 3D Viewer Tab
    This topic describes preferences that apply globally to the 3D Viewer.
