Plot Viewer Window

Use the ASAP Plot Viewer to verify that your optical system models are correct before proceeding with analysis.

Create a plot by choosing the appropriate commands from the Display and Analysis menus on the main menu bar. Add annotations to the plot window with the Annotation toolbar.



Plot Menu

When a Plot Viewer window is open, the main window menu bar changes dynamically to include commands on the Plot menu.

Isotropic Forces the plot to maintain its 1:1 aspect ratio regardless of the aspect ratio of the window; otherwise, the plot fills the available space in the window. If the window is not the same aspect ratio as the plot, the plot is scaled differentially to accommodate the window dimensions. By default, Normal Mapping is enabled.
White Background Reverses the background color of the Plot window (that is, white goes to black and vice versa) to match your preference.
Monochrome Forces the plot colors to monochrome.

Context Menu

Most of the plot commands on the Plot menu are also available on the context menu (by right-clicking) in the Plot Viewer. Another command that is also on this menu is Object Properties, which opens the dialog box for setting properties of the selected object.

  • Annotation Toolbar
    The Plot Viewer and Chart Viewer include an annotation toolbar for adding text, lines, and shapes to the Viewer windows for presentation purposes.
  • Plot Menu
    When a Plot Viewer window is open, the main ASAP menu bar changes dynamically to include commands related to this window.
  • Plot Viewer Object Properties Dialog
    To edit properties of annotated objects in an open Plot Viewer plot, select the existing property setting on the dialog box, and enter the change.
  • Plot Viewer Toolbars
    The Plot Viewer offers two toolbars: standard and annotation. The standard toolbar is described in this topic. See the topic, Annotation Toolbar, for a description of buttons used to add text, lines, and shapes to the Viewer window.
  • User Interface Preferences - Plot Viewer Tab
    This topic describes preferences that are applied globally to the Plot Viewer and Chart Viewer.
