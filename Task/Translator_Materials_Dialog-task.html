Translator Materials Dialog

The Materials dialog box applies to both the IGES- and GTX-to-ASAP translator. The dialog opens from the Translator Modify Object dialog box.


Coatings Select the type of coating, and specify the Reflect/Transmit values.
MediaSelect the type of media and specify its Refractive Index.
Click to add a coating or media. Opens the Coating dialog box for entering the Name, and Reflect and Transmit values.
Click to delete the selected coating or media.
ColorsSelect the name of the color and its RGB (Red, Green, Blue) values

Click OK to accept the new color, or Cancel if you do not want to add the color.
