BRO 3D Viewer Context Menu in Tree View

Includes 3D Viewer commands. Using your mouse, right-click in the Tree View area (left pane) of the 3D Viewer window to open the menu. The following commands are available:


PropertyKeyboard KeysDescription
PropertiesEnterOpens Object Properties dialog box for the currently selected object.
Optional PropertiesShift-EnterIf applicable, opens Optional Properties dialog box for the currently selected object. Applies to rayset and *.dis files, and the test scene.
Close FileDeleteCloses selected file (applicable only when a top-level object is selected.
Make VisibleVMakes the currently selected object(s) visible.
HideHHides the currently selected object(s).
WireframeWChanges draw mode of the currently selected object(s) to wireframe view.
LitWireFrameLChanges draw mode of the currently selected object(s) to illuminated wire frame.
Flat ShadedFChanges draw mode of the currently selected object(s) to flat shading.
Smooth ShadedSChanges draw mode of the currently selected object(s) to smooth shading.
Highlighted EdgesEChanges draw mode of the currently selected object(s) to highlighted edges.
Zoom ExtentsNZooms the view so that the currently selected object(s) fit within the window.
Isolate and Zoom ExtentsIMakes visible only the current selection, and zooms the view so that it fits within the window.
Isolate w/o Zoom ExtentsMMakes visible only the current selection without (w/o) zooming the view.

Polar Intensit Plot

Opens the Polar Intensity Plot dialog when a DIS file (radiant sphere) is open.

Poincare Sphere Options

Opens Poincare Sphere Options dialog

Display Polarization Information

Displays the S1, S2, and S3 polirization state data, reference vector, ray flux, and ray diretion for one or more selected points in a popup box to the right of the selected point(s).

Save Polarization Information

Saves ray data for multliple selected points to a user-specified text file.

Compute Ellipse

Computes the polirization ellipse parameters and displays data in the Command Output window.

Render Flux

Switches on and off the rendering of flux data.

  • Display File Optional Properties Dialog
    Sets the color for specific *.dis files. To access the Display File Optional Properties dialog box, right-click in left tree pane and select Optional Properties.
  • Object Properties Dialog
    The Objects Properties dialog box sets the properties of the currently selected object . It is accessed from the BRO 3D Viewer, in the object tree-view pane.
