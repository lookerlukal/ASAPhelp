Sort Rays by Path Dialog

Produces a table of summarizing propagation paths of currently defined rays.


Criterion:If the ray path contribution is greater than zero (>0), sorts and prints paths by:
Total FluxTOTAL flux (default)
Average IrradianceAVERAGE irradiance
Peak IrradiancePEAK irradiance
Object Flux OBJECT flux

Quantity:


Sum, Minimum, MaximumDetermines whether the flux-related quantity for each distinct path is the minimum, sum, or maximum of the individual path contributions.
Only Show Paths with Greater than __% Contribution Prints only those paths whose percentage contribution is greater than the decimal number.
Sort PathsSelect if flux sorting threshold is greater than 0.
