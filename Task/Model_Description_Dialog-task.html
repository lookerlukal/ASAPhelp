Model Description Dialog

This dialog box opens from either the Roughness Models tab or the Scatter Models tab on the Quick Start toolbar. Double-click a model name to open the dialog.


Model NameDisplays the name of the model you selected from the Quick Start toolbar.
Description:Displays a description of the model, and notes pertaining to the model.
View PlotClick to view a plot of the model.
