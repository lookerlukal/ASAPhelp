BSDF Fit Utility: Open or Import Files

BSDF Fitting supports any file type as long as the open file mode is set to BSDF Fit Utility Files .

To import measured BSDF data, select File > Open. In the Open Files dialog box, set the Open Mode to BSDF Fit Utility before selecting a file type or file name.

To open a previously saved file, select File > Open, set the Files of Type to BSDF Fit Utility Files (*.fit, *.atm, *.hs), and select a file.

Saved files include the extension *.fit.
