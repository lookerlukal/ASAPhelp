Interactive Scripts - Uses

The Interactive Scripts page on the Quick Start toolbar includes files under the categories of Bio-Optical Models, Array Enhancers, Lens Creation, and User-Defined.

Open an Interactive Script


  1. Select an interactive script file from the tree view of the Interactive Scripts page.
  2. Double-click the selected interactive script file, or right-click the selected file to open a menu.
  3. Select Open on the menu.

Insert an Interactive Script into an Existing INR Script File


  1. Open an existing INR file, and place the cursor at the location in existing INR file where contents will be inserted.
  2. Select an interactive script file displayed in the tree view.
  3. Hold down the Ctrl key, and drag and drop the selected script file to the open INR file.

Run an Interactive Script


  1. Select an interactive script in the tree view.
  2. Double-click the script file or right-click the file and select Run on the menu.

Add a New Interactive Script

Method 1


  1. Right-click the interactive script node in the tree view and select Add New on the menu.
  2. Highlight text in an Editor window or Command Output window, and drag and drop it onto the tree view for Interactive Scripts.

Method 2


  1. Right-click the interactive script node on the tree view and select Add New on the menu.
  2. Complete the Add New dialog box, and click OK to add the new script name to the group.

File Formats

All interactive script files in folders for Bio-Optical models, Array Enhancers, Optimizers, and Lens Creation must be in an INR file type (*.inr).

These files must include a comment (!!) section of the script file that contains: Title, Description, Category, and Keywords.
