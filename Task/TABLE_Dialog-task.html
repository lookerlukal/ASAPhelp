TABLE Dialog

Creates a table of the current data. Optionally overlays values onto a contour plot.

Table of Data Values:


Number of columns

Number of rows

Specifies the table range.
Number of characters per cellSpecifies the number.
Skip cells with value:Leaves table entries blank for specified cell values.

Plot:


Create plot If checked, creates a plot of the table.
Scale charactersRescales characters by the specified factor.
Overlay next Plot If checked, next plot created is placed on top of previous plot.
