Optimization - Defining Design Variables

This topic describes how to define and enable a design variable for optimization.

Pre-requisites:

To begin, a suitable ASAP INR script file must be open in the ASAP Editor window, and the Optimize menu must have been activated by selecting Optimize Script from the Editor menu.

  1. Select text in the script containing a scalar real variable. The selected text is a valid expression (in the form name=value, where value is a constant) that can be evaluated to create a design variable and nominal value.
  2. Right-click the selected text and select Define Design Variable from the context menu. Alternatively, from the ASAP menu select Optimize> Define Design Variable. The Design Variables tab on the Optimization Setup Summary window is displayed, with the name and nominal value entered in the dialog, based on user selection in the script.

  3. On the Optimization Setup Summary window, enter an alias name, minimum, maximum and step-size, and manufacturing tolerance, if desired. You may also switch the Enabled State from Enabled to Disabled. The alias, which is editable, is prefilled with the variable name.
Result:

The design variable is defined. ASAP validates the defined design variables and displays them in the list on the Optimization Setup Summary window.

Post-requisites:

Define variables, design objectives, optional objective constraints, exit criteria, and optimization methods, and select on the Optimization Method tab.
