Plot Directional/Positional Ray Data Dialog
Plot Directional/Positional Ray Data Dialog

Creates a geometric spot diagram for the currently selected ray data.


Title Optional title for the plot (up to 64 characters).
Color:
Use Source Color Applies different color for each source.
Use Object Color Selects the color associated with the ray's current object, rather than the ray's source, when plotting a spots diagram.
Use This Color Applies color selected from dropdown list.
Plot:
Plot all rays
Plot every nth ray Plots only every nth ray instead of all the currently selected rays (which are still summed in the distribution file).
Do not generate plot
Plot symbol Uses designated symbol from dropdown list when drawing plots.
Overlay Next PlotOverlays next plot with the current plot. Useful for combining system plot with ray trace plots and multiple spot diagrams.
