Ray History Tab

Sets the defaults for volume bin resolution and log cutoff in the 3D Viewer. This tab is on the File Preferences dialog box, which is accessible from the ASAP File menu. Select 3D Viewer File Preferences.


Default Volume Bin ResolutionThe data, which is plotted logarithmically, represents flux that passed through a given cell and was divided by the cell's volume.
Default Log CutoffFORM command specifies the lower cutoff (in orders of magnitude), relative to the data maximum.
