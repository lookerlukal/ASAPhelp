Software Keys

To fully activate your Breault program, enter your software authorization key numbers in the Registration dialog on the Help menu, if they do not already appear in the License Key area.

Click Update to verify the numbers and store them in the program. If the numbers are incorrect, a message asks you to re-enter them and click Update again. In some BRO programs, the numbers are displayed on the Help menu, under Registration. The first set of software key numbers (from the left) is your BRO customer identification (ID).

Click Close when you are ready to return to the program.

Tips


  • You can copy the keys from an application like Notepad or your e-mail, and then paste them into the Registration dialog via the Clipboard. Use the Import From Clipboard command on the File menu in the Registration dialog box. You can also copy the numbers to the Clipboard and paste them into another application by using the Export to Clipboard command on the File menu in the Registration dialog box.
  • The key numbers that are entered into the Registration dialog box are linked to the current user's name.
