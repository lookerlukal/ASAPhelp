OSLO to ASAP Translation Step 1

The first step in the OSLO to ASAP translation process involves selecting a *.LEN file produced by OSLO.

Either type the name of the file in the edit box or press the Browse button to display a File Open dialog to select a LEN file.

The translator confirms the file’s existence by attempting to open it. If it is successful, the name of the selected file appears in the edit box, and the translator creates a default name for the ASAP file. (In this case, the default name for the ASAP file is Anaprism.inr). If the translator cannot open the OSLO file, it displays a message box.

Tip

The translator does not move to the next dialog unless a valid filename is selected.

After the OSLO file is selected, the translator reads the file for compatibility with ASAP and displays a summary in the scrolling list box. The list box has four columns of adjustable width for read truncated text. The columns are:


Surface Surface number to which the translator calls attention.
CommandExact text of the OSLO command the translator read.
MessageBrief message summarizing the translator’s response to the OSLO command.
ExplanationDescribes commands that are not supported in OSLO command.

Messages

Messages do not necessarily indicate that there is a problem. These are intended to alert you to features and capabilities present in one program that may be imprecisely translated into the other. Messages fall into the following categories:

Aperture specification is missing in the OSLO lens. The OSLO file did not contain an AP command specifying aperture information on one or more surfaces.

Valid but unsupported OSLO command was read by the translator. OSLO is a comprehensive lens design program with numerous features such as drawing structure around elements, setting up element arrays, and so on, which do not have direct analogs in ASAP and therefore cannot be translated. Certain surface (eikonals, etc.) and phase diffractive types also fall into this category.

OSLO feature is only partially supported in ASAP. OSLO can describe certain surface types that can be modeled only partially in ASAP. To accomplish the translation it is therefore sometimes necessary to truncate the order of the polynomial, restrict the translation of a polynomial description to specific terms, or call attention to a specific term that is not supported.

OSLO file contains “special” surfaces or materials. The translator flags all types of aspherics, gradient index (GRIN) materials, and diffractive surfaces.

After creating the list, the translator writes out a summary recommendation, which fall into five categories:

Solves and/or pickups were found; delete before proceeding. Solves and pickups are used in lens design to establish relationships between surfaces, locations and so on. Unfortunately OSLO only writes the relationship into the LEN file, not the data being related; therefore, the LEN file does not contain a complete prescription.

CAUTION: Translating a LEN file with solves and/or pickups almost certainly result in an erroneous ASAP model and so these should be deleted prior to translation. (The OSLO commands, CSD and TSD may be used to delete curvature and thickness solves and pickups.)

Special surface types and/or materials were found; check ASAP model. The translator noted these while reading the OSLO file. Carefully review the notes in the list box and the ASAP model for potential problems.

Missing apertures on some surfaces; correct LEN file or set default in next dialog. Since precise aperture information is essential to ASAP, this information must be supplied in order to perform the translation. While it is probably best if you return to OSLO and properly set the aperture(s), you can also set default apertures or ask to be prompted for apertures in the second translation dialog.

Unsupported commands were found and could affect translation accuracy. The translator noted these while reading the OSLO file. Carefully review the notes in the list box and the ASAP model for potential problems.

OSLO lens is ready for translation. No warnings were found.

At the completion of this dialog, click NEXT to continue, or CANCEL to terminate the translator.
