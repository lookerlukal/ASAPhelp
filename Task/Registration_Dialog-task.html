Registration Dialog

The Registration dialog is accessible from the ASAP Help menu. It includes customer information listed in this topic.


Key ID Hardware key identification number. ASAP detects the key and automatically inserts it in the dialog box
Licensed Products

ASAP/Basic and ASAP/PRO

Your licensed products are those that are checked.
ModulesCAD, Optical, ELTM, and Network LicenseYour licensed configurations are those that are checked.
Support is current, or

Support has expired

The status of your maintenance agreement is displayed. An active agreement entitles you to free upgrades, updates, and technical support.
License Key
Current UserIf selected, indicates the license is assigned only to a specific user for the machine on which ASAP is installed. Only the specified user can use ASAP.
Local MachineIf selected, indicates the license is assigned to a specific machine. Any user can then access ASAP on that machine.
Customer ID Identification number assigned to a specific ASAP user.
Software Keys Identification numbers assigned to a specific ASAP license.
Network License Servers If you are using ASAP on a server, your network license identification number must be set. See your Network Administrator.
UpdateStores new information that you enter and closes the dialog box.
Close Closes the dialog box without updating the information.
Demo Mode If you have a demonstration version, click this button to open use without a license. Demo Mode offers restricted use of ASAP.
Import from/Export to clipboard Copies the software keys to or from the Microsoft Clipboard for pasting. See Notes.
Note:
  • To fully activate ASAP, enter your software authorization key numbers in the License Key area, if they do not already appear in there. The numbers are on the BRO Software Authorization Keys document provided with your ASAP shipment.
  • Click Update to verify the numbers and store them in ASAP. If the numbers are incorrect, a message asks you to re-enter them and click Update again. The numbers will appear on the ASAP Help menu, under Registration. The first set of software key numbers (from the left) is your BRO customer identification (ID).
  • You can copy the keys from an application like Notepad or your e-mail, and then paste them into the Registration dialog via the Windows Clipboard. Select Import from clipboard in the Registration dialog box. You can also copy the numbers to the Clipboard and paste them into another application by selecting the Export to clipboard. Using the Clipboard will assure that the keys are transferred accurately.
