Polarization - Viewing a Poincare Sphere

This topic describes how to view a Poincaré sphere in the 3D Viewer window.

Pre-requisites:

A suitable ASAP DIS file that contains polarization data must be available.

  1. Select 3D Viewer from the list of document types on the Views tab in ASAP Workspace.
  2. Right-click 3D Viewer and select Open from the context menu. The Windows dialog, Open Files, is displayed.
  3. Change Files of type to Display Files (*.dis), browse to the folder that has an existing DIS file that contains polarization data, and select it. ASAP verifies that the DIS file contains polarization data, and opens the file in the 3D Viewer window.
Result:

A Poincaré sphere is drawn in the 3D Viewer graphics pane.
