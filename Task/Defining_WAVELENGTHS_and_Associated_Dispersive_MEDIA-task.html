Defining WAVELENGTHS and Associated Dispersive MEDIA

This topic lists the input arguments and return for defining WAVELENGTH(S) and associated dispersive MEDIA.

To define a dispersive medium in ASAP, enter the following commands in this order:
UNITSwith an appropriate selection to specify the overall system geometry units
WAVELENGTHSwith ascending or descending wavelengths, followed by a UNITS option to properly scale wavelength units to system units
MEDIAwith corresponding refractive indices to the previously specified WAVELENGTH(S)

CAUTION


  • After you enter the UNITS command, do not enter it again until the system geometry is replaced by another system. Similarly, enter the UNITS option on the WAVELENGTH command only once.
  • To change the wavelength (for setting up rays for tracing, for example), enter the WAVELENGTH command with the appropriate wavelength. ASAP remembers the wavelength units.
