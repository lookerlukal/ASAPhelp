Calculate Flux Distribution Dialog

Creates a geometric spot diagram, and, if selected, creates a far-field incoherent intensity or radiance for the currently selected ray data.


Method:
Flux/Area (Irradiance using SPOTS POSITION)Creates a distribution file and, optionally, a spot diagram of positional ray data for the current ray set. Coordinate directions are set under WINDOW.
Flux/Steradian (Intensity Using SPOTS DIRECTION)If selected, creates a spot diagram of directional ray data for the current ray set.
Flux/Steradian (Intensity using RADIANT)If selected, calculates the far-field incoherent intensity, or radiance, of the currently selected ray data, and creates a distribution file.
PlotSee Plot Dialogs.
WINDOWSee WINDOW Dialog.
PIXELSSee PIXELS Dialog
RADIANT Options:These options display in the dialog only when Flux/Steradian is selected.
Polar Axis of Spherical Coordinate System Specifies the axis.
Rescaling Factor Alters the sizes of the polar diagrams relative to the spatial grid spacing.
Generate 3D Radiant Sphere If checked, creates a 3D polar diagram for each point on the spatial grid.
Zenith/Latitude Angles From Axis Specifies range in degrees for the angles about the axis (zenith/latitude), and the number of subdivisions of this angular range.
Azimuth/Longitude Angles Around Axis Specifies range in degrees for the angles about the axis (azimuth/longitude angles), and the number of subdivisions of this angular range.
