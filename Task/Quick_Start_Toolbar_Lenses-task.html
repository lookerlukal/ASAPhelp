Quick Start Toolbar - Lenses

The Lenses button on the Quick Start toolbar opens the Lenses page—a catalog of pre-defined lenses available in ASAP. Hovering the mouse pointer over a lens displays a brief description of the lens, such as focal length, shape, and diameter.


  • To create a lens directly in ASAP, double-click the lens name. The output window displays the ASAP command that is being used to create the lens.
  • To create a lens in script, drag/drop the lens onto the script. The script to create the lens is inserted at the current cursor location.
  • To create a lens in the Builder, drag and drop the lens on the builder. The appropriate Builder entries are created.

You can search for a lens that meets your criteria by right-clicking the lens name and selecting Search For Lenses on the popup menu.
