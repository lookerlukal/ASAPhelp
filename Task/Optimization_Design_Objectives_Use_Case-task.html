Optimization - Defining Design Objectives

This topic describes how to define and enable a design objective for optimization.

Pre-requisites:

An open INR file that contains optimization elements such as a scalar real variable must be open in the Editor window.

  1. Select text in the script containing a scalar real variable. The selected text must be a valid expression in the form name=value, where value usually contains defined design variable names that can be evaluated to create a design objective and target value.
  2. Right-click the selected text and select Define Design Objective from the context menu. Alternatively, from the Optimize menu on the ASAP toolbar, select Define Design Objective. The design objective displays in the list on the Design Objectives tab of the Optimization Setup Summary window. The name and target value are filled in based on user selection in the script.

  3. You may change the target value, weight, or enabled state of the objective.
Result:

The design objective is defined. ASAP validates the defined design objectives and displays them in the list on the Design Objective tab.

Post-requisites:

Define design variables, design objectives, objective constraints, exit criteria, and optimization methods, and select on the Optimization Method tab.
