3D Viewer - Shortcut Keys
3D Viewer - Shortcut Keys

Activates a particular command in the 3D Viewer via a predefined keyboard shortcut.


KeyDescription
AToggles whether axes are displayed. If axes are displayed, they are drawn at the current rotation center.
CToggles whether the control panel is displayed.
EChanges draw mode for the current selection to Highlighted Edges.
FChanges draw mode for the current selection to Flat Shaded.
HHides the currently selected object(s).
IIsolates the current selection (makes it the only thing visible) and zooms the view so it fits within the window (zoom extents).
LChanges draw mode for the current selection to Lit Wireframe. This mode is similar to conventional wireframe except that surface normals are included for lighting effects.
MIsolates the current selection (makes it the only visible item).
NZoom extents on the current selection.
OZoom extents on only visible objects so that all currently visible (non-hidden) objects fit within the window.
SChanges draw mode for the current selection to Smooth Shaded.
TSwitches keyboard focus to the object list tree.
VMakes the currently selected object(s) visible (the opposite of hide).
WChanges draw mode for the current selection to Wireframe.
XRotates about the -X axis. If Shift key is also pressed, rotation is about +X axis.
YRotates about the -Y axis. If Shift key is also pressed, rotation is about +Y axis.
ZRotates about the -Z axis. If Shift key is also pressed, rotation is about +Z axis.
EnterOpens Object Properties dialog box for the current selection.
Shift-EnterOpens the Optional Properties dialog box for the current selection (applicable only if the selection has optional properties).
DeleteCloses the currently selected file. This is only applicable to top-level objects (for example, *.vcr files).
1, 2, 3, 4, 5, or 6Changes the current model view to standard view (1 to 6), and performs a zoom extents for all visible objects. Standard views 5 and 6 are user definable.
8, 9Incrementally moves slices, one cell at a time, in negative (8) or positive (9) direction along the axis, to determine concentration of light. (Shift key plus left mouse button and dragging accomplishes the same moves.)
