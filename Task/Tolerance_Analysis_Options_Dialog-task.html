Tolerance Analysis Options Dialog

The Tolerance Analysis Options dialog box designates the number of random perturbations you want to perform for the automatic Monte Carlo analysis.

Select the Tolerance Options button on the Editor toolbar to open the dialog box.


IterationsDesignates number of random perturbations for the Monte Carlo analysis. Default is 1 time. This field is ignored when the Sensitivity Analysis option is selected.
Figure(s) of MeritASAP searches in the *.inx file for the text string to grab and save; for example, FOM1, FOM2... where FOM is a Figure of Merit. Each Figure of Merit can be viewed in the Command Output window. The number of FOMs selected determines the number of plots. See Note below for explanation.
Analysis optionsSelect type of analysis: Nominal, Monte Carlo, or Sensitivity.
Compensator VariablesThese fields are available only when Sensitivity Analysis is selected. Enter the relative precision and zero tolerance values when a compensator parameter is defined. The default setting is 1.0e-6 for the relative precision, and 1.0e-14 for zero tolerance.

Save Options


Save Tolerance DataSelect to save the data to a *.sys file that can later be viewed using the SYSTEM FROM command. By default, tolerance data is NOT saved. Select this box if you want to designate the number of tolerances to save.
# Tolerances to SaveEnter the number of tolerances you want to save. Results are stored in toldata*.sys files in your working directory.

Note:

Click OK to begin the automatic Monte Carlo or sensitivity analysis. Data sets that are created for the defined Figure of Merit(s) will be placed in an FOM*.DIN file in your working directory. Data from the FOM plots are displayed.

Before performing a Monte Carlo or sensitivity analysis, a nominal Figure of Merit value may be obtained by selecting the Nominal Only option in the Tolerance Analysis Options dialog box.
