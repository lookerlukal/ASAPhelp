Translator Modify Object Dialog
Translator Modify Object Dialog

The Modify Object dialog box applies to both the IGES- and GTX-to-ASAP translator.

Properties


Coating Select Bare, Absorb, Reflect, Transmit, or Default.
MediaSelect Air for media 1 and 2.
ColorSelect from drop-down list of 27 colors.
FacetsSelect Intra-Edge (number of facets along edges or curves) or Inter-Edge (number of facets between edges or curves).
Scatter Half AngleEnter degrees.
Fresnel Average & Split 1 AllSelect or deselect.
Name/AliasIf you have created an alias name in the Translator Dialog for IGES and XML, the alias is listed in this drop-down box.
Reset, Reset AllReturns user-named object(s) to the default name.
New Coating buttonOpens Coating dialog box to select coatings (Name/Reflect/Transmit).
New Media buttonOpens Media dialog box for naming and refractive index.
Advanced or StandardExpands and contracts the Advanced Objects Definition box, which includes the Recursive and Clear options.
RecursiveAutomatically applies properties to all child objects so that they receive same attributes as their parents.
ClearIf parent selected, deletes attributes for parent including all children; if some but not all children are selected, clears only those children.
