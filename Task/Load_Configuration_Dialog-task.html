Load Configuration Dialog

When you apply any of the four Display Viewer window user-defined view, use the Load Configuration dialog box.

The Load Configuration As dialog is accessible from the Display Viewer menu when the Display Viewer window is active.

File > Configuration > Load

Select a user-defined view name from the drop-down list in the Configuration box.

Click OK to apply the selected file or Cancel to close the dialog without making the changes.

When you want to save changed settings to a user-defined view file, use the Save Configuration As dialog box.
