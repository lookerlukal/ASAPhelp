Lit Appearance: Spot Diagram Approximation

This topic describes how to use the lit appearance feature in ASAP for spot diagram approximation.

  1. Save the ray data on a transmissive dummy surface after the last object of the lamp. This data contains position, flux, and direction cosines.
  2. "Bin" the data to create lit appearance.
