SPREAD and FIELD Calculations (Point Sources)

Prior to defining the rays, specify the input listed in this topic.


  1. BEAMS COHERENT DIFFRACT
  2. WAVELENGTH (required)
  3. PARABASAL (required)
  4. WIDTH (usually recommended to smooth out bumps)
  5. USERAPOD (used only for Gaussian or user-defined apodization)
  6. POLARIZATION (used only for polarization ray tracing)

To specify the rayset, use the following input in this order:


  1. GRID or RAYSET (one of these is required)
  2. SOURCE POSITION/DIRECTION/FOCUS (one of these is required), or GAUSSIAN

After the GRID and SOURCE commands, specify the following input in any order. These inputs are used to modify the characteristics of the SOURCE :


  1. FLUX (used to modify fluxes of rays)
  2. SHAPE (used to modify coherence of the rays)

To specify where and at what resolution to do the calculation, use the following input:


  1. CONSIDER/SELECT (not required unless you want to restrict the ray data)
  2. WINDOW (required so that ASAP knows where to do the calculation)
  3. PIXEL (not required unless you want higher resolution)

After following the above order, run SPREAD or FIELD.

After the calculation, use DISPLAY to examine the distribution data.
