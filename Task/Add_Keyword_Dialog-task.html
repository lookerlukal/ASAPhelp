Add Keyword Dialog

Enters a new custom keyword to the Keyword Editor dialog box for the ASAP Editor window. The keyword is typically an abbreviation of an ASAP command name, and will be assigned to the ASAP command that you selected from the Keyword Editor.


Enter new custom keyword.Type the name of the new user-defined keyword.
OK Click to accept the new keyword and return to the Keyword Editor dialog box.
Cancel Click to return to the Keyword Editor dialog box without accepting the new keyword.
