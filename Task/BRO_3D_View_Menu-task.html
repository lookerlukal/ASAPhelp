3D View Menu

The 3D View menu on the ASAP toolbar is visible when the 3D Viewer window is open.

The 3D View menu includes the following commands:


Display AxesIncludes coordinate axes drawn at the current center of rotation.

Display Paths

Disabled when a Poincar√© sphere is displayed in the 3D Viewer graphics pane.

White BackgroundSwitches between white and black background.
Swap FG/BG Line ColorIf checked, foreground color is not swapped when white background is used. (Normally, foreground lines are drawn in white when black background is selected, and drawn in black when white background is selected.)
Viewer ControlsOpens Viewer Controls dialog box for adjusting ambient light, directional light, and object manipulation.
Bounding BoxDisplays bounding box when performing a rotate, shift, or zoom with the mouse.
Zoom ExtentsCenters visible objects and scales them to fit the window
Set Lighting to Startup ValuesResets lighting intensities to the default values.
Set Lighting to Original ValuesResets lighting intensities to the BRO default values.
Save Current Lighting as Startup ValuesSaves lighting intensities to values that are currently set.
Standard View 1 (Y-Z)Rotates object to standard Y (up) and Z (right) view, and scales (zooms) the scene so that all visible objects fit within the window.
Standard View 2 (Y-X)Rotates object to standard Y (up) and X (right), and scales (zooms) the scene so that all visible objects fit within the window.
Standard View 3 (X-Z)Rotates object to standard X (up) and Z (right), and scales (zooms) the scene so that all visible objects fit within the window.
Standard View 4 (Isometric)Rotates to standard isometric view and zoom, and scales (zooms) the scene so that all visible objects fit within the window.
Standard View 5 and 6 (user-defined)Rotates to standard user-defined view 5 or 6, and scales (zooms) the scene so that all visible objects fit within the window.
Current View InformationOpens Current View Information window to display useful information; for example, rotation center, model view, projection matrices, pixel format, and OpenGL implementation details.
Save Current View As 5 or 6Saves the current Viewer window settings to a file for Standard View 5 (user-defined). Overwrites previous settings.
Axes PreferencesOpens Axes Preferences dialog box for adjusting axis line thickness and origin placement.

View Object Tree

Opens or closes the Object Tree in the left pane of the 3D Viewer window.
