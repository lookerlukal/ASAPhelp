SCR Checkbox Control Properties Dialog
SCR Checkbox Control Properties Dialog

Defines the label associated with a checkbox, and the name of a variable associated with the checkbox.


Label Enter name of the checkbox and designate the monospacing between the label and the checkbox. (A check box with no label and no spacing between the label and checkbox forces the screen to be displayed in the dialog box.)
Variable Name Enter the name for the variable field.
