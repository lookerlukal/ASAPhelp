User Interface Preferences - Custom Commands Tab

The ten customizable tool buttons provide users with an easy way to program command strings for frequently used commands. The buttons are programmed on the Custom Commands tab of this User Interface Preferences dialog.

You can open this dialog by clicking on the Custom Toolbar. The Custom Toolbar:


  • is dockable
  • expands to a maximum of 20 defined buttons (10 pre-defined plus up to 10 user-defined) as new buttons are created
  • supports user-defined bitmap image on any Custom Toolbar button.

Button NameLists available buttons; the first ten are pre-defined and color coded.
Move up or downClick arrow to change position of button on the button bar.
AddClick to add a button after creating the text.
Edit Bitmap Opens the Button Editor for customizing one of the bitmap graphic files associated with the Custom Toolbar buttons.
RemoveDeletes a button. Highlight the button name and click to delete it.
Tooltip textDisplays text for current button, if one exists. Highlight text to change or delete it, or type new text.
Status bar textDisplays text that appears on ASAP status bar.
Bitmap file nameDisplays file name if a graphic is assigned. Highlight to edit, or browse to the file location to add a new graphic file.
Tool textDisplays tooltip text in box. Highlight text to edit or type new text.
