ASAP Status Bar

The ASAP Window status bar provides feedback about default settings and quick tips. From the left, the status information is as follows:


Errors and Warnings The number of errors or warnings detected in the output window. To go to the Command Output line that has the error, double-click the error box and use the up/down arrows to locate each error in the Command Output window.
WU / WS Wavelength Unit or Scale specified.
SU System Unit specified.
Directory path Path of current $PATH command, which is the Working Directory.
