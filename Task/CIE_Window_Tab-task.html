CIE Window Tab

The CIE Windows tab is part of the CIE Color Analysis dialog. Use it to designate settings for the CIE window.


Window for CIE Color Analysis Select the orientation of the analysis window.
Use Current Window SettingsSelect to force the CIE calculation to use the current WINDOW settings in the ASAP kernel. When deselected, the window size defaults to the extents of the object that is selected above. However, you can easily override these setting by changing the Axis Minimum and Maximum extents.
Horizontal and Vertical axes

Axis/Min/Max

If Use Current Window Settings is not selected, select the axis you want to be considered. Options are X, Y, and Z. The Min and Max values set the signed values.

(ASAP knows the bounding of the selected object.)

Vertical Elements in WindowSets the number of equally spaced pixels along the vertical axis of the plotting window for this plot only.
Aspect Ratio (Horizontal/Vertical)Ratio of pixels per system unit along the horizontal axis to pixels per system unit along the vertical axis.
