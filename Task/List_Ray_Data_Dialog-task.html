List Ray Data Dialog

The List Ray Data dialog is accessible from the Rays> Ray Information menu. The dialog generates a list of ray data.


PositionsLists ray positions, fluxes, current objects, and optical path lengths.
Directions Lists ray directions, fluxes, current objects, and optical path lengths
Positions and Directions Lists ray positions and directions.

List Polarization States

Lists polarization states of the current ray set. If selected, Polarization Form is enabled.

Polarization Form

Select from these states: None (default), Jones, and Stokes

Source Data (Wavelengths) Lists wavelength and ray numbers for each source.
Ray Data Information Lists ray database information (current medium, split level, source number, number of objects, current object, and previous starting objects for each ray.
Polarization Lists polarization parameters of selected ray data.
Relative to Ray NumberLists wavelength and ray numbers for each source.
