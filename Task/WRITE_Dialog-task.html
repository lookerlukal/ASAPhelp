WRITE Dialog
WRITE Dialog

Writes the current data to a disk file.

Display: WRITE Data to Disk


File NameIdentifies file name of binary and/or text file to write to disk.

Format:


Both Binary (filename.dis) and Text (filename.din)Writes to both file formats.
Only BinaryLimits writing to filename.dis.
Only TextLimits writing to filename.din.

Override Existing Title

New Title

Overrides the title record in the resulting file, and inserts a new title (up to 64 characters).
