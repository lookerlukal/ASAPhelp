PLOT Dialogs
The Plot tab is part of several ASAP dialog boxes. The following description applies to:
  • Plot Rays: Plots vectors along the direction of each ray contained within the currently specified graphics window.
  • Plots Gaussian: Plots extent of each Gaussian beam with the currently specified graphics window.
  • Plot Polarization: Plots polarization state ellipses.

Plot:


Rays, Beams, Polarization Specifies type of plot.
Title Inserts optional title for a plot (up to 64 characters).
Overlay Next Plot Overlays next plot with the current plot. Useful for combining system plot with ray trace plots and multiple spot diagrams.
Override Color Applies the color selected from the dropdown list to all plot lines.

Ray Options:


Scale Factor for Ray Plot Vector Produces 3D far-field intensity distribution point clouds, when the number is much larger than any inter-ray distance.
Plot Only End Points If checked, only the end point of the vector is plotted.
Plot Every _th Ray Plots only every ith ray
