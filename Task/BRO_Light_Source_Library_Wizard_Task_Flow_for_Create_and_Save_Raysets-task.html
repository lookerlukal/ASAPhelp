BRO Light Source Library Wizard - Task Flow for Create and Save Raysets

This topic describes the task flow for creating and saving raysets in the BRO Light Source Library Wizard.


  1. Select Use BRO Light Source Wizard on the Rays menu in ASAP.
  2. When the Welcome page of the BRO Light Source Library Wizard is displayed, select a source by part name from the drop down list. Once a source library file is selected, a short description of the selected source is displayed and the Next button will be enabled (not grayed).
  3. Choose View Model button to view the source model specifications and picture.
  4. Select the Next button and the Rayset Options page is displayed.
  5. Select Option 1: Create and Save Rayset(s) to display the Source Geometry Setup - Create Raysets page.
  6. Enter a rayset name and change the major and minor axis and number of rays, if needed. By default the major axis is set to +Z, the minor axis is set to +Y, and the number of rays is set to 10,000. The major and minor axis can be one of the following +X, -X, +Y, -Y, +Z or â€“Z.
  7. Select the Next button and the Spectral Selection page is displayed.
  8. Select one or more points on the curve and then click the Finish button or click the Back button to go to previous wizards to make desired changes. If you selected a dual filament bulb, you are prompted to choose either the high (H) or low (L) filament after the wavelengths are chosen and before the raysets are created.

At this point, ASAP runs the script and macro commands, and displays the results in the Command Output window. Raysets are saved for future use.
