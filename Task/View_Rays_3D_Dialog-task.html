View Rays 3D Dialog

Plots a line from the current position of each ray along its direction vector.


Scale factor for ray plot vector The actual distance plotted is the scale factor times the flux of the ray divided by the maximum ray flux. The value entered is a negative number, only the endpoint of this vector is plotted.
