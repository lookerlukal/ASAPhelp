ELTM Contour Setup Dialog

Select the type of ELTM plot contours in the results table, and set parameters for scaling, range, and number of contours.


  • Click New to view the default setup.
  • To change default settings, enter Scaling, Range, and Contours settings under Parameters. Changes are automatically updated for the description selected; the number of new descriptions is limited only by your computer memory. The Clone button may be used as a convenience for making new descriptions, based on descriptions you have already created.
  • Check the descriptions that you want to apply to the plot. By selecting Fixed rather than Auto parameters for Scaling or Range, you gain latitude in defining height and targeting a specific segment of the plot for more detailed viewing.

For example, if you designate values of 7 contours, autoscaled in red, within a range of 0.000000 and 5000.000000, this produces an autoscaled plot showing seven contours with a view fixed between a minimum height of zero and a maximum height of 5000 units. The units are based on the imported geometry file. However, choosing a fixed plot gives you control over the palette and where on the plot you want the seven contours focused.


  • Click OK to accept the changes and continue, or Cancel if you do not want to accept the changes
