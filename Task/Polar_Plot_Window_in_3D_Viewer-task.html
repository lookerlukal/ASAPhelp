Polar Plot Window in 3D Viewer

Right-click the file name in the Tree pane of the 3D Viewer window, and select Polar Intensity Plot to open the Polar Intensity Plot dialog.

Pre-requisites:

To access the Polar Intensity Plot dialog, you must first open a saved display (*.dis) file (radiant sphere) in the 3D Viewer.



To view the polar plot, you must complete this dialog box and click OK. When you do this, the 3D Viewer creates radial data on the polar plot in the Polar Plot window.

The radial data on the polar plot represents flux values at plane/radiant-sphere intersection points (the triangles on the sphere are intersected). Angular data comprises the angle the intersection points make with the x-axis direction.

  • Polar Intensity Plot Dialog
    The Polar Intensity Plot dialog defines parameters for creating the radial plot intersection plane and y axis direction in the 3D Viewer.
  • Rayset Optional Properties Dialog
    Sets the sphere size and position, polar reference grid, and color for specific *.dis files. To access this dialog box, right-click in the left tree pane of the 3D Viewer and select Optional Properties.
