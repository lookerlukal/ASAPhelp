Add New Dialog

The Add New dialog box opens in the following situations from the Quick Start toolbar:


  • Open document: Highlighting text in an open document, such as an INR file, and dragging and dropping it onto the Interactive Scripts page on the Quick Start toolbar.
  • Command Output text: Highlighting text in the Command Output window and dragging and dropping it onto the Interactive Scripts page.
  • Explorer tab: Dragging and dropping a file from the Explorer tab of the Command Output window onto Interactive Scripts page.

File NameRequiredEnter the name of the new INR file for the script you are adding.
Script NameRequiredEnter the name of the script as it will display on the tree.
Script CategoryOptionalEnter a category name. The script is displayed on the tree under the category name. If you do not enter a category name, the script is displayed at the top level of the tree, under Interactive Scripts.
DescriptionOptionalEnter a brief explanation of the script to assist you in recognizing its content. This text is displayed as a Tooltip when the cursor hovers over the script on the tree.
ScriptRequiredChange or add script text. You may edit the script in this area before clicking OK. (Note: this field is not displayed when you drag and drop a file from the Explorer tab.
OKRequired to saveSaves the script and exits the dialog box.
CancelRequired to exit without savingCloses the dialog box without saving the content.

After adding a script to the page, you are prompted to save the script file to your system. If you do not designate a folder, the script is automatically saved in asap install\Projects\UserGeom.
