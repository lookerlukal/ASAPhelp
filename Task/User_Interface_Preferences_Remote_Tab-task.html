User Interface Preferences - Remote Tab

User interface preferences for ASAP Remote apply globally to ASAP Remote sessions.


Remove ASAP generated files from remote machinesSelect to delete the files that are generated on a remote machine during a remote session
Allow network users to run ASAP on your machine Select if you want to permit network access from other ASAP keyed machines to your machine, via Remote. To determine the status of remote connections to your machine, open the Spyke Utility for ASAP Remote.
MaxEnter number for maximum number of users who may simultaneously run ASAP on your machine.
All UsersSelect if all network users may run ASAP on your machine.
User's List Select only the names of users from the User List. This feature is not available when All Users is selected.
Automatic Remote Selection List Specify remote machines from your local domain's list. ASAP automatically selects one of the machines to start up a remote kernel.
Process Exclusion List From the Processes tab on the Windows Task Manager dialog, identify the names of processes to exclude from the Remote kernel. Enter these process names on the Remote tab. Process names are not case sensitive. Paths to the processes are not required.
  • ASAP Remote Preferences
    Use the User Interface Preferences/Remote tab to set preferences for network access from other ASAP keyed machines to your machine, via Remote.
