Combining queries in Path Explorer
Path Explorer - Combining Queries

Use the Query Combiner tab in the Path Explorer to work on the active path-set and update the active path-set.

Pre-requisites:

Path-sets must be created before using the Query Combiner.

To combine queries from start to finish using the Path Explorer Query Builder dialog:

  1. Click the Query Combiner tab on the Path Explorer Query Builder dialog, and select an available path-set name.
  2. Complete all fields and click the Run Query button.
  3. Repeat steps 1 and 2 to perform additional query combinations.
    Note: Each query acts by default on the active path-set, unless the GLOBAL modifier is used, and updates the active path-set with its results, unless the optional setname for the results is provided. If the GLOBAL modifier is used, the query acts on all paths, instead of the active path-set. If the optional setname is provided, the results of the query are written to that setname and the active path-set is not updated.
  • Path Explorer Query Builder/ Query Combiner Tab
    The Query Combiner is used to combine found path-sets into composite path-sets.
