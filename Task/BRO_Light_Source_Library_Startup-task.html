BRO Light Source Library Startup

This topic describes how to access the BRO Light Source Library.

To use a library source, you have three choices:


  1. Select Rays> Use BRO Light Source Library Wizard from the main menu.
  2. Select a source from the Quick Start Toolbar, which displays a tree view of the library where you may browse by type of source (LED, filament, ARC, CCFL), manufacturer, or subtype.
  3. Select a source from the ELTM module.
