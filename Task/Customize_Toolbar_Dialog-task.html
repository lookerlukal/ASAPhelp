Customize Toolbar Dialog

The toolbars on the ASAP window and its Builder window can be customized. You can select buttons to add, remove, or move from the dialog.

Note: Depending on where you right-click, the list of current and available toolbar buttons differs.
  • To view the toolbar buttons for the ASAP window, right-click within the empty area of the main toolbar and select the Customize button:
  • To view the toolbar buttons for the Builder, you must right-click within the Builder toolbar itself, and not in the empty area next to it:

Current Toolbar Buttons Lists buttons currently on the toolbar.
Available Toolbar Buttons Lists buttons that are available but not currently on the toolbar.
AddTo add a button to the toolbar, select the button name from Available Toolbar Buttons list, and click Add.
RemoveTo remove a button from the toolbar, select the button name from Current Toolbar Buttons list, and click Remove.
Description Shows a brief description of the selected button name from either list.
Move UpTo move up (or to the left) a button on the current toolbar, select the button name from the Current Toolbar Buttons list, and click Move Up.
Move DownTo move down (or to the right) a button on the current toolbar, select the button name from the Current Toolbar Buttons list, and click Move Down.
ResetClick to reset the toolbar to the default ASAP settings.
Close Closes the dialog and saves the settings.
