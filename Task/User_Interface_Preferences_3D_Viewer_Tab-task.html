Enter the number of spheres needed for the maximum subdivision of rays. See Notes below.
Default log Specifies the lower cutoff (in orders of magnitude), relative to the data maximum.
Note:
  • RaySet: The BRO 3D Viewer maps the direction sphere to a collection of triangles formed by successive subdivisions of an icosahedron. Resulting bins have roughly equal solid angles, typically with more even statistical behavior than bins that are formed from conventional theta-phi division of the direction sphere.

The BRO 3D Viewer displays ASAP geometry data that is stored in the vector (*.vcr) file. You can also use the 3D Viewer to:


  • Calculate and view the far-field incoherent radiant intensity pattern (flux per solid angle) of a rayset (written from ASAP with the DUMP command).
  • Display two-dimensional, positional, distribution data files in their proper spatial context (for example, files produced by the SPOTS POSITION or SPREAD NORMAL commands in ASAP).
  • Bin, in volume space, a collection of ray paths that are stored in an ASAP ray history file (generated by the SAVE command in ASAP).
