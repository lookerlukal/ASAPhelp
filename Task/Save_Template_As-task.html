Save Template As

Save or load a template file in the Chart Viewer. This dialog opens from the Chart Viewer toolbar.

You can load different user templates from the pull-down list. The template list is filtered so that only templates that were created for a similar type of chart are available to apply to the current chart. For example, a template created for a bar chart is not available for a line graph.

Use the Default template to revert current chart properties to their original state.
