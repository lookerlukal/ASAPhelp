Display Viewer Plot Display

This topic describes commands that are accessisble from the plot display area in the Display Viewer window.

Right-click the plot display area of the Display Viewer window to view the context menu.


PropertyDescription
User RangeSpecifies the range of the data values that are mapped into the palette.
ContoursSets Contouring status: off, on, overlay, or setup (to open Contour Setup dialog).
IsotropicExtends the picture to the bounds of the window.
CursorsDisplays cross-hair lines over the plot area.
Cursor SizeIncreases definition from least (1) to most (3).
Locate MaximumMoves pointers to position of maximum data value.
Locate MinimumMoves vertical pointer to position of minimum data value.
PaletteOn or off
LabelsOn or off
Frame ColorOpens color palette to select color for frames within the Viewer
