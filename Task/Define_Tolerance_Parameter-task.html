Define Tolerance Parameter

This topic describes how to define a tolerance parameter in the Editor using one of three methods.


  1. Select the parameter text of an ASAP command line and manually enter angle brackets (< >) at the beginning and end of the text string. For example, to define a tolerance parameter for the radius of curvature for the OPTICAL command, enter: OPTICAL Z 0 <1>.
  2. Select the parameter text of an ASAP command line and add the angle brackets automatically using the Tolerance Parameter command. When you select valid numeric text, the Tolerance Parameter button becomes active. Click this button to insert the brackets. After the brackets are inserted at the beginning and end of the text, the Tolerance Parameter button becomes inactive.
  3. Select the Tolerance Table Update button on the Tolerance Editor toolbar to update the Tolerance Data Table.
  • Tolerance Parameter Limits Dialog
    When a tolerance data file (*.inx) is displayed in the ASAP Editor window, the Tolerance Parameter Limits dialog is available on the View menu, or by clicking on the Editor toolbar.
