ELTM Options Tab

On the ELTM Options tab, select the settings for the type of output you want to view (traced rays, contoured plot, or meshed plot) in the Results area, and check the other settings.


OptionsWhen box is selected:
Trace RaysDisplays the geometrical ray spots in results table, including horizontal and vertical angles; flux; and area, pass/fail status, and candela. If unchecked, does not trace rays, and displays a Results.dis file of your choice from a previous run.
Contours Displays data in the results table, using 10 evenly spaced contours. Click Contour Setup dialog box.
Plot MeshOpens a Plot window and graphically plots each ray (if speed is an issue, this option is not recommended).
Prompt on WarningsOpens error message box if warnings occurs. Processing always stops when errors occur.
Bi-Linear Interpolation Returns a more accurate representation of a value at a real position, and assigns a weighted pixel value.
Pixels Affects vertical pixel resolution in the Display Viewer window. See PIXELS (ASAP Command)
Source Flux Normal See NORMALIZE (ASAP Command)
Number of Averages See AVERAGE (ASAP Command)
Plot Window Axis Displays values on the Display Viewer for vertical and horizontal axes, which are perpendicular to the light source, and start/end degrees or position.
Print Template Displays the path and file name for the currently selected print template
Click New to create a new template.
Click Edit to open the ELTM Designer window and make changes to an existing print template.
