Decompose Coherent Field Dialog

Decomposes an existing field into a new set of Gaussian beams.


Position Creates a straightforward spatial distribution of beams, one per input field pixel; used only when pixel size is a few wavelengths or larger.
Direction Fourier transforms the field and creates Gaussian beams, which all have the same widths but different propagation directions.
Delete Old RaysChecked box indicates that old rays will be deleted.
Limit Number of New Beams ToNumber of Gaussian beams created.
Limit Emitting Cone Angle To Degrees to which cone angles are limited for the Gaussian directions. Set to the acceptance cone of the subsequent optical system.
By DegreesSpecifies degrees.
Emit Into Rectangular Region If checked, limits cone angle pattern to rectangular shape, instead of elliptical.
