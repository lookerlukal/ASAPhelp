Tolerance Data Dialog for Builder

The Tolerance Data dialog box displays all the fields in the current Builder spreadsheet that have tolerance parameters.

To view or edit a cell that has a tolerance associated with it, select a field in the dialog box. The corresponding field in the Builder spreadsheet is also highlighted.

To change the values in the Nominal, Min, or Max columns, double-click the appropriate cell and enter the change.

Click OK to accept the change.
