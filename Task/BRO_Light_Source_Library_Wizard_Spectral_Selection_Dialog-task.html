BRO Light Source Library Wizard - Spectral Selection Dialog

This page displays source spectral data on a relative intensity graph for the selected source model only if multispectral mode is selected.


Source ModelDisplays the selected source.
WavelengthLists values for each selected wavelength and its relative spectral weight. See notes on selection.
CancelDeletes previously selected points that are displayed on the spectral list.
FinishClick Finish to sort selected wavelengths and store them in a text file, RaysetName_SPECTRUMLIST.TXT.
BackClick Back to return to a previous wizard page, or Cancel to close the wizard.
Note:
  • Select the desired wavelengths by clicking anywhere on the red curve on the Lamp Spectra chart. Selecting a point on the curve adds the wavelength and index of the selected point to the Wavelength table on the right of the chart.
  • To manually enter points within the minimum and maximum wavelengths displayed in the spectrum, click the Wavelength table.
  • To delete selected entries from the Wavelength table, right-click an entry and select Delete.
  • To modify the wavelength, right-click an entry in the Wavelength table and select Edit.

Rayset naming convention

Multispectral sources: filename_wavelengthNM_grouptotalnumberofrays.dis.

Example:

MYPW09_450NM_5000K.DIS

MYPW09_520NM_5000K.DIS

MYPW09_611NM_5000K.DIS
