Quick Start Toolbar - Sources

The Sources button on the Quick Start toolbar opens the Sources page--a catalog of all the sources available on your system.

Pause the mouse over a source to view a brief pop-up description.

Double-click a source to display the BRO Light Source Library dialog box.

This is the same dialog box that is accessible from the Rays menu by selecting Create Sources> Use BRO Light Source Library.

When you download a new source, ASAP automatically adds it as a node to the Sources tree.
