User Interface Preferences - Plot Viewer Tab

This topic describes preferences that are applied globally to the Plot Viewer and Chart Viewer.


Stretched window mapping Select to force plots to completely fill the plot window.
Reverse Video Select to switch background color from black (default) to white in the plot window.
Pause at end of each plot If checked, pauses plot window after plot is complete. Click Continue when ready to proceed with next plot. If this setting is changed, restart ASAP to activate the new setting.
Use stroke (line-draw) font Select to alternate font selection.
Reset to home Select (default) to reset window zoom position.
Relative line thickness Select to set the default line thickness in the plot window.
Number of display windows Enter the maximum number of windows.
Number of plots per display window Select to enter number of plot windows that can be displayed (default is 1).
Optimal Plot Size Set the size of the plot; default is 500 pixels
Use chart viewer for displaying graph Select to substitute the Chart Viewer, which produces output suitable for color presentation. If not selected, ASAP will display graphs in the Plot Viewer.
