Modify INR file for Tolerancing

This topic describes how to open an INR file for tolerancing purposes.


  1. Open an existing *.inr file, select File> Open or right-click INR Input in ASAP Workspace and select Open.
  2. Select the Save As command to save the file to an *.inx file type. After the file is saved, ASAP opens the Tolerance Data Table at the bottom of the window.
  3. Define the tolerance parameter and compensator parameter. You may define any number of tolerance parameters, but only one compensator parameter can be defined.
  4. Define Figure of Merit (FOM).
    Note: All FOMs must be placed at the beginning of the *.inx file using the format: FOM ‘GrabString’ (similar to the $GRAB macro). “FOM” cannot be used as the name of a FOM function (that is, ‘GrabString’ cannot have the name, “FOM”).
  5. Click the Tolerance Table Update button on the Tolerance Editor toolbar.
  6. Enter tolerance data values for the applicable columns into the tolerance data table. Tolerance parameter limits are displayed in applicable columns.
  7. Save the tolerance data file.
