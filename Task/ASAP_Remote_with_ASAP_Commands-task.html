ASAP Remote with ASAP Commands

Two types of commands are supported for Remote: common Windows commands, and ASAP commands.

Right-click the remote tab (next to the Local tab) on the Command Output window to access ASAP commands that apply to the remote machine. Available commands include: Close Session, Stop Processing, END Command, Restart Kernel, and Properties, as illustrated and described below.




Close SessionTo close a server, right-click the server tab and select Close Session.
Stop ProcessingTo stop a kernel process from continuing, click Stop Processing.
END Command

To partially reinitialize (remove geometry and ray files, but not library files) to a clean start during an ASAP session, click END Command.

NOTE: $GRAB is not available once this feature is enabled, nor is previous information stored in the kernel's memory.

Restart Kernel

To initialize all variables in the kernel to a clean start, click Restart Kernel.

CAUTION: Undo does not apply to this command.

PropertiesDisplays the Properties box for the active remote machine with the following information:
Number of remote kernel processesValue may be greater than 1 if there is more than one user connecting to the remote machine.
Number of ASAP processesApplies to ASAP processes that are currently running on the active remote machine.
Available disk spaceApplies to each fixed drive for the active remote machine.
Note:

Commands that are run in ASAP, either from a document window or from the Command Input window, apply only to the currently active computer, as indicated by the active machine tab (local or server). The active tab is white in color.
