Profiles Dialog
Profiles Dialog

Title Optional title for plot (up to 64 characters).
PROFILE Parameters
Number of Slices Plots n (number of ) slices of each object by planes parallel to the window and between depth coordinates.
Pixel Resolution of Plot Plots designated resolution value.
Overlay Next Plot If checked, overlays next plot with current plot. Useful for combining system plots with ray trace plots, multiple spot diagrams, and other plots.
Override Color If checked, one color is selected for all plot lines, rather than default mode of multi-colored plot lines.
