BRO Light Source Library Wizard - Rayset Options Dialog

Using the Light Source Library Wizard's Rayset Options page, select an option to create a rayset or write ASAP script commands to a template.


Option 1:Create and save new rayset(s) for a selected source.
Create and save raysetsRun this option first if you need to create a rayset for use in Option 2.
Option 2:Requires an existing rayset.
Write ASAP script commands to template, using existing rayset(s)Constructs ASAP commands that are needed to recall source geometries at a later time, and apply them to existing raysets. No ASAP commands are run at this time. Builder files cannot use these templates.
Option 3:Automatically combines Options 1 and 2 in sequence.
Perform Option 1 and Option 2
Next, Back, or CancelClick Next to proceed, Back to return to a previous wizard page, or Cancel to close the wizard.
