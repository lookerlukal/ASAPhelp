ELTM - Text Properties Dialog

The Text Properties dialog box is associated with the ELTM Designer window.


PropertyDescrition
TextEnter or edit the text for the box.
FontDisplays the selected font. Click Change to open the Font dialog box.
UnderlineSelect to underline the text.
Text ColorSelect a text color from the drop-down list.
Background ColorSelect a background color for the text control from the drop-down list.
Text AlignmentSelect the type of alignment: left, right, and centered. Text is always centered vertically.
Show BorderSelect to display a border around the text control.
Border SizeSelect the border thickness. Options are 1 to 5, with 1 being the thinnest border.
Border ColorSelect the color from the drop-down list.
