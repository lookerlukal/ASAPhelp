CODE V File Conversion

ASAP OPTICAL+ and ASAP ADVANCED include the fully integrated capability of importing CODE V® files.

Open this file type in ASAP from the Open dialog box on the File menu.
  • CODE V to ASAP Translation, Step 1
    The first step in the translation process involves selecting an *.SEQ file written by CODE V. You can either type the name of the file in the box below Select CODEV Lens (sequence) File, or click BROWSE. Either of these actions will display a FILE OPEN dialog, from which you can select an SEQ file.
  • CODE V to ASAP Translation, Step 2
    The second step in the translation process is to select the ASAP filename, and to set several translation options.
  • CODE V to ASAP Translation, Step 3
    The final step in the translation process is to choose the zoom position(s) to translate and, optionally, the surface for the origin of the global coordinate system.
  • CODE V to ASAP Translator Restrictions
    While this translator supports most of the optical system modeling capabilities of CODE V, there are inevitable differences in the database structure and intended applications, so that a precise translation is not always possible. In most cases, the equivalent functionality can be accomplished in ASAP with user intervention. Specific features and limitations of this translator are described in this topic.
