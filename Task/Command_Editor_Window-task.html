Command Editor Window

This topic lists keyboard commands available when using the Builder Command Editor window.

While in the Builder's Command Editor window, the following keyboard commands are available:

Press:


  • Ctrl-Enter to create a new line in the Editor
  • Enter to exit the Editor and save changes
  • Esc (escape) to exit the Editor without saving changes
