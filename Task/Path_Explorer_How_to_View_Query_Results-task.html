Path Explorer - Viewing Query Results

This topic describes how to view query results in the Path Explorer.

Pre-requisites:

You have combined your queries.

To view query results using the Path Explorer Query Builder dialog:

  1. Select Display Results in 3D Viewer on the Query tab or Query Combiner tab before you perform a query (or click the Run Query button), or
  2. On the main ASAP toolbar, click the 3D Viewer button . The 3D Viewer window displays the visualization of the VCR file and the current query results that are contained in the currently active path-set.
