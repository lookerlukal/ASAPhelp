Polarization - Setting Poincaré Options

This topic describes how to set options for displaying a Poincaré Sphere.

Pre-requisite:

A Poincaré sphere must be displayed in the 3D Viewer graphics pane.

  1. Select an item in the 3D Viewer Tree View pane with a right-click. The 3D Viewer context menu displays.
  2. Select the Poincaré Sphere Options command. Displays the Poincaré Sphere Options dialog.
  3. Select settings in the dialog and select OK.

Selected options are applied, and the Poincaré Sphere is redrawn in the 3D Viewer graphics pane.

Post-requisites:

None.
