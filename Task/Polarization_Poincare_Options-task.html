Polarization - Setting Poincare Options

This topic describes how to set options for displaying a Poincaré Sphere.

Pre-requisite:

A Poincaré sphere must be displayed in the 3D Viewer graphics pane.

  1. Right-click the 3D Viewer Tree View left pane with a right-click. The 3D Viewer context menu displays.

  2. Select the Poincaré Sphere Options command. Displays the Poincaré Sphere Options dialog.
  3. Select settings in the dialog and select OK.

Selected options are applied, and the Poincaré Sphere is redrawn in the 3D Viewer graphics pane.

  • Poincare Sphere Options Dialog
    This dialog is accessible from the 3D Viewer context menu in the Tree View pane. It is enabled when polarization data is present in the current DIS file, an includes options for displaying the Poincaré sphere. Its fields are described.
