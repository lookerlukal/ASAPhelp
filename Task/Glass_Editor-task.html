Glass Editor

From the Quick Start toolbar, double-click a glass name on the Glass Catalog tab to open the Glass Editor, where you view the dispersion function for the selected glass.

Select a glass from the tree list in the Glass Editor to view its dispersion function.


New GlassClick to add a glass name to the tree list. The New Glass Editor dialog box opens.
New PairsClick to assign wavelength/refractive index pairs. The New Pairs dialog box opens.
DeleteClick to remove a glass name from the tree list.
