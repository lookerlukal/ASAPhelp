Interactive Scripts Menu

When you right-click an interactive script file name on the Interactive Scripts tab of the Quick Start toolbar, a context menu displays these commands: Open, Run, Add New, Remove, Delete, and Properties.

Depending on the type of script file you select, the results vary. The table below indicates the commands that are available with each type of script file.


Type of Script File Menu Commands
User-DefinedAll menu commands

Each menu command is described in the table below.


Menu CommandAction
OpenOpens the selected interactive script file in an ASAP Editor window.
RunRuns the selected interactive script file, which must be open in the ASAP Editor window.
Add New Opens the Open File dialog box. Browse to an INR script file to add to the tree view.
RemoveRemoves the selected script from the tree, and keeps the file in your install folder.
DeleteRemoves the script from the tree and deletes the file from your install folder.
Properties Opens the Properties window to display properties associated with the selected script.
