Graph Window Properties Tab

The tab on the BSDF Fit Graph window properties dialog includes the following options:


OptionDescription
Coordinates rangeEnter the x and y ranges of the view. The x-axis spans from x1 (left) to x2 (right), and the y-axis spans from y1 (bottom) to y2 (top).
Point marksDraws a square mark at every x, y point location.
Mouse coordinatesDisplays current x, y value of the pointer position as the pointer moves.
AxisDisplays axis lines in the background of the Graph window.
