polar intensity plot properties
Rayset Optional Properties Dialog

Sets the sphere size and position, polar reference grid, and color for specific *.dis files. To access this dialog box, right-click in the left tree pane of the 3D Viewer and select Optional Properties.

Sphere Size and Position


CenterSets the center at which the radiant sphere is drawn.
RadiusRadius of the radiant sphere. The sphere size and position have no impact on the actual directional data that is plotted.
Reset to DefaultsBy default, the center is set to the rayset focus, and the radius is set so that the sphere encloses all ray positions.

Polar Reference Grid


OrientationWhen using the mouse to query values on a sphere, the selected option (from the drop-down box list) controls which axis is used as the pole for the polar reference grid, and for the displayed angles.

Polar Angle Range

Azimuthal Angle Range

Specifies ranges for the displayed angles when data is queried with the mouse.
Color SchemeSpecifies the colors used when drawing output as a raster image. Reverse color controls whether the order of the colors is reversed.
Note:

A polar grid serves as a reference, and you can change its orientation (polar axis). While the grid has no impact on how data is binned, it does influence the way polar and azimuthal angles are reported when you query data with the left mouse button. By default, the sphere is centered on the rayset's focus, and is sized to enclose all ray positions. You can also change the center and radius.
