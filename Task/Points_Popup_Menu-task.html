Points Popup Menu

Right-click in the BSDF Fit Utility Graph window to display the popup menu.


OptionDescription
Delete Removes selected x, y point from the table. Harvey graph points are automatically created and refreshed by the utility, based on the Harvey parameters.
Copy to Clipboard Copies x, y data for all graphs to the Clipboard as ASCII text.
Properties Opens window for changing graph properties and axis (title and viewing precision of displayed points).
Hide Points Window Removes the window from view.
