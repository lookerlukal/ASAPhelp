Glass Catalogs for Refractive Media

The glass databases accessible in ASAP include: French, Hikari, Hoya, Hoya New, Ohara, Schott, Schott New, Sumita, and Unusual. The catalogs are in text files named *.CAT. The data for a specific glass consists of the glass name and six interpolation constants. No transmission, absorption, cost, staining data are contained in this database.

To access a glass catalog from the Quick Start Toolbar, click the Glass Catalog button on the toolbar. If the toolbar is not visible in your ASAP window, select Quick Start from the View menu. Select the glass catalog and then double-click the medium you want to use.

To access a glass catalog from the Command Output window:

Type the catalog filename connected with an underscore to the glass name directly on an INTERFACE command; for example, SCHOTT_BK7.


  • If ASAP cannot find a requested medium referenced on an INTERFACE command in the currently defined MEDIA table, it automatically searches the glass catalogs.
  • If ASAP finds the requested medium (glass) in the glass catalogs, ASAP internally updates the MEDIA table with the refractive indices and name of the manufacturer's glass computed at the wavelengths specified by the previous WAVELENGTHs command.
  • If no WAVELENGTHs are specified, ASAP computes and stores the refractive index of the glass for a wavelength of 0.58756 microns.
  • If the wavelengths are subsequently changed later in the analysis, ASAP does not return to the glass catalogs to recompute the refractive indices. It linearly interpolates between refractive index data stored in the MEDIA table.
  • If ASAP cannot find the glass in any catalog, the program issues the error, NO MATCH FOUND TO A GIVEN NAME OR ABBREVIATION.

A manufacturer's glass may be used just like any other medium in the specification of INTERFACEs and COATINGs.

Note: Breault is not responsible for the accuracy of data in these glass catalogs since they are obtained from the manufacturers.
