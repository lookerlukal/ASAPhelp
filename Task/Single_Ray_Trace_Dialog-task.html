Single Ray Trace Dialog
Single Ray Trace Dialog

Defines and traces a single ray.

Trace Single Ray:


Global Position Coordinates X,Y,Z Sets starting coordinates of ray.
Global Direction Vector A,B,C Sets initial direction vector of ray.

Ray Options:


Print Ray DirectionsPrints ray coordinates and direction cosines at each intersection.
Reset Search Strategy After RaytraceResets allowable object intersections for future rays according to the sequence of objects encountered during the current single ray trace.
Keep Ray Data in StorageAdds ray to current ray storage after ray tracing; otherwise deletes.
Plot/Plot OptionsSee Plot Dialog for option descriptions
