User-Definable Routines - Changing Defaults

To change the default operations of the user-defined routines, you must edit (and most likely debug) the file, userprog.f90, located in the ASAP installation directory.

The userprog.f90 file contains the FORTRAN source code for all available user-defined routines. Review the supplied source code from BRO in this file for programming and implementation ideas.

Note:

You are solely responsible for implementing and debugging any code changed or added to userprog.f90.

This procedure replaces the existing userprog.dll supplied with ASAP. You may want to rename the existing file prior to creating a new one. BRO recommends that you leave the userprog.dll file in the installation directory to avoid confusion. However, you can place it in the current directory or any other directory in the path.

To recompile and create a new *.dll with Compaq Visual Fortran, enter: 

df userprog.f90 /dll /libs:static

This input creates a new file, userprog.dll, which is linked to ASAP during runtime.
