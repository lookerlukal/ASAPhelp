Hardware Key

The SafeNet Sentinel SuperPro or SuperProNet hardware key, included with your ASAP shipment, provides a unique identification number for authorizing software use. ASAP communicates with the Sentinel SuperPro or SuperProNet key to determine if the user is authorized to use the program. The Sentinel SuperPro driver is 'Windows Hardware Quality Labs' certified for Windows® 2000 and Windows® XP (32-bit).

The key looks like a small block with a printer connector on each end. During installation, ASAP installs a driver so that the program can communicate with the hardware key, and then automatically configures the driver to use the first parallel port. The Registration dialog box, on the ASAP Help menu, includes the identification (ID) number of your hardware key.

Note:
  • You can install ASAP on more than one computer; however, you can run ASAP on only one computer at a time by connecting the hardware key to the computer on which you plan to run ASAP.
  • If you have entered your software authorization keys while logged into one computer network account and then log into a different account on the same or different computer, you need to re-enter the keys. Breault assigns the keys to a specific user and matches the software keys to your hardware key.
  • Confirm that your computer system date is set correctly. If it is not, the keys could fail.

Tips


  • Plug the hardware key into the printer port as the first key.
  • To configure the driver, you must log in as Administrator.
  • If you have more than one parallel port, you may need to configure the driver manually by using the utility, Setup Hardware Key Driver. An icon for this utility is created when ASAP is installed.
  • Set the port for bi-directional or high speed. If this causes a problem, try standard settings.
  • Use the latest version of the Sentinel SuperPro or SuperProNet driver. See the SafeNet Web site.
  • If you experience print problems after upgrading your computer, try to reinstall the Sentinel SuperPro or SuperProNet driver or check the SafeNet Web site, which also has a troubleshooting page where you can also download the most current drivers and find other support information.
  • If you are installing ASAP on a Wide Area Network, please contact BRO Customer Service.
  • Hardware Key Replacement or Repair
    The Sentinel SuperPro or SuperProNet hardware key, included with your ASAP shipment, is worth the full purchase price of the software. Breault recommends that you insure the hardware key as you would any other personal or business asset of comparable value.
