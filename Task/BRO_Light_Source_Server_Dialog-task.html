BRO Light Source Server Dialog

Describes the fields on the Light Source Server dialog.

Note:
  • Your ASAP maintenance must be current to access the library and distribution files from the BRO Light Source Server.
  • Connection to the Internet is required before the download process can be initiated.

FoldersEach folder contains library and help files, and the folders are categorized by source type.
Sources Lists the light sources for the selected folder, which are available on the BRO Light Source Server. Select a source to display the related files available for download ASCII text (*.lib), Help (*.hlp), and binary distribution (*.dis) files (see Note below). Sources that have been updated on the server are displayed with the green lightbulb icon. Sources that have not changed are designated with the yellow lightbulb icon.
Files to DownloadSelect files you want to download from the Server for the selected source.
View ModelClick to display a drawing of the selected source model.
CancelClick to close the dialog box, and return to the BRO Source Library dialog box.
DownloadClick to connect to the BRO Light Source Server to download the selected file(s). A status box displays download progress.
Note: ASAP cannot perform any other work during this download.

Click Cancel to stop the download and return to the BRO Light Source Server dialog box.
