ELTM Tests Tab

Select the test on the ELTM Test tab that you want to apply to your geometry. The list includes over 75 supported test standards (SAE, FMVSS, and ECE) and a brief description of each.


Available Lighting TestsLists the supported tests; select the test you want to apply to your geometry. A description of the selected test is in the right column.
The table in the Select Test dialog box lists the applicable Test Point, Max Candela and Min Candela for the selected test.
