Builder Find Command Dialog

This topic describes ow to find a command on the Buider context menu.

To find a command on the Builder context menu, select Find at the bottom of the menu. The following Find Command dialog window opens.



From the Find Command dialog box, enter the name of the command you want to find in the Command Name box and click Find.

The tree menu in this dialog window opens to the command you are searching for, and highlights it.

Double-click the command name in the Find window to insert the command in the Builder window.

Note:

If you try to do another search from the same open Find Command window, you may get an error message if the command you are searching for precedes the last one on the tree. In this case, place your cursor at the top of the tree and search again.
