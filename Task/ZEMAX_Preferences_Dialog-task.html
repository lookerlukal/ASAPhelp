ZEMAX Preferences Dialog

Prior to initiating the translation of a ZEMAX file in ASAP, you need to set the preferences for lens design data, surface type options, and output options. This dialog box is accessible from the ZEMAX StartUp dialog box.

ZEMAX Lens Design Data


Basic Surface DataOutputs basic system description – See Table 1 for an example.
Surface Aperture Outputs surface size data - See Table 2
Surface Global CoordinatesOutputs projection of local surface axes on to Global Coordinates - See Table 3
Surface Transformation Variables Outputs surface transformation variables – See Table 4
Surface Transformation MatricesOutputs the surface Rotation Matrices and vertex coordinates – See Table 5
List Duplicate Surfaces Outputs a list of duplicates surfaces. Duplicate surfaces are surfaces that have identical vertexes, surface shape, and rotation matrices. (no example is given).

Surface Type Options


Output Cordbrk SurfaceCoordbrk surfaces are not used in ASAP. If unchecked (the default), coordbrk surfaces are not listed in the translator text output file. If checked coordbrk surfaces are listed in the translator text output file.
Output Artificial Surface>Artificial surfaces are not used in ASAP. If this feature is not selected (the default), the surfaces are not listed in the text output file. If selected, artificial surfaces are listed in the text output file.
Output Duplicate SurfaceBy default, only the first occurrence of any duplicate surfaces are listed in the text output file. If selected, all occurrences of duplicate surfaces are listed in the text output file.

Output Options


Default OutputOnly Basic Surface Data is stored in the output text file.
Full Output If selected, all the output options shown under the "ZEMAX Lens Design Data" and "Surface Type Options" are stored in the text output file.
No OutputIf selected, none of the output options shown under the "ZEMAX Lens Design Data" and "Surface Type Options" are stored in the text output file.
Select Output OptionsIf selected, only the selected options under the "ZEMAX Lens Design Data" and "Surface Type Options" are stored in the text output file. Select the options you want to include.

Misc


Add 3D View to ASAP File

If selected, the following two ASAP commands are added to the INR file:

PLOT FACETS

$VIEW

OK Click to accept the changes and close the dialog box.
CancelClick to close the dialog box without accepting the changes.
ResetClick to reset preferences to the default settings.
SaveClick to save the changes.
