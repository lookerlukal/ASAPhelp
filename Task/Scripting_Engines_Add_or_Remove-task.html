Scripting Engines - Add or Remove

To add or remove a script language engine for use in the ASAP Editor, use this dialog.

Script Engines

Configured script engines include: ASAPScript, VBScript, JScript, and PerlScript. You can add other script engines, like Python, if the *.stx file is available for the scripting language in asap install folder\bin\stx.

Add inserts an undefined, new script engine to the list in the Script Engines column.

Remove deletes the selected script engine.

Properties opens the Script Engine Properties dialog box for configuring the script engine.

Move Up/Down changes the position of the script engine on the list of in the Script Engines column, which is reflected on the Configure Script Engines drop-down list on the Editor toolbar.

Click OK to accept the changes and close the dialog box. Click Cancel to close the dialog box without accepting the changes.
