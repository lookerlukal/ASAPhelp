New Glass Editor Dialog

Use the New Glass Editor dialog to designate properties for a glass listed in the glass catalog of the Quick Start toolbar.


Glass NameInsert a custom name for the new material
Dispersion FormulaSelect the dispersion formula from the drop-down list (Schott, Sellmeier, Herzberger, Conrady, or Cauchy) to apply to the new material.
Formula Constants Enter the applicable constant values in each box (for example, A1 to A8, or n0 for Conrady). The number of required formula constants may vary depending on which Dispersion Formula you select.

Sellmeier, Schott, Herzberger = 6 constants

Conrady = 3 constants (where the first item, n0, is a user-defined index)

Cauchy = 2 constants

Note: You must enter at least one constant before you can add any wavelengths.

Wavelength/Refractive Indexes Enter your own wavelength values within a 350 nm and 2500 nm range. Click Add to include the value in the Wavelength column. ASAP automatically inserts the corresponding value in the Refractive Index column.
OKAdds the new material and updates the user.cat file in your asap\doc directory.
CancelCloses the dialog box without making any changes.

Tip

You can delete a wavelengths by selecting the line in the grid and pressing the Delete key.
