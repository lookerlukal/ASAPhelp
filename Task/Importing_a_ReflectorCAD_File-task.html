Importing a ReflectorCAD File

You can use ASAP to check the output of a reflector you created in ReflectorCAD. Follow instructions in ReflectorCAD Help for changing appearance-related settings in ReflectorCAD for output appearance and contouring.

To import the ReflectorCAD file to ASAP, the file must be converted to an INR input file from ReflectorCAD. Use the Export > To ASAP command in ReflectorCAD on the File menu to do this step.
