3D Viewer Window

The BRO 3D Viewer window consists of a navigation pane (tree view, left), graphics pane (object view, right), and a toolbar (top). This topic describes these window elements.




ASAP menus

When the 3D Viewer becomes the active window, the File, Edit, 3D View, and Help menus in ASAP dynamically change to include related commands.

Tree View (left pane)

Context menu with shortcut keys:

Lists all the currently open files in the 3D Viewer in an expandable tree hierarchy.

A right-click context menu of commands is accessible in this pane.

The Polarization Marker Legend and Polarization Flux Legend are displayed in this pane.

Object view (graphics pane)

Displays the object(s). To rotate an object, right-click and drag the mouse. Right-click object to view its name. To move an object, press Shift and drag the mouse.

3D Viewer toolbar:

Adds files to tree view

Print graphics

Copies to Clipboard

Switches background between white and black

Switches rotational axes on and off. Color order of RGB on button corresponds to XYZ axes.

Switches standard view

Select triangular arrow on button to view menu.

Changes view mode of objects or assemblies.

Select triangular arrow on button to view menu.

Opens Viewer Controls dialog, which includes controls for ambient light, zoom, shift, and rotate.

Displays bounding box when performing a rotate, shift, or zoom with the mouse.

Switches Flux on and off

Poincaré Sphere Options toolbar (disabled if there is no active Poincaré sphere)

Degree of Polarization (DOP) slider: specify a DOP value of 0 (completely unpolarized light) to 1 (completely polarized light) for viewing DOP of rays within the current ray set displayed in the graphics pane.
