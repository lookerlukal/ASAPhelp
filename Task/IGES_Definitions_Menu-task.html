IGES Definitions Menu

IGES translation involves a Definitions menu for setting properties.

The Definitions menu activates a submenu with the following entries:

System Units

Selects the optical system unit of measurement. The default is the unit of measurement contained in the IGES file.

Wavelength Units

Selects the optical wavelength unit of measurement. The default is the same as the optical system unit of measurement.

Media Table

Defines the optical wavelengths and refractive indices of any optical material (such as BK7 glass) that you may have in your optical system. These media are used in the Object list in the main dialog. For more information, click Help in the media table dialog box.

Coating Properties Table

Applies coatings to objects in the Object list on the main menu when you define reflection and transmission coefficients at the given wavelengths. For more information, click Help on the Coating Properties dialog box.

Coating Layers Table

Specifies thin film coatings that may be applied to objects in the Object list on the main menu. For more information, click Help in the Coating Layers dialog box.

Object Color List

Displays each object in the drawing on the main dialog in one of three user-selectable colors, depending on the state of the object as it occurs in the Object list. The states are:


  • Selected (highlighted) in the Object list.
  • Removed from the Object list.
  • Neither of the above (default).

Select from a palette of 16 colors for each state by opening the Object Color List dialog box.
