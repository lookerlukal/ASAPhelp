Translator Window for IGES and GTX Files

The translator window for IGES and GTX opens when you initiate an IGES-to-ASAP or GTX-to-ASAP file translation.


File Menu
FinishFinishes building the INR file for use in ASAP.
CancelCloses the Translator.
Edit Menu
ModifyOpens Modify Object dialog box.
MaterialsOpens Materials dialog box.
View Menu
PlotSelect to open the plot window in the dialog box.
Fast DrawDraws only the bounding cube for faster processing.
Log FileOpens a statistical log for the currently translated file.
Object window All items in the Object window are user editable, either singly or as part of a group, via the Edit button on the dialog. The window Includes all parent and child objects. To hide a parent, select the box; all children are also hidden. To hide only some child objects, select applicable boxes. Right-click an object to access the Modify Object or Materials dialog boxes.
Object NameThe object column contains the name of each object in the system model. Objects highlighted in the Object Name column correspond to the area highlighted in the plot window.
CoatingThe Coating column specifies the coating for each object.
MediaThe Media columns specify the media assigned to each object.
ColorThe Color column specifies the color assigned to each object.
OrientationUse arrow buttons to change rotation and translation positions of the plot, and zoom in and out. Alternatively, right-click the object in the display window to change its orientation.
