BRO Light Source Library Wizard - Source Geometry Setup - Create Raysets Dialog

The Create Raysets dialog opens with the Multispectral mode as the default.


Source ModelDisplays the name of the selected source model.
Color Temperature (K)Enter number in Kelvin units (applies only to filament lamps). The default value is displayed for the selected source, if one is available.
Flux (Watts)Enter number in Watts units. The default value is displayed for the selected source, if one is available.
Mode:
Monochromatic Select for monochromatic spectral output.
MultispectralDefault for multispectral output.
Primary and Secondary Axis (only available in Monochromatic mode)Select the axis (+X, -X, +Y, -Y, +Z, -Z, signed literal).
Ray CountEnter the total number of rays.
Rayset Name Enter the name of the rayset file.
NextClick Next to send the commands to the ASAP kernel and continue.
Back or CancelClick Back to return to a previous Wizard page, or Cancel to close the wizard.
Note:

Rayset is created at the global origin (x=0, y=0, z=0).

Flux may be assigned in radiometric or photometric units. A valid CIE analysis can be performed only when the source flux is specified in radiometric units (Watts). The use of photometric units implies a pre-imposed eye response. Since eye response is taken into account during CIE calculations, using photometric flux units (lumens) will lead to erroneous results.

If you selected a dual filament bulb (for example, B1154) in option 1, you are prompted to choose either the high (H) or low (L) filament.
