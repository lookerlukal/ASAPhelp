Creating queries with Path Explorer
Path Explorer Query Builder/ Query Tab

Use the Query tab dialog box to create queries of path-sets.


Query TypeSelect a type from the drop-down menu to create a query with one of the following criteria:
NONENo paths
ALLAll paths in history file
STARTSAll paths in the target of the query, which is either the active path-set or ALL paths if the GLOBAL modifier is used, starting on a specified object
ENDSAll paths, ending on a specified object
SCATTER Paths scattering from a specific object
DIVIDEPaths split from a specific object
ESCAPESAll paths that missed rays
INTERSECTS – object All paths intersecting a specified object
INTERSECTS – volume All paths intersecting a specified volume. Selecting this query type opens the INTERSECTS-volume Query Entry dialog
SPLITS All paths including a specified number of, or upper bound or lower bound on, split events
LEVELS All paths, including a specified number of, or upper bound or lower bound on, scatter events
GENERATION All paths, including a specified number of, or upper bound or lower bound on, the sum of split and scattering events
HITSSpecify number of, or upper bound or lower bound on, hits (that is, intersection events).
Comparison TypeSpecify a logical comparison by selecting one of the following values from the drop-down list:
EQEqual to (=)
LTLess than (<)
LELess than or equal to (=)
GTGreater than (>)
GEGreater than or equal to (=)
Value Specify a positive integer number for the selected logical comparison type.
Object Specify an object name or number for the selected query type, if applicable.
What name do you want to assign to your resulting query?Enter a path-set name.
Apply query to all paths Select to apply test globally to all paths.

Note: With the exception of AND, OR, and NOT, all queries apply by default to the result of the most recent query. This default behavior can be overridden with this option to apply query to all paths.

Display results in 3D Viewer Select to display results automatically in 3D Viewer window.
Results Path-sets A list of path-set names from query results of the current Path Explorer session is displayed.
Run Query Click to run the query after making all appropriate selections on the Query tab.
Close Click to clear all entries and close the Path Explorer Query Builder dialog. This ends the Path Explorer session. See Caution below.

Caution

Keep open the Query Builder dialog if you want to continue performing queries. When you close the Path Explorer Query Builder dialog, the ASAP Path Explorer session has ended, and you can no longer combine previously saved Path-set files.

Note: Depending on which query or comparison type you select, some of the fields may be unavailable.
