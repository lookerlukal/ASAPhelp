New File Type

When opening a new file, you can select a file type from the dropdown list on the New button of the ASAP toolbar.


Supported File Types: Project (*.apf), script (*.inr), library (*.lib), macro (*.mac), screen editor (*.scr), text (*.txt), BSDF Fit Utility (*.his), Builder (*.inx), ELTM, Plot Viewer (*.plr), 3D Viewer (*.vcr) or Chart Viewer (*.plx)
