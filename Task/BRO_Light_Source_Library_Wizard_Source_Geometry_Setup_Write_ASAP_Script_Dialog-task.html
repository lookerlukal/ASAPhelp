BRO Light Source Library Wizard - Source Geometry Setup - Write ASAP Script Dialog

Describes the Source Geometry Setup - Write ASAP Script Dialog in the Light Source Library Wizard.


Source ModelDisplays the name of the source library file selected on the first page of the BRO Light Source Library wizard.
Script TemplateDisplays the name of the ASAP Script Template file selected from the previous wizard page.
Rayset NumberDisplays current rayset files.
Primary and Secondary AxisSelect the major and minor axis (+X, -X, +Y, -Y, +Z, -Z, signed literal).
X,Y,Z Displays positional coordinates.
Rayset NameEnter the name of an existing rayset file name. A different rayset file name can be specified for each entry, or the same name can be used for all entries.
Ray FilesDouble-click the cell to open the Rayset File Selection dialog.
Add, DeleteClick the Add button to add current rayset files or the Delete button to remove files from the rayset list. These two buttons open the Rayset File Selection dialog.
