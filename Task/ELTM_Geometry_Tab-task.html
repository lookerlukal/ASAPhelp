ELTM Geometry Tab

Add, remove, and edit your geometry files to the Exterior Lighting Test Module (ELTM). You can also import a source model from the BRO Light Source Library.

Pre-requisites: Before starting to use the ELTM module, make sure you have set your working directory (File > Set Working Directory) for storing the ELTM files.

Add File Opens the Select File dialog box to select a geometry file. File types include:
ASAP (*.inr)
IGES (*.igs)
OSLO (*.len)
ZEMAX (*.zmx)
You must open a file before proceeding. See the Warning below regarding Builder files.
Edit FileOpens the native editor for the selected file type. For example, if you selected a Builder file (*.enx), the file opens in the Builder Window.
Remove File Removes the selected file from the Geometry tab.
SourcesOpens the BRO Source Library dialog box. To access the Source Library files, you must have the applicable files installed.
UpChanges order of files.
DownChanges order of files.

WARNING

ASAP does not support the inclusion of Builder files (*.enx, *.enz) in ASAP Projects or in the ELTM Module. If you attempt to add a Builder file in a project or ELTM, you will receive a warning message.
