SELECT Rays Dialog

Designates current set of rays that ASAP is to consider in calculations and output. This dialog box is accessible when you right-click in the Display Viewer window.


Select Rays From dropdown list, select ray selection method (ALL, ONLY, EXCEPT, AND OR). Refer to SELECT (ASAP Command) for descriptions.
Options Defines an option to add to the Current Selections list.
Ray Set:
Rays from source numberSpecifies entry type (from dropdown list). Refer to SELECT (ASAP Command) for table of entries.
Entry/Entry'Specifies parameters that correspond to the ray set selection.

Current Selections:


Add Adds the current defined option as the initial SELECT option.
AND/OR Inserts the Boolean AND/OR to the currently defined option as an additional SELECT option.
Remove Removes the highlighted option from the Current Selections list.
