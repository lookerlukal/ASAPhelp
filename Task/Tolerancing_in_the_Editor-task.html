Tolerancing in the Editor

Tolerancing in the ASAP Editor is a two-step process: First, you create a tolerance data file, and then perform the Monte Carlo or Sensitivity analysis.

Before you begin using either step, review the Syntax Rules for Tolerancing in the Editor to assure that your script is properly constructed.

Step 1: Creating a tolerance data file


  1. Create a new *.inx file, right-click INX Input in ASAP Workspace and select New. A tolerance editor window labeled “ASAP Inx” is displayed with empty content.
  2. Enter ASAP commands in the tolerance editor window to define tolerance parameters or compensator parameters.
  3. Define tolerance parameter and define a compensator parameter. You may define any number of tolerance parameters, but only one compensator parameter can be defined.
  4. Define Figure of Merit (FOM). Note that all FOMs must be placed at the beginning of the *.inx file using the format: FOM ‘GrabString’ (similar to the $GRAB macro). “FOM” cannot be used as the name of a FOM function (that is, ‘GrabString’ cannot have the name, “FOM”).
  5. Click the Tolerance Data Table button on the Tolerance Editor toolbar. ASAP opens the Tolerance Data Table at the bottom of the window.
  6. Enter tolerance data values for the applicable columns into the tolerance data table. Tolerance parameter limits are displayed for applicable columns. When you make script changes in the Editor, you must update the Tolerance Data Table by clicking the Update Tolerance Editor button on the Editor toolbar.
  7. Save the tolerance data table and ASAP script.

Step 2: Performing Monte Carlo Analysis


  1. Select the Tolerance Options button on the Editor toolbar. The Tolerance Analysis Options dialog box is displayed. The names displayed are the Figure(s) of Merits that are defined at the top of the *.inx file.
  2. Click the Monte Carlo Analysis button or the Sensitivity Analysis button on the Tolerance Analysis Options dialog box, and click OK.
Note: Before performing a Monte Carlo or sensitivity analysis, a nominal Figure of Merit value may be obtained by selecting the Nominal Only option in the Tolerance Analysis Options dialog box.
  • Tolerance Analysis Options Dialog
    The Tolerance Analysis Options dialog box designates the number of random perturbations you want to perform for the automatic Monte Carlo analysis.
  • Tolerance Data Table
    The Tolerance Data Table contains editable and non-editable cells. The first five columns (Parameter Name, Nominal, Minus, Plus, and Distribution Type) are non-editable, and the remaining cells are editable. You cannot manually add or delete rows from the table.
