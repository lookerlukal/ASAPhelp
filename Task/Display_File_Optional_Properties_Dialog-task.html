Display File Optional Properties Dialog

Sets the color for specific *.dis files. To access the Display File Optional Properties dialog box, right-click in left tree pane and select Optional Properties.


Color SchemeSpecifies the colors used when drawing output as a raster image. Reverse color controls whether the order of the colors is reversed. Colors indicate the height values along a sampled axis: Blue range = low height. Red range = high height.
