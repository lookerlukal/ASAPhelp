User Interface Preferences - Quick Start Tab

Describes user interface preferences for the Quick Start toolbar.


Reload Sources If you manually add a .lib source file to your LightSource directory in your install directory, you need to include the file name in your .lib file. Then when you select Reload Sources on this Preferences tab, ASAP will recognize the file and display your new, manually added source on the Sources page of the Quick Start toolbar. You can set parameters in the Light Source Library dialog to generate a source, but the original source files remain unchanged.
Reload Example Scripts If you manually add an example script to the Examples directory in your install directory, then select Reload Example Scripts on this Preferences tab so ASAP recognizes it and displays your new, manually added script to the Example Scripts page of the Quick Start toolbar.
Note: User interface settings for the Quick Start toolbar are not persistent; you must reset them prior to closing ASAP to reactivate the preferences.
