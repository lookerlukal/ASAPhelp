Logo Properties Dialog

This dialog box is associated with the ELTM Designer window.

The Logo Filename is the name and location of the selected bitmap file. If a file is selected but ASAP cannot find it, the dialog box will show "Default Logo", and the default Breault splash screen is displayed in its place.

Once you have selected a bitmap and clicked the OK button, the bitmap is displayed in the design panel. You can then select the control and re-size the bitmap using the resize handles.

Use the Browse button to look for another bitmap file.
Note: Only bitmap files are supported.
