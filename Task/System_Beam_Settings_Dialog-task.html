System Beam Settings Dialog

This topic describes options in the System Beam Settings dialog.

Setup System Variables


BEAMS
CoherenceDrop-down box includes Coherent, Incoherent - Illumination, Incoherent - Straylight, or nonspecific.
PropagationCorresponds to specified type of coherence. DIFFRACT for Coherent, or GEOMETRIC for Incoherent.
ShapeDesignates beam shape of the individual rays of the currently selected rayset.
PARABASALSets the number of parabasal rays to be traced around each base ray.
Secondary Rays:Indicates number of rays to trace.
Beam Semi-divergence (in Radians)Sets the semidivergence values of each geometrical incoherent beam.
Clip to Object BoundariesIf checked, signals ASAP to check for clipping of the parabasal rays by boundaries and reduce the beam flux accordingly.
XMEMORYControls the amount of ray data stored in the virtual paging ray storage file.
Polarization StateSets the polarization properties for future ray creation.
Polarization DirectionSets the polarization direction.

Directional Amplitude

Orthogonal Amplitude

Sets the complex amplitudes of the two orthogonal polarization components of rays, for future ray creation and analyses.
WIDTHSModifies the default parabasal ray settings, scaling the width of the parabasal rays.
Beam Width Scale FactorFactor by which ASAP scales the actual width of each beam.
Add 1st Order Hermite-Gaussian Beams at EdgeAdds narrow, odd-order Hermite-Gaussian beams to preserve sharpness of overall beam pattern.
