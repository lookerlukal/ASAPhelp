CIE Display Calculations Tab

The Display/Calculations tab is part of the CIE Color Analysis dialog. Use it to designate settings for the display and calculations.


Select Display/Calculations Profile Click to select from the drop-down list the default profile, or select a user-defined profile from the list.
NoteDisplays notes associated with the selected profile, and the calculation and display items associated with the profile.
Add buttonClick to display the CIE Add dialog box.
Remove buttonSelect to remove the current user-defined calculation/display profile in the Select Data Display Profile list. Default profiles cannot be removed.
Clone buttonClick to display the Add dialog box, which shows the display columns of the currently selected data display profile.
Default button Click to reset the item in the Select Data Display Profile list to its default setting.
