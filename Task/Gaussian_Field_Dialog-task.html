Gaussian Field Dialog

Creates a beam set simulating a coherent astigmatic Gaussian mode.


WavelengthMust be previously defined (System Preferences). The field created has total flux identical to a unit peak-irradiance fundamental mode.
Propagation axis Specifies direction of astigmatic Hermite-Gaussian field distribution.
Starting coordinate Specifies rays/beams starting plane.
Waist semi-width units: Specifies unit type for angular divergences.
Spatial semi-width Does not apply degrees nor radians.
Degrees semi-divergence angleApplies designated angle in degrees.
Radians semi-divergence angle Applies designated angle in radians.
Z and Y waist:
X location Determined by the right-hand rule.
Number of rays Number of rays/beams along waist directions.
Mode number Mode numbers along waist directions.
Waist semi-width Specify angular divergence. Avoid a coincident ray starting plane and waist locations.
