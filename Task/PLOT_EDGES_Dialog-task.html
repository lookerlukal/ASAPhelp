PLOT EDGES Dialog
PLOT EDGES Dialog

Title Optional title for plot (up to 64 characters).
Overlay Next Plot Next plot created is overlaid with the current plot.
Override Color Applies the selected color to all plot lines, rather than default mode of multi-colored plot lines.
Plot Edge Control Points If checked, plots curve for all currently defined curves and their control points.

Edge Options:


AllPlots all edges.
Range: Range of edges to be plotted in increments of n.
Start Start of range
End End of range
Steps Plots designated range of edges in n (number of) steps.
Last _ EdgesIf value is entered, plots last n (number of) edges.
