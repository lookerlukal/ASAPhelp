Script Templates: How to Export

You can create and save a script template (*.int) from script in an existing INR file.

You must have an Editor window open to create the script template.

The ability to save existing script to a template file can save time in creating future scripts.

  1. To create a template, select the script in the Editor window.
  2. From the File menu, select Export Template.
  3. Assign a name to the new script and click Save to save the template as an INR file.

The name of the script is added to the Import Template dialog box.
