BRO Light Source Library Dialog

ASAP includes detailed source models that BRO has created for your convenience.

To open a BRO source model in ASAP from the BRO Source Library dialog box, click the Browse button, next to Source Model File and select the file name of the model you want. If the file you need is not on your system, click the button, Download.

Once you select a source model, a description of the model appears in the dialog box. You can then enter or select parameters (see below) specifying the source's position and orientation.

Source Model File:


BrowseSelect a light source file from your local directory. If the file is not on your local directory, you can view a catalog of available sources at the BRO Web page, www.breault.com/software/asap-lightsourcelib.php
Description Describes the parameters of the selected source model.
View ModelYou must have a source model file selected to view the model. Click View Model to see where the origin of the source model is located, and how the primary or secondary axes are defined.

Source Position and Rayset:


Position
Center FilamentClick to position the filament in the center of the source.
X, Y, and ZDisplays values for each axis.

Number of Rays:

(0 to 20 million)

Specifies the number of rays to generate. The actual number generated may differ slightly from the value entered. If you enter zero, only the bulb's geometry is imported to ASAP.

Model Alilgnment Axis


PrimarySpecifies axis and direction with which to align the source's primary axis.
Secondary Specifies axis and direction with which to align the source's secondary axis.
Model Usage MethodSelect method from the drop-down list.

SelectionSelect filament from the drop-down list.
Use Stored Model Flux

Uses stored ray data that was pre-generated with the model, rather than using the complete geometry and sources that make up the model. For this option to be available, stored ray data must be present for the selected model.

The maximum number of rays generated is limited to the actual number of pre-generated rays that are available.

For a significantly faster process, use stored ray data. If you use the full model definition, ray tracing from the source itself is slower.

The disadvantage of using stored data is the absence of the source's geometry on your system. Any effect the geometry might have had on rays, after the rays leave the source, is absent (that is, the source does not cast shadows).

Use Custom Flux ValueEnter your own flux value.
ExitCloses the dialog box.
Create Rays and TraceCreate the rays for the selected source based on the positional settings in the dialog, and trace them. An existing rayset may be used, initializing it to the number of rays desired. In this case, system geometry should be built before using this button. It will automatically trace from the outer envelope of the source and through the entire system. Source geometry may be included as a selection. The other option is to trace new rays from filament or emitter. In this case, the button should be used before creating system geometry (Caution: do not use SYSTEM NEW; RESET when running system geometry file). Source geometry is always included. Run a subsequent TRACE to continue the rays from the outer source envelope and through the entire system.
Create Rays and Distribution Trace rays (from selected filament or other emitter) to the outer envelope of the source, and store the rayset DUMP file (named after the source type and number of rays) in the working directory. Uses the number of rays selected in the dialog, only at the default position of model. Does not use Model Alignment Axes and Position entries. Source geometry is also created at this time. The DUMP file may be used later with EMITTING DATA or the Create Rays and Trace button. The former method is generally used. Manual positioning of the source with SHIFT and/or ROTATE may be necessary to align with system geometry before tracing. This button is generally run independently, or before other system geometry is created. Note: Create Rays and Distribution is not available with arc sources or certain LEDs.
DownloadClick to open the BRO Light Source Server Dialog
New SourcesDisplays the View New Souces dialog box. Click the button to open list of sources created within last six months. If you are current on ASAP maintenance, you may also access the list of sources in the queue for future development. If you are not current, this feature is not available.
Request SourceDisplays the View New Sources dialog box so that you can request a new source for Breault to develop. If you are current on ASAP maintenance, you may receive new source files. Click the button to send an e-mail message to sources@breault.com.
