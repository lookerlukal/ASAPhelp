Window Dialog

WINDOW:Defines the plane of plotting window along the vertical and horizontal axes.
Vertical Min/Max Defines the vertical axis, and the minimum and maximum vertical window coordinates.
Horizontal Min/MaxDefines the horizontal axis, and the minimum and maximum horizontal window coordinates.
Default Window Clicking the button applies default window settings.
Auto ScaleAutomatically scales window coordinate setting.
