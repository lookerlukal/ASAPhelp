This topic provides tips on starting Remote in ASAP.

Remote Process Startup


  • Remote is accessible only from the toolbar button, not from the main menu.
  • To avoid a long network checking process, preselect computers that have ASAP 7.1 or later versions installed.
  • Before running files in ASAP while a remote machine is active, go to File > Preferences > General tab, and deselect Set $PATH when executing INR files.
  • To start a remote session, click on the toolbar (ToolTip: Start Remote), and select a computer from the Browse for a Computer dialog box. The remote computer must, of course, have ASAP installed, but it does not need an ASAP key installed. The remote computer must be configured to allow access to its KernelServ process.
