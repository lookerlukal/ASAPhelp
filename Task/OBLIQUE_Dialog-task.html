OBLIQUE Dialog

Switches graphical output between normal (orthogonal) projection and oblique (nonorthogonal) projection.

OBLIQUE:
Use Oblique Projections If unchecked, specifies orthogonal projections (default).

If checked, specifies nonorthogonal projections.

Settings:Apply only when Use Oblique Projections is checked.
Pivot Around Depth Coordinate Applies third depth coordinate pivot point.
Vertical Rotation in DegreesSpecifies rotation angle in degrees about the vertical axis.
Horizontal Rotation in DegreesSpecifies rotation angle in degrees about the horizontal axis.
