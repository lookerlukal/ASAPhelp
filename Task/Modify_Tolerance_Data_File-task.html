Modify Tolerance Data File

This topic describes how to modify a tolerance data file in the Editor.


  1. Open an existing *.inx file, select File > Open or right-click INX Input in ASAP Workspace and select Open
  2. Make changes to command script in the Tolerance Editor Window.
  3. Select the Tolerance Table Update button on the Tolerance Editor toolbar to update the Tolerance Data Table.
  4. Select Save on the File menu to save the tolerance data file.
