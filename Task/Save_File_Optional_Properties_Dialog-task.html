3D Viewer - save file properties
Save File Optional Properties Dialog

Specifies the log cutoff and color scheme in 3D Viewer graphic window.


Log cutoff Specifies the lower cutoff (in orders of magnitude), relative to the data maximum.
Color SchemeSpecifies the colors used when drawing output as raster image. Pseudo-Spectral is the default. To reverse the order of the list from top to bottom, select Reverse order .

The lower half of the dialog box lists data about the currently displayed file.
