User-Definable Routines

With ASAP, you can create user-definable routines for surface intersection, surface functions, BSDFs, and other features. ASAP is dynamically linked with a library (userprog) of user-definable routines. The library contains routines that ASAP uses for surface intersections, surface functions, BSDFs, and other functions.

The following user-definable routines are included in the source code.
Note: Please refer to the code for examples, program defaults, and integer/real/double precision argument and return value types.
  • User-Definable Routines - Changing Defaults
    To change the default operations of the user-defined routines, you must edit (and most likely debug) the file, userprog.f90, located in the ASAP installation directory.
