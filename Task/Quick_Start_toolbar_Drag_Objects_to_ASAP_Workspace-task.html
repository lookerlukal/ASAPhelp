Quick Start Toolbar - Dragging Objects onto ASAP Workspace

This topic describes how to drag objects from the Quick Start toolbar onto ASAP Workspace.

Pre-requisites:

Objects must already exist in ASAP Workspace before you can drag and drop properties from glass or model catalogs from the Quick Start toolbar.

From the Quick Start toolbar, right-click glasses:
  1. Scatter models (except Diffuser Sheet models), and roughness models.
  2. Drag the item and drop it on an object of the Object tab tree.
    Note:
    • Dragging and dropping onto any item that is not an object, or the parent of an object, has no effect.
    • Properties can be applied only in the object tree and cannot be deleted from it.
    • If you drop onto a parent of an object, the item is applied to all child objects.
