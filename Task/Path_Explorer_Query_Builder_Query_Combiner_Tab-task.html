Query Combiner in Path Explorer
Path Explorer Query Builder/ Query Combiner Tab

The Query Combiner is used to combine found path-sets into composite path-sets.


Available Path-sets Select a path-set from the list of names.
Combination TypeThis field is enabled only after you select a path-set from the list of available path-sets. Subsequent selections give you choices.

Select a logical combination type from the drop-down list to apply to the selected path-set.

ORLogical union between any two path-sets
ANDLogical intersection between any two path-sets
NOTLogical exclusion of the selected path-set
Query PreviewPreview the selected logical combination type path-set script in the read-only box prior to running the query.

Note: This box is cleared each time you select a combination type.

Clear Click to clear the contents of the query preview
What name do you want to assign to your resulting query? Enter a path-set name.
Display results in 3D Viewer Select to automatically display results in the 3D Viewer window.
Run QueryClick to run the query after making all appropriate selections on the Query Combiner tab.
Close Click to clear all entries and close the Query Builder.
