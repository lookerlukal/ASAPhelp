CIE Color Analysis - Getting Results

This topic describes how to start CIE color analysis is ASAP.

Pre-requisites:

Create or open an INR file that includes rays, and run it in the Editor window. You must have a substantial number of rays to see the results in the output display files. The number of rays depends on your application and processor speed.

You can start CIE Color Analysis from either the Analysis menu or Command Input window.

  1. From the Analysis menu, select the Calculate CIE command. Opens the CIE Analysis dialog
  2. Select your settings on the three tabs: Object, Window, and Display/Calculations.
  3. Click the OK button on the dialog. Initiate the calculations.
  4. Alternatively, from the Command Input window, enter the $GUI Calculate CIE command with selected parameters.

  6. Press the Enter key to run the command.
Result:

The CIE window shows the analysis results.
