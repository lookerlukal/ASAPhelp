Projects in ASAP

ASAP uses a file management concept called projects. For easy viewing and quick access, you can organize each project by linking all the files associated with it into a project file. The ASAP project file (*.apf) tracks which files are associated with a project, and the directory in which they are stored.

Creating a Project


  1. From the main toolbar in ASAP, select the triangular arrow next to the New button. From the drop-down list, select Project.
  2. In the New Project dialog box, assign a project name and specify the directory in which you want to store the ASAP project file (*.apf). If the documents are open, answer "yes" when asked to close current document windows.
  3. Although it is not required, BRO recommends that you store your ASAP project file in the same directory as its associated files.
  4. From the File menu, select Projects > Preferences, and select your preferences for the project on the General tab of the User Interface Preferences dialog box.
Note:

The Files tab on ASAP Workspace becomes visible only after a project is created or loaded.

Loading a Project


  1. From the File menu, select Project > Load. From the Open Project dialog, select Files of type: ASAP Project (*.apf), and browse to the directory that includes the project file you want. Click Open.
  2. After the project files open in the ASAP Workspace window, on the Files tab, click a file name to open it in its associated window (for example, Editor, or Plot).
Note:

ASAP does not support the inclusion of Builder files (*.enx, *.enz) in Projects or in the ELTM Module. If you attempt to ad a Builder file in a project, you will receive a warning message.

Opening a Sample Project

Sample projects are complete working examples of optical systems built in ASAP, along with brief documentation in the form of text files.


  1. From the File menu, select Project > Load.
  2. Browse the ASAP\Project\Samples directory.
  3. Double-click a project directory of your choice.
  4. Double-click the project (*.apf) file you want to open.

Closing a Project

To close a project, right-click the project name in ASAP Workspace, and select Close from the pop-up menu.
