SolidWorks Files

If you have SolidWorks (from SolidWorks Corporation) installed and you have ASAP CAD+ or ASAP ADVANCED, you can export geometry files from SolidWorks to ASAP.

To export a file from SolidWorks, use the Save As dialog and select ASAP (*.gtx) in the Save as type drop-down box. If there is an active SolidWorks geometry, an ASAP Export dialog opens. Select one of the three options:


  • Selected Face(s)
  • Selected Bodies
  • All Bodies

Click OK.

To import a GTX file into ASAP, select File > Open in ASAP, and browse to the *.gtx file you want to open.
