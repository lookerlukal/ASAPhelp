Object Properties Dialog

The Objects Properties dialog box sets the properties of the currently selected object . It is accessed from the BRO 3D Viewer, in the object tree-view pane.

Select an object from the left pane, and right-click, and select Properties from the context menu.


ObjectDisplays full name of selected object.
Draw modeSets the current drawing mode. Select from drop-down list box menu (Flat Shaded, Wire Frame, Smooth Shaded, and Highlighted Edges).
VisibleIf checked, displays current object. If unchecked, hides current object.
Bounding BoxMinimum and maximum values for the bounding box that encloses the object(s).
