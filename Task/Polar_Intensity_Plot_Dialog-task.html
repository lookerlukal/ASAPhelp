radial plot in 3D Viewer
Polar Intensity Plot Dialog

The Polar Intensity Plot dialog defines parameters for creating the radial plot intersection plane and y axis direction in the 3D Viewer.


Intersection PlaneParameters for the intersection plane
Angle from AxisDefault = 0
Angle Around AxisDefault = 0
Reference DirectionDefinition of the parameter for the x axis direction
Angle About Intersection PlaneDefault = 0
OKClick to accept the parameters. A polar plot of the data is generated.
CancelClick to close the dialog without generating the plot

Radial data on the polar plot represent flux values at plane/radiant-sphere intersection points,. Angular data are the angle the intersection points make with the x axis direction.

Note:

The default position of the intersection plane and the x axis is the Y-X plane, and the x axis = (1,0,0). This corresponds to Angle From Axis = 0, Angle Around Axis = 0 and Angle About Intersection Plane = 0.

The definition of the intersection plane is equivalent to first rotating the default intersection plane about the x axis (1,0,0), (Angle From Axis), followed by a rotation about the z axis (0,0,1), (Angle Around Axis).

The definition of the reference direction is equivalent to drawing a line on the X-Y plane prior to rotation of the plane, and then rotating this line about the normal of the rotated plane (Angle About Intersection Plane).

See the figure and examples below.



Example

To create a plane with a local x axis = (0,1,0), y axis = (0,0,1) and normal to the plane = (1,0,0), enter the following parameters in the Polar Plot Intensity dialog:

Angle From Axis = 90, Angle Around Axis = 90, and Angle About Intersection Plane = 0

To create a plane with a local X axis = (0,0,1), y axis = ( -1,0,0), and normal to the plane = (0,-1,0), enter the following parameters in the Polar Plot Intensity dialog:

Angle From Axis = 90, Angle Around Axis = 0, and Angle About Intersection Plane = 90
