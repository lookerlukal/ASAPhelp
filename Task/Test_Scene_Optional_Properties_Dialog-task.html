3D Viewer and Test Scene
Test Scene Optional Properties dialog

If you add a test scene file (from the Edit menu, select Add Test Scene) to the current object, the Test Scene Optional Properties dialog box is available. To open it, select Test Scene in the Tree View pane, right-click to open the popup menu, and select Optional Properties.

Dialog box options for the Test Scene apply to the current test scene, which is a sphere. The values you enter control the position (X, Y, Z), size (radius), and faceting (polar and azimuthal) of the sphere.
