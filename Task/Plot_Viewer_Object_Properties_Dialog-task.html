Plot Viewer Object Properties Dialog

To edit properties of annotated objects in an open Plot Viewer plot, select the existing property setting on the dialog box, and enter the change.


Bottom Distance of the selected object from the bottom of the plot edge.
Fill Color Select the current color to open the color pallette, and select a color to fill the object.
Left Distance of the object from the left of the plot edge.
Line Color Select the current color to open the color pallette, and select a color for the object line.
Line Width Select the width of the object line from the drop-down menu.
Right Distance of the selected object from the right of the plot edge.
Top Distance of the selected object from the top of the plot edge.
