Open Save File Properties Dialog
Open Save File Properties Dialog

Defines the following options for saving ray trace data from the 3D Viewer:


Resolution Enter the X, Y, and Z values.

Size

Automatically size to enclose all ray intersections/positions

Check to perform this option. If you do not check it, enter lower and upper limits for box.
Note:

ASAP currently does not store the ray path for gradient material in the SAVE file.
