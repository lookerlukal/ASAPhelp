Interactive Scripts - Geometry Template Dialog

The Geometry Template (formerly User-Defined Geometry Template) dialog box opens when you drag and drop an INR file onto the Interactive Scripts page of the Quick Start toolbar for the purpose of creating a template. Fill in any missing variables for MEDIA in this dialog box.


MEDIA1Enter the name of the first media; for example, SCHOTT_BK7.
MEDIA2Enter the name of the second media; for example, AIR.
MEDIA3Enter the name of the third media; for example, AIR.
OK Click to accept the variables and close the dialog.
CANCELClick to close the dialog without accepting the changes.
