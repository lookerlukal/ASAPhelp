User Interface Preferences - General Tab

The user preferences described in this topic apply globally to the within ASAP.


Prompt when Exiting ASAPSelect to display message confirming exit request.
Set $PATH when executing INR filesSelect to set the default directory path for future file openings. If unselected, reverts to the previous directory path. When using ASAP Remote, deselect this option.
Load BASIC version of ASAP at startup

Select to reduce memory requirements for all ASAP products except ASAP lite.

Note: The BASIC version is equivalent to ASAP lite.

Reset window environment to ASAP defaults Select before making temporary changes to environment if you want to return to default settings.
Show splash screen at startup Select to view the ASAP splash screen each time you start the program.
Default Setup FileDesignates a file to automatically send to the kernel every time an END command is issued (either via the Command Input window or via the toolbar button). For example, the file could contain portions of an *.INR file that you frequently use.

Projects:


Automatically load project on startup Loads the last-run project immediately when ASAP opens.
Auto Save Project Select to automatically save window placement from last time opened.

Clear console window

Default Project Directory

Displays the current path for the default directory, to which files will be saved. Click Browse button to change the path to a different directory.
Editor: Show Command Tooltip Select to view the pop-up command tooltip window as you enter commands in the Editor window.
File Extension SupportSelect file types to include in Windows Open dialog. Windows will open ASAP for viewing these file types.
