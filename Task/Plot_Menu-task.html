Plot Menu

When a Plot Viewer window is open, the main ASAP menu bar changes dynamically to include commands related to this window.

On the File menu:


Open. . . Selects an ASAP plot file. Once a file is selected, ASAP copies the contents of the file into the Plot window.
Save As. . . Saves the contents of the Plot window to a new file. The Page Range area includes options for saving plots as separate files.
Print. . . (Windows) Selects various options to print the contents of the Display window to the Windows printer.
CloseCloses the Plot window.

On the Plot menu:


Copy to Clipboard Copies the current plot on the Clipboard. Only the portion of the plot visible in the window is copied to the Clipboard.
Write to > HPGL Creates a Hewlett-Packard Graphics Language file.
Write to > Postscript Creates a Postscript file.
Write to > Metafile Creates a Windows® metafile.
Isotropic Forces the plot to maintain its 1:1 aspect ratio regardless of the aspect ratio of the window; otherwise, the plot fills the available space in the window. If the window is not the same aspect ratio as the plot, the plot is scaled differentially to accommodate the window dimensions. By default, Normal Mapping is enabled.
White Background Reverses the background color of the Plot window (that is, white goes to black and vice versa) to match your preference.
Monochrome Forces the plot colors to monochrome.
Stroke Font Uses a fixed, sans serif font.
Tile Positions plot windows in square pattern, rather than overlapping them.
Home Returns to first plot window.
Lock Enables a lock on the current Plot window. If the lock is active, a new plot can’t overwrite the Plot window. Future plots are directed to other Plot windows. The lock remains active until disabled. This feature is useful if you want to compare the current plot window with another one. If only one Plot window is open, the lock is disabled.
Line Thickness Sets the line thickness in the plot window.

Using the mouse:


Shift key + mouse Displays coordinates of the Plot window location when the Shift key and mouse are held down anywhere in the graphics window.
