Quick Start Toolbar - Interactive Scripts

The Interactive Scripts button (formerly named the User INR Scripts) on the Quick Start toolbar, opens the tree view and gives you quick access to interactive and user-defined script files. You can add, remove, open, and run example interactive script files and user-defined script files.

The tree view includes several categories in which you can add script files: Bio-Optical Models (available only with ASAP BIO), Array Enhancers, Optimizers, Lens Creation, and User-defined.

All interactive scripts are stored on your system in the following folders, and each folder contains example interactive script files:


Script Category Script Folders
Bio-Optical Models asap install folder\Projects\InteractiveScripts\Bio-Optical
Array Enhancers asap install folder\Projects\InteractiveScripts\ArrayEnhancer
Lens Creation asap install folder\Projects\InteractiveScripts\LensCreation
User-Defined asap install folder\Projects\InteractiveScripts*

Adding Scripts: If you highlight text in an open document, such as an INR file, and drag and drop it onto the Interactive Scripts page, the Add New dialog opens so that you can add a script on the toolbar.

Note: *For backward compatibility, user-defined files that were created prior to ASAP 2006 are stored in the pre-existing folder, asap install\Projects\UserGeom. You can move these files to the "User-Defined" folder.
