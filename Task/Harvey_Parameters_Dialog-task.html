Harvey Parameters Dialog

Automatic fitting for both the Harvey and Polynomial fit models is accomplished with a down-hill simplex method.



Fitting the Harvey Model

Click the Auto Fit button on the parameters panel to open a dialog box, Harvey Parameters. For both Harvey 1 and Harvey 2, you can choose either to fit only the B, S, and L parameters, or all of them. Values for those parameters that you do not choose are set to zero.

Fitting the Polynomial Model

To begin the fitting process, click the Auto Fit button on the parameters panel. The terms that appear in the grid and are unlocked will be fit. The fit limit is 10 parameters.

Remarks


  • During the fit process, a dialog box opens. To stop the fitting process and display the unresolved fit results, click the Stop button before the fit is resolved.
  • Only those data points for each set of data that appear in the current graph window are used in the fitting process.
  • Fit Button for Model 1 (BSDF Utility)
    Fits the Harvey model 1, based on points selected on the slope line in the graph window.
