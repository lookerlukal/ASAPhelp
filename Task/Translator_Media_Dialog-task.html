Translator Media Dialog

The Media dialog box applies to both the IGES- and GTX-to-ASAP translator. The dialog opens from the Modify Object dialog box.

Use the Media dialog to create a new medium. Enter the name of the new medium and its refractive index value.

Click OK to accept the new nedium, or Cancel if you do not want to add the medium.
