Command Input Window

The dockable and expandable Command Input window is where you enter ASAP commands. You can easily work in the Command Input window, and move among a variety of other windows, such as the Builder or Editor.



Enter commands either by typing the command name or selecting a previously submitted command (click the down arrow on the right of the box to see the list).

To submit commands you enter in this box to ASAP, select Run from the File menu, or press Enter.

The applicable ASAP> prompt is displayed in the status bar. For example, the prompt changes to EDG> if you are running an edge in the Builder, or to DIS> if ASAP is in display mode. For a list of prompts, see ASAP Prompts.

Note:

You can work interactively from the Command Input window, or in the background (at your system prompt) in batch mode. Since the output files are fully compatible with the graphic capabilities in ASAP, you can easily view or manipulate graphics.

Commands that are run in ASAP, either from a document window or from the Command Input window, apply only to the currently active computer (Local or server).
