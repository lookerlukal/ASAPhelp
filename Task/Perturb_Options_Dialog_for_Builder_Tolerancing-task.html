Perturb Options Dialog for Builder Tolerancing

The Perturb Options dialog box designates the number of random perturbations you want to perform for the automatic Monte Carlo analysis.

Select Perturb on the Builder menu to open the Perturb Options dialog box.
Iterations Designates number of random perturbations for the Monte Carlo analysis. Default is 1 time.
Figure(s) of MeritASAP searches for the text string (Figure of Merit) to grab, save, and plot a number that is in the Command Output window. Each Figure of Merit can be viewed in the drop-down list. The number of variables selected determines the number of plots. See Note below for explanation.

Click OK to begin the automatic Monte Carlo analysis. The data set(s) created for the defined Figure of Merit(s) is placed in an FOM*.DIN file in your working directory.

Note: Entries in the Figure(s) of Merit allow any numerical value to be plotted as a function of iteration number. To do this, enter a predefined variable in the Figure(s) of Merit box. For example, if we want a graph of power on an object called DETECTOR as a function of iteration number, we enter a line in the Builder, CONSIDER ONLY DETECTOR, followed by STATS POSITION. ASAP prints the total power in the Command Output window on a line that contains the string Total Flux. To save the value, we select the variable, Total Flux from the Figure(s) of Merit drop-down list, and click the Add button to add it to the Figure(s) of Merit list box. For each iteration, ASAP then grabs the first number (which is the power) after the string Total Flux, saves it, and plots the values for total power as a function of iteration number.
