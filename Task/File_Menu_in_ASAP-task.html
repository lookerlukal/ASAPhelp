3D Viewer and ASAP File Menu
File Menu for 3D Viewer

When the 3D Viewer is the active window in ASAP, the File menu on the ASAP menu bar includes commands specific to this window.


NewCloses all files in the Tree View and contents in the Object View panel (equivalent to starting a new Viewer).
Open

Activates the Open dialog box. Select a file to open in the Viewer. The following file types are supported:

ASAP vector (*.vcr)

2D planar display files from ASAP (*.dis)

Ray set files produced by ASAP DUMP command (*.dis)

Add File to Current ViewActivates the Open dialog. Select a file to add in the Viewer.
CloseCloses currently active *.vcr file.
3D Viewer File PreferencesOpens 2-tab dialog box for setting Ray Set and Ray History
PrintPrints the current Object View scene.
PreferencesOpens User Interface Preferences dialog box. Select tab for the Viewer to set its preferences. Click the Help button for information on this dialog box.
  • Ray History Tab
    Sets the defaults for volume bin resolution and log cutoff in the 3D Viewer. This tab is on the File Preferences dialog box, which is accessible from the ASAP File menu. Select 3D Viewer File Preferences.
  • Ray Set Tab
    The BRO 3D Viewer maps the direction sphere to a collection of triangles formed by successive subdivisions of an icosahedron.
