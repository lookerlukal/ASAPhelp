Display Viewer Tile Dialog

The Tile dialog box opens from ASAP Workspace by right-clicking Display Viewer and selecting Tile from the context menu.


Copy To ClipboardCopies the image of all tiled Display Viewer windows to the Clipboard.
CloseCloses the dialog box.

You can resize the dialog box. Resizing the dialog box also resizes the view.
