ELTM - Icon Properties Dialog

The Icon Properties dialog in ELTM is associated with the ELTM Designer window.

Icon Filename is the name and location of the selected icon file. If a file is selected, but ASAP cannot find the file, "Default Icon" is displayed and the default Breault icon is displayed in its place.

Use the Browse button to look for another icon file.
Note: Only *.ico files are supported.

The icons are standad size, and cannot be resized.
