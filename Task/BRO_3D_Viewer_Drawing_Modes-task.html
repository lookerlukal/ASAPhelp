3D Viewer - Drawing Modes

The 3D Viewer supports five drawing modes for viewing an object.

To select a mode, select on the 3D Viewer window toolbar.

A sample of each drawing mode is shown in the table, along with the keyboard shortcut key for displaying a drawing mode.


Drawing ModeShortcut KeySample View
Highlighted EdgesE
Smooth Shaded S
Flat ShadedF
WireframeW

Lit Wireframe

Similar to conventional wireframe, except the surface normals are included for lighting effects.
