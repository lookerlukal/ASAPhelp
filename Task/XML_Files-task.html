XML Files

Builder (*.enx, *enz), Chart Viewer (*.plx), and Editor tolerance (*.inx) files can be saved as XML files by exporting data to the corresponding XML spreadsheet format.

Select Save As on the File menu.

From the Save As list, select:

Builder (XML) Files (*.enx) to save as an uncompressed XML file.

Chart Files (XML) Files (*.plx) to save as an uncompressed XML file.

Builder (Compressed XML) Files (*.enz) to save as a compressed XML file. This file format reduces file size, and is automatically decompressed when you open it. You can set the *.enz format as a default file preference in the User Interface Preferences/Builder dialog box.

Editor (XML) Files (*.inx) to save Tolerance Editor data to an uncompressed XML file.

Type a file name into the File name list, then click Save.
