PLOT FACETS Dialog
PLOT FACETS Dialog

Title Optional title for plot (up to 64 characters).
Overlay Next Plot If checked, next plot created is overlaid with the current plot.
Override Color If checked, one color is selected for all plot lines, rather than default mode of multi-colored plot lines.
Intra-edge Number of facets along the edge or curve
Inter-edge Number of facets between edges or curves.
Maximum Facets Maximum number of facets per object.
Remove Filler LinesRemoves nonessential facet edges to reduce crowding.
