Volume settings in Path Explorer
Path Explorer INTERSECTS-volume Entry Dialog

The INTERSECTS-volume Entry dialog is displayed when you select the INTERSECTS-volume query type from the Query tab on the Path Explorer Query Builder dialog.

Specify an axis, volume type, and min and max values for each point of the specified volume type.


TypeSelect a value for the volume type from the drop-down list: ASLAB, ATUBE, or ABOX.
Axis, Min, Max

For each volume type, enter an axis, min, and max value of X, Y, or Z.

For each row in the Volume Data Table, enter a different axis value.

If you select the ASLAB type, one row in the Volume Data Table is displayed for entering Axis, Min, and Max.

If you select the ATUBE type, two rows in the Volume Data Table are displayed for entering Axis, Min, and Max.

If you select the ABOX type, three rows in the Volume Data Table are displayed for entering Axis, Min, and Max.
