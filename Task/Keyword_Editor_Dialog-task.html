Keyword Editor Dialog
Keyword Editor Dialog

Customizes the name of an ASAP command, typically by shortening its name. Keywords appear in the ASAP Editor window in the same color coding as command names.


ASAP Commands From the drop-down list of all ASAP commands, select the command name for which you want to create an abbreviated keyword.
AddClick to open the Add Keyword dialog box and add a keyword.
Custom Keyword Lists all the user-defined keywords.
Delete Select a custom keyword and click Delete to remove it from the list.
OK Click to accept changes to the custom keyword list.
  • Add Keyword Dialog
    Enters a new custom keyword to the Keyword Editor dialog box for the ASAP Editor window. The keyword is typically an abbreviation of an ASAP command name, and will be assigned to the ASAP command that you selected from the Keyword Editor.
