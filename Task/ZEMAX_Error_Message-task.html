ZEMAX Error Message

An ASAP message box is displayed during a ZEMAX-to-ASAP translation when ASAP encounters a lens design error condition and stops the translation. See the message box content for specific error information.

  • ZEMAX Field Type Error
    An ASAP message box opens when there is a field type error in the ZEMAX-to-ASAP translation for infinite conjugate systems. The message notifies you that ASAP needs object space field angles.
  • ZEMAX Lens Design Error
    The Lens Design Problem message box displays when the ZEMAX stop surface is buried inside a glass element, which results in an edge violation between the stop surface and a glass surface.
  • ZEMAX Zero Thickness Error Dialog
    The Zero Error dialog box is displayed when a surface in the ZEMAX design has zero thickness. ASAP may have ray-tracing problems when two objects are defined to occupy the same location in space.
