BRO Light Source Library Wizard - Task Flow for Write ASAP Script Commands to a Template

Writing ASAP script commands to a template in the BRO Light Source Library Wizard.


  1. Select Create Light Source Wizardâ€¦ on the Rays menu to open the BRO Light Source Library Wizard.
  2. Select a Source Library file by typing a source library name in the box or by selecting a source library name from the drop-down list. A description of the selected source library is displayed and the Next button is available.
  3. Click the View Model button to display the source model help information.
  4. Click the Next button to go to the Rayset Options page.
  5. Select Option 2: Write ASAP script commands to template and click Next to display the ASAP Script Template Entry page.
  6. Enter a script template name and click Next to open the Source Geometry Setup - Write ASAP Script page. If you selected a dual filament bulb, a warning message is displayed to add an H (High) or L (Low) to the Rayset name so that raysets are found.
  7. Enter positional values for X, Y and Z, change the major and minor and enter an existing rayset file.
  8. Click Ray Files to open the Rayset Files Selection dialog.
  9. Select rayset files and select the OK button.
  10. Select the Finish button.

At this point, ASAP writes a set of script and macro commands to the specified script template file, and opens a script editor window for viewing and editing the script.
