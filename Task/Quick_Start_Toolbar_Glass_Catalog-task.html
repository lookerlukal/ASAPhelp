Quick Start Toolbar - Glass Catalog

The Glass Catalog button on the Quick Start toolbar opens the Glass page—a list of all the available glass catalogs: French, Hikari, Hoya, Hoya New, Ohara, Schott, Schott New, Sumita, and Unusual. You can also add your own glass definitions to the User catalog.

From the Glass page, you can:

Drag and drop the glass onto an object (or parent) to apply the glass, and view the Command Output window to see the commands that ASAP is using to apply the glass to the selected object(s).

Note: If the glass is dropped onto an object that has a coating, and that coating does not have any transmissive properties, a message displays to notify you that the applied glass will have no effect. You may choose to not view this dialog again by selecting Do not tell again in the message box.


  • View the dispersion function of the selected glass by double-clicking the glass name on the page, which opens the Glass Editor.
  • Drag/drop a glass into script. When you select a glass and drop it into script, the catalog and glass name are inserted into the script at the current cursor location. For example: If you select the BK7 glass from the Schott catalog, the text inserted into the script would be “SCHOTT_BK7”.
  • Drag/drop a glass into the Builder. Glasses can only be dropped into cells that have a “Media” heading.
  • Glass Editor
    From the Quick Start toolbar, double-click a glass name on the Glass Catalog tab to open the Glass Editor, where you view the dispersion function for the selected glass.
