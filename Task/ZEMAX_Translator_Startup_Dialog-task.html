ZEMAX Translator Startup Dialog

The ZEMAX to ASAP Translator dialog box first displays when you open a ZEMAX file.


ZEMAX Input File NameDisplays the path for the ZEMAX input file (*.zmx) that will be translated to an ASAP inr file. This path cannot be edited.
ASAP Script File NameDisplays the path for the ASAP input file (*.inr). This path cannot be edited.
Text Output File Name Designates the path for the text output file (*.txt). This path can be edited.
OKCick to accept changes and close the dialog box.
CancelClick to close the dialog box without accepting the changes.
PreferencesOpens the dialog box for setting preferences for data to include in the text output. If you click OK without changing the preferences, ASAP uses the current settings.
  • ZEMAX Preferences Dialog
    Prior to initiating the translation of a ZEMAX file in ASAP, you need to set the preferences for lens design data, surface type options, and output options. This dialog box is accessible from the ZEMAX StartUp dialog box.
  • ZEMAX Output Table 1
    ZEMAX output table 1 displays basic surface data output from the translation.
  • ZEMAX Output Table 2
    ZEMAX output table 2 displays surface aperture data output from the translation.
  • ZEMAX Output Table 3
    ZEMAX output table 3 displays projection of local surface axes on the global coordinates from the translation.
  • ZEMAX Output Table 4
    ZEMAX output table 4 displays surface transformation variables from the translation.
  • ZEMAX Output Table 5
    ZEMAX output table 5 displays surface rotation matrices and vertex coordinates from the translation.
