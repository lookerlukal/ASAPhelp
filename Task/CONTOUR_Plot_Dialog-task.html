CONTOUR Plot Dialog
CONTOUR Plot Dialog

Creates a contour plot of the current distribution file.


TitleInserts title in plot (up to 64 characters).
Levels
Equally Spaced: Levels Specifies the number of equally spaced contour levels.
Specific LevelsLists contour levels with a space between each number. If all values are less than 1, the levels specify fractional contours relative to the full range of the function being plotted.
Color Fill:Specifies resolution level of plot:
None, High, Low Resolution line contour, greyscale, or color map.
Tic Marks/Grid Lines Specifies coordinate tic mark spacing, or grid line spacing for vertical and horizontal lines.
Write Contours to Vector File If checked, writes contour plot to 3D vector file.
