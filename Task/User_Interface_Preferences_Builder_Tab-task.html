User Interface Preferences - Builder Tab

The following preferences apply globally to the Builder.

Builder Run Settings:


Append geometry with warningAdds current geometry to an existing file, after issuing a warning.
Append geometry without warning Adds current geometry to an existing file without issuing a warning.
Perform END before run to create new geometryImmediately terminates the current session of ASAP. Reinitializes ASAP after a run is completed by issuing SYSTEM NEW and RESET commands.

File Settings:


Set default file type to compressed .ENZ formatSelect to set the Save as type on the Save As dialog to smaller Builder (Compressed XML) Files (*.enz). The file is automatically decompressed when you open it in the Builder. Do not select to set the Save as type to the larger, uncompressed Builder (XML) Files (*.enx).

Menu Settings:


Show Menu Pictures

Small

Large

Select to view thumbnail pictures of geometry (edges, lenses, surfaces, emitters, and grids) on the Builder pop-up menu. Select Small or Large to designate picture size.
