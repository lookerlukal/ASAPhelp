RESTRICT (ASAP Command)

Further controls ray search within lens.

Syntax 

RESTRICT BACKWARD
         FORWARD
         OFF

Remarks


  1. Instead of a conicoid seeing itself and adjacent ones, the command further restricts the ray search to either the logical FORWARD or BACKWARD direction.
  2. Use OFF to go back to the default. Like the object SEARCH command, RESTRICT can be used to speed up ray traces.
