SHOW (ASAP Command)

Displays current settings.

Syntax 

SHOW [ ALL ]

Remarks


  1. SHOW presents the complete complex coefficients of POLARIZ.
  2. Prints out the current status of some or ALL of those commands that set flags or data used by other commands. These commands include:

    ACCURACY

    ARROWS

    AXIS

    BEAMS

    BILATERAL

    CLIP DIRECTION/POSITION

    CUTOFF

    FRESNEL

    HALT
