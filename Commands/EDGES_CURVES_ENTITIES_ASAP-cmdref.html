EDGES/CURVES/ENTITIES (ASAP Command)

Signals ASAP that edge definition commands follow.

Syntax 

EDGES   [ i ]
CURVES
ENTITIES

OptionDescription
istarting number for defining EDGES/CURVES

Remarks


  1. The default value for i is one more than the highest edge number previously defined. The i is initialized to one at start of program processing.
  2. EDGE, LENS, and SURFACE data currently reside in the same internal storage locations. Therefore, an EDGE number cannot be the same as an already defined LENSES or SURFACE number.
