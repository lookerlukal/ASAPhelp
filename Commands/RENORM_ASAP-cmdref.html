RENORM (ASAP Command)

Renormalizes coefficients of surface/function polynomials.

Syntax 

RENORM [ c ]
       [ term c ]
       [ MIN  c ]
       [ MAX  c ]
       [ FUNC c [ x y z ] ]

OptionDescription
cnormalization constant
term, MIN, or MAXnormalizes the specified coefficients to be exactly equal to c
x y zpoint at which function value is to be set

Remarks


  1. Renormalizes the coefficients of the previous surface function.
  2. If it is actually a surface (function is 0), the renormalization does not affect the location of the surface, but may change the direction of the normal to the surface or avoid numerical overflow problems.
  3. If no additional entries are made, the coefficients are normalized such that the largest and smallest ones are equally spaced in magnitude about 1.
  4. If a single numerical value is entered, all the coefficients are divided by c.
  5. If a term is specified (that is, XiYjZk or the MIN/MAX), then the coefficients are normalized so that the particular coefficient becomes exactly equal to c.
  6. The last form allows you to set the value of the FUNC to c at the point (x,y,z) (default is the reference point).
