$REG (ASAP Macro)

Displays the contents of registers and associated variables.

Syntax 

$REG [ register register' ... ] [ 'text' ]
&REG

Remarks


  1. Macro displays the contents of a given register or set of registers with optional 'text'
  2. If no registers/variables are entered, all of the registers/variables whose contents have changed since program startup are displayed.
  3. $REG prints out the internal ASAP register name in addition to the variable name(s) and value(s). &REG prints only the variable name(s) and value(s).
