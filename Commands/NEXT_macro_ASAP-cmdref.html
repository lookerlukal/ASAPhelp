$NEXT (ASAP Macro)
$NEXT (ASAP Macro)

Immediately begins processing of the next $DO or $ITER iteration by forcing a branch to the top of the current loop block.

Syntax 

$NEXT

Example 

$DO 1 10
{ :

This block is executed every iteration
:
$IF ? LE 5; $NEXT
:
This block is executed only for iterations 6 through 10
:
