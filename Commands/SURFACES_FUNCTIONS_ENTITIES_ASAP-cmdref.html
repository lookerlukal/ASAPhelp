SURFACES/FUNCTIONS/ENTITIES (ASAP Command)

Signals ASAP that surface definition commands follow.

Syntax 

SURFACES  [ i ]
FUNCTIONS
ENTITIES

OptionDescription
istarting number for defining SURFACES/FUNCTIONS

Remarks


  1. The default value for i is one more than the highest surface number previously defined. The i is initialized to one at start of program run.
  2. EDGE, LENS, and SURFACE data currently reside in the same internal storage locations. Therefore, a SURFACE number cannot be the same as an already defined EDGE or LENS number.
