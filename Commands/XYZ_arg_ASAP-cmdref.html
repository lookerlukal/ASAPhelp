XY[Z] and Other Plot Window Overrides (ASAP Command Argument)

Sets the window to the given directions and size for only the currently active plot.

Syntax 

XY[Z] [ m ]
YZ[X] a a' d d' 
ZX[Y]
YX[Z]
ZY[X]
XZ[Y]

Remarks


  1. ASAP command arguments are optional and must follow a command.
  2. If the third direction is given, an OBLIQUE plot is generated.
  3. If the plot ranges, a a' d d' are not given, the plot is autoscaled (and then the window optionally magnified by a factor m). A good use of the option is for controlling the direction cosine space window without affecting the current spatial WINDOW command. For example,
SPOTS DIR YX -4@1
