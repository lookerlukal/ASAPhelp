RANGE (ASAP Command)

Adjusts the vertical plotting scale of current distribution data.

Syntax 

RANGE [ d [ d' ] ]

OptionDescription
dminimum values to be plotted
d'maximum values to be plotted

Remarks


  1. Expands or recalculates the minimum d and/or maximum d' values to be plotted by the GRAPH, PLOT3D, ISOMETRIC, CONTOUR and MESH commands.
  2. Plots different data at the same fixed scale.
  3. Makes the plotting range only larger than the actual data range.
  4. Use the THRESHOLD command to "clamp" the data values to a smaller range.
