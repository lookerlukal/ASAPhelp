MOVE PARABASALS (ASAP Command)

Moves the parabasal rays for each ray/beam to its base ray plane.

Syntax 

MOVE PARABASALS

Remarks


  1. Moves parabasals to a plane perpendicular to the base point. It does not in any way change the beam characteristics, but does affect the PLOT BEAMS display since, by default, the parabasals lie on the current object surface.
