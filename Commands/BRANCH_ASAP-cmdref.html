BRANCH (ASAP Command)

Specify an absolute or relative branch of the object name tree for future object definitions.

Syntax

BRANCH [ name ]
        .name
          ^
       ^.name

Remarks


  1. Specify an absolute or relative branch of the object name tree for future object definitions and display the current or new full branch name.
  2. Periods (.) are used as node separators in the name. Carets (^) are used to move up one node. If the entry begins with either of these two characters, then the branch is defined relative to the current branch.
  3. Otherwise, it is an absolute complete branch specification (with possibly many embedded periods).
  4. The number of levels is limited by the total object name length, which is 344 characters.
