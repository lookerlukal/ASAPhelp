CONSIDER (ASAP Command)

Selects a set of objects for ray tracing and plotting.

Syntax 

CONSIDER [ ALL  ]
           NONE
           ONLY   [ i i' ... ]
           EXCEPT
           ADD
           REMOVE

OptionDescription
ALLconsider all known objects
NONEconsider no objects
ONLYconsider only the objects specified (default is the current GROUP)
i i' ...OBJECT numbers or names to be considered
EXCEPTconsider all objects except those specified (default is the current GROUP)
ADDadds the specified objects to the previous CONSIDER command
REMOVEremoves the specified objects from the previous CONSIDER command

Remarks


  1. Provides control over the current set of objects ASAP is to consider in all calculations and output. All objects remain in the database at all times (that is, even when they are temporarily ignored with options other than ALL, they are not deleted).
  2. A CONSIDER command by itself (with no entries) produces a list of the currently considered objects.
  3. By default all objects in the system database are used.
  4. The EXCEPT option restricts the current object set to all objects except those specified (default is the current GROUP). All objects except those may be drawn or ray traced.
  5. Particular object numbers (or names with "?" wildcards) can be either excluded with the EXCEPT option or the ONLY ones considered. If no object list is given, the last GROUP is used.
  6. The ADD option adds the specified objects to the previous CONSIDER command.
  7. The current state (that is, a previous CONSIDER command) can be updated using ADD or REMOVE instead of ONLY or EXCEPT, respectively.
  8. Ray data are initially referenced by OBJECT 0. Before ray tracing, CONSIDER EXCEPT 0 effectively turns off the ray data just as it would any other object.
  9. Ray data currently associated with a given object, that is, after a ray trace, are affected in exactly the same way as the object itself.
  10. When using OBJECT names that are allowed to be up to 344 characters in length, you can use only 40 characters in the name when you CONSIDER them. If you distinguish objects by a number past the last decimal point. You must limit the name to 40 characters or less including that number, or it cannot be seen.
