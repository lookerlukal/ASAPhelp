$ARGS (ASAP Macro)

Controls argument prompting in user-defined macros.

Syntax 

$ARGS [ ALL ]
        NONE
        USER
        SCR

Remarks

The SCR feature displays a dialog box for prompts with arguments. After pressing Enter after each entry, click the OK or Cancel button as appropriate to exit the dialog.

Interactive prompts are displayed for either ALL NONE or only the ones defined by the USER in the macro definition. The default in the absence of any $ARGS command is ALL. If no argument is given on the command, the state before the last $ARGS command is restored.
