$PATH (ASAP Macro)

Sets the default directory "path" for future file openings.

Syntax 

$PATH [ "path" ]

Remarks


  1. If no path is given, then reverts to the previous one.
  2. "path" must include the closing directory character (\, / or ] depending on the operating system).
  3. The default files (see $IO) are closed in the old directory, and then reopened in the new one.
