IMAGE (Global Point) (ASAP Command)

Images a global point through a specified LENS entity.

Syntax 

IMAGE k x y z k'

OptionDescription
k k'starting/ending spaces
x y zglobal point

Remarks


  1. Images the given point x y z in space k of the current lens into space k' and displays the coordinates.
  2. The spaces are numbered from 1 (before the first conicoid) to N+1 (after the last conicoid N).
  3. Uses the auxiliary axis technique that steps the image through the centers of curvature of the conicoids (the aperture semi-diameters, conic constants and obscuration ratios of the conicoids do not affect the imaging) and always produces an image even if it is virtual or unphysical. The resulting imaging transform is stigmatic (points go into points) but not necessarily collinear (lines go into lines). Therefore, it is only an approximation since in any real optical system, the image is aberrated (not a perfect point focus).
  4. The IMAGE command used in pre-ASAP 5.1 versions is now named PICTURE.
