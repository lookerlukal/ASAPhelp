EMITTING IES (ASAP Command)

Creates a ray set that emits according to Illuminating Engineering Society (IES) data.

Syntax 

EMITTING IES name X x y z n [ a ]
                  Y
                  Z

OptionDescription
namename of IES data file with extension .IES
X, Y or Zcoordinate axis to associate with the file's polar axis
x y zemitter coordinates
nnumber of rays to create
aazimuthal angle (in degrees)

Remarks


  1. Creates a ray set that emits according to the data in an IESNA Standard Photometric Data file called name.ies. EMIT IES and DISPLAY;IESFILE follow LM-63-1995 standard.
  2. The coordinate axis corresponds to the file's polar axis (vertical if type A or C photometry, horizontal if B).
  3. An optional azimuthal angle a (in degrees) can also be specified to rotate the angular pattern around the polar axis.
  4. The USERAPOD settings do not affect EMITTING IES since the equivalent information is found in the IES file.
