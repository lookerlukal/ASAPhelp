SUBSET (ASAP Command)

Selects a subset of the current ray set and rejects remaining ray data.

Syntax 

SUBSET [ RESET [ m ] ]

Remarks


  1. Collapses ray storage to include only those rays determined by the last CONSIDER, SELECT, and CUTOFF commands. All other ray information is lost.
  2. Useful when a lot of rays have been traced and only a small subset is analyzed extensively.

    Optionally RESET the following information:


    RESET InformationRESET value
    Current object0
    Previous objects0
    Number of hits0
    Specular splits0
    Scatter levels0
    Parent rayitself
    Optical Path Length0 for ray 1
    Current mediumm if specified
  3. Useful for preparing the output from one optical system (for example, a complex source) to be the input for a completely different one.
