$ERR (ASAP Macro)

Branches on an input error.

Syntax 

$ERR label
     :
label
     :

Remarks


  1. Sets a flag so that when an input error occurs, ASAP skips records until a record starting with the label string is found. Normal input processing resumes on the next record, and the flag is reset.
  2. The label must start in column 1 of the input record.
