$UNVAR (ASAP Macro)

Determines the type of message to be issued for future uninitialized (used before set) variables/registers.

Syntax 

$UNVAR [ NONE ]
         WARN
         ERROR

Remarks

The default is WARN, that is, a message but not a fatal ERROR. If no entry is given, the state before the last $UNVAR command is restored.
