$STO/$RCL/&STO (ASAP Macro)

Stores or recalls variable data to or from a file.

Syntax 

$RCL [ file ]
$STO          [ reg reg' ]
&STO

Remarks


  1. Stores or recalls the range of given registers (or the entire register set) to or from a binary($) or text(&) "file") The default file is LASTVALS.REG.
  2. $RCL reads in either a binary or text file. If it does not recognize the requested format, it tries the other.
  3. &STO writes the .REG file in an ASCII format.
