$LEAVE (ASAP Macro)

Causes premature exit from a loop, macro, or input file.

Syntax 

$LEAVE [ n ]

Remarks


  1. Macro forces an immediate exit from the current DO loop, macro, or input file in that order.
  2. The n is the number of construct levels to exit (default is 1). If the above loop was also inside a macro and you wanted the test to exit not only the loop but also the macro, then set n to 2.
