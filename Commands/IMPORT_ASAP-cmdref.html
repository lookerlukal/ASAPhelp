IMPORT (ASAP Command)

Import a lens from a ZEMAX file.

Syntax 

IMPORT X x file [ k ] [ options ... ]
       Y y
       Z z

OptionDescription
filespecifies ZEMAX file name
klens configuration
optionsfrom ABERRATIONS command

Remarks


  1. Import lens (configuration k) from ZEMAX file position reference (GLRS or first) surface at a given plane, and run any ABERRATIONS options.
  2. Supports these ZEMAX surface types: STANDARD, ODDASPHE/EVENASPH (approximate), DGRATING (Sweatt model), PARAXIAL, and TILTSURF.
Note: Surfaces are restricted mostly to centered systems, since COORDBRK "surfaces" are currently ignored. This command assumes that any referenced ZEMAX AGF glass catalog files have been converted to ASAP CAT format with the GLA2CAT command line utility.
