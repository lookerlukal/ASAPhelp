SCATTER REPEAT (ASAP Command)

Assigns the scatter characteristics of a given object to the current object.

Syntax 

SCATTER REPEAT [ i ]

OptionDescription
iobject number

Remarks


  1. Assigns the scatter characteristics of object i (or the previous object) to the current object.
  2. If i is zero, then all the scatter properties are removed.
