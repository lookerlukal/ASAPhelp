COARSEN (ASAP Command)

Converts the current edge to a piecewise linear curve.

Syntax 

COARSEN [ m ]

OptionDescription
muse every mth point

Remarks


  1. Converts the current edge to a piecewise linear curve using every mth (default 1) point. This is the opposite of the SMOOTH command.
