GRIN (ASAP Command Argument)

Similar to the command argument, ABSORB, GRIN handles the gradient index (GRIN) materials by assigning to the medium a GENERAL polynomial or USERFUNC function.

Syntax 

... [ GRIN k p t [ l ] ]

OptionDescription
kits magnitude is the SURFACE designation for this function
tstep length used for tracing a ray when tracing a ray in this inhomogeneous medium
lmaximum number of steps a ray can take in the medium (default 1000)
