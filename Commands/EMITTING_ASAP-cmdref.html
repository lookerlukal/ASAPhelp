EMITTING (ASAP Command)

Creates a composite single-source emitter from several EMITTING volume commands.

Syntax 

EMITTING
w   DISK    ...
    RECT
    ENTITY
    OBJECT
    CONE
    PYRAMID
    BOX
    SPHEROID
    RAYS
    FILAMENT
    HELIX
    IES
    DATA
w' ...
w" ...
   :

OptionDescription
w w' w"emitting commands flux weighting factors

DISK, RECT,

ENTITY, OBJECT,

CONE, PYRAMID,

BOX, SPHEROID,

RAYS, FILAMENT,

HELIX, IES, DATA

specific emitting commands

Remarks


  1. Any of the EMITTING types may be used in a composite (single-source) emitter using the EMITTING format.
  2. The w's are the flux weighting factors for the individual component emitters.
  3. Only linear transformation commands are allowed between the individual emitter commands.
  4. Any USERAPOD commands defined before EMITTING is applied to all of the emitting types.
