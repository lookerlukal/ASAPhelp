COMPOSITE (Edge Modifier) (ASAP Command)

Combines several edge entities into a single edge entity.

Syntax 

COMPOSITE [   -n    ] [ GAPS q [ q' ] ]
           i [ i']

OptionDescription
-nlast n edges
i i'edges i through i'
GAPS q q'connection factors

Remarks


  1. COMPOSITE combines the last n edges, edges i through i', or all edges defined since the last EDGE command into one edge.
  2. GAPS between the endpoints of one edge and the start of the next can be handled as follows.
    qACTION
    -1merge gap points into one
    0leave gaps open (default)
    1linearly connect gaps
  3. Specify q' for the last-to-first point gap to make a single edge object from a set of discontinuous edges.
