$PAGE (ASAP Macro)

Pauses text output.

Syntax 

$PAGE [ n   ]
        OFF

Remarks


  1. Macro causes future text output to be paused every n lines (default 12); that is., it waits for you to press Enter to continue.
  2. Pausing can be turned OFF with this command or by typing a caret “^” at a pause.
