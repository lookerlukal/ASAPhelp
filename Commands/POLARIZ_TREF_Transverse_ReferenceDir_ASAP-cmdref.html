POLARIZ TREF (Transverse Reference Direction) (ASAP Command)

Sets polarization reference coordinate system.

Syntax 

 POLARIZ TREF [a b c]
 POLARIZ TREF [X|Y|Z]
 POLARIZ TREF AUTO

Option

Description

TREF Polarization transverse direction for the reference rays
a b c Direction cosine in the global coordinate of the polarization transverse reference direction for the reference ray. This option can be global X, Y, Z.
AUTO ASAP automatically determines the local transverse reference polarization axis with a process similar to conformal wrapping; that is, directly aligning global Z axis to the reference ray direction with a minimum rotation.

Remarks


  1. ASAP enforces the orthogonality between the reference ray direction and polarization transverse reference direction.
