INTERFACE REPEAT (ASAP Command)

Assigns the interface characteristics of either a specified object or the previous object to the current object.

Syntax 

INTERFACE REPEAT [ i ]

OptionDescription
iobject number

Remarks


  1. If i is zero, all the interface properties are removed.
