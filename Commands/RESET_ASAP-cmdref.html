RESET (ASAP Command)

Reinitializes all control settings to those at program startup.

Syntax 

RESET

Remarks


  1. Resets all ray data and performs the equivalent of the RAYS 0 command.
    Note: PARABASALS and WIDTHS are also set to 0 and 1, respectively. This effectively disables any previously stated BEAMS COHERENT DIFFRACT.
  2. Does not affect system description (use SYSTEM NEW to reset this).
