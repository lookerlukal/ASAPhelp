SMOOTH (ASAP Command)

Quadratically (or cubically) smoothes the current curve.

Syntax 

SMOOTH [ 2 ]
         3

OptionDescription
2quadratically smooth the current curve (default)
3cublicly smooth the current curve

Remarks


  1. Quadratically (2, the default) or cubicly (3) smoothes the current curve assuming it is piecewise linear, that is one created by the ELLIPSE, ROUNDED, or OVAL commands.
