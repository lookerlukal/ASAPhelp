EXTEND (ASAP Command)

Linearly extends one or both ends of a curve edge.

Syntax 

EXTEND d [ d' ]

OptionDescription
dstart of edge
d'end of edge

Remarks


  1. Linearly extends the start and end of the current edge a distance d and d' respectively.
  2. The default for d' is d. Enter a zero distance if you want only to extend the other end.
