$FCN (ASAP Macro)
$FCN (ASAP Macro)

Defines new in-line math functions.

Syntax 

$FCN [ name [ e e' e" ... ] ]

Remarks


  1. Macro defines a new in-line math function with up to a 32-character name. The new function may be used in future arithmetic expressions just like the intrinsic functions SIN, COS, SQRT, and so on.
  2. The function is defined by the expressions ee'e" entered after the name. Up to 60 of these functions can be defined at any one time. The result of the last expression is the function value returned during processing.
  3. The dummy argument of the function is represented in the defining expressions as the base argument register "_". For example:

    $FCN SECH 2/(EXP(_)+EXP(-_))

  4. If no expressions are given, the function, name is deleted from internal storage. A $FCN command with no other entries deletes all current user functions.
  5. These user-definable functions are recursive; that is, you may include one in the argument of another. Also, any registers/variables defined in the function before they are used are automatically assumed to be local to that instance of the function, and therefore do not conflict with any external (for example, global) ones of the same name.
