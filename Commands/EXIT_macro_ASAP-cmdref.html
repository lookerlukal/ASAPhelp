$EXIT (ASAP Macro)

Starts immediate program termination.

Syntax 

$EXIT

Remarks


  1. $EXIT is supported only in batch mode, not in the user interface.
  2. Immediately terminates program and returns control to the operating system. If the program is in interactive mode, you are asked to verify your decision to terminate program.
  3. $EXIT bypasses the normal ASAP shell program. As a result, it does not rename certain files from their BRO0XX.DAT designation.
