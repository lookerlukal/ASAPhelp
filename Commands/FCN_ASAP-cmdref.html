FCN (ASAP Command)

Specifies a function to wrap around the function of the previous surface.

Syntax 

FCN fcn

OptionDescription
fcnname of the specified macro function

Remarks


  1. Specifies a macro function (intrinsic, for example, SIN, or user-defined $FCN) to wrap around the previous surface's base function.
