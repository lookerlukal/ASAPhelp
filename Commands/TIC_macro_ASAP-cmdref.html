$TIC (ASAP Macro)

Displays elapsed and CPU time.

Syntax 

$TIC
&TIC

Remarks


  1. $TIC macro displays the number of CPU seconds that have elapsed since the last $TIC or &TIC macro was processed.
  2. &TIC macro displays nothing and resets the timer.
  3. Normally the time "units" are allowed to float, but can be fixed for this and all future outputs to MICroseconds, MILIiseconds, SEConds, MINutes, HOUrs, DAYs, WEEks or YEArs.
  4. Use OFF to go back to the default floating units. The display resolution is always 1/100 of the selected units and is limited to less 10,000 total units.
