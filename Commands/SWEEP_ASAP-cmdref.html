SWEEP (ASAP Command)

Specifies the optional sweeping of a curve into a surface.

Syntax 1 

SWEEP DIR  d [ a,b,c ]
      TO   x y z
      POS  f [ x y z ]
      AXIS t   a,b,c  [ x y z ]

Syntax 2 

SWEEP [ OFF ]
        d
        f
        t

OptionDescription
DIR POS AXISsweep type (see Remarks)
dactual sweep distance
a b csweep direction or axis
TOsweep first point on edge TO
x y zsweep point
ffractional sweep distance
tsweep angle
OFFturns off sweep

Remarks


  1. If DIRection, sweep an actual distance d along direction a,b,c (default is normal to best fit plane through edge points) or sweep the first point on edge TO x y z.
  2. If POSition, sweep edge a fractional distance f towards point x y z (default is centroid of edge points).
  3. If AXIS, sweep the edge an angle t in degrees around the axis specified by the given direction and point.
  4. This information is used to define the surface of a single edge object or a BOUNDS edge.
  5. The SWEEP can be turned OFF or modified with the second syntax.
