$RAN (ASAP Macro)

Resets the random number seed for the "~" operator to the integer i (default 2000000001).

Syntax 

$RAN [ i ] [DEC]
      BEST

Remarks


  1. The default uniform random number generator (upon which all others are based) is identical to that used on the DEC (Digital Equipment Corporation) systems. You can switch to the BEST possible generator based on the same algorithm but using different numerical constants and thus a different seed sequence.
  2. $RAN affects only the ~ (tilde) operator. Use SEED to reset the seed for random numbers for all other random number generation; for example for GRID RANDOM and EMITTER commands.
