ENCLOSED (ASAP Command)

Calculates an encircled (ensquared) energy using a square array of points.

Syntax 

ENCLOSED [ i j ] [ 'title' ]
           MAX

OptionDescription
i jpixel center
MAXmaximum point
'title'optional title for plot (up to 64 characters)

Remarks


  1. Sums the data within a square array of points centered about the pixel i j, the MAXimum point, or the centroid and plots the percent enclosed as a function of the size of the square.
  2. The title is delimited by a single quote ' , as shown.
