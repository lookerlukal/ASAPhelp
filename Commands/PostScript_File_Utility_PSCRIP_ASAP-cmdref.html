PostScript File Utility (PSCRIP) (ASAP Command)

The PSCRIP.exe utility reads ASAP PLR (plot) files and creates PostScript files for printing. The utility is stored in your ASAP installation area under the \bin folder.

Syntax 1 

PSCRIP

Syntax 2 

PSCRIP <‘file’>

‘file’ is an input text file of six lines with the following contents:

plrname

s

‘printer’

c

p

[blank line]


OptionDescription
plrnamename of a file with an extension .plr that contains the plots stored in asaptemp.plr
sscaling factor (1 = 1 plot on a printed page; .5 creates 2 x 2 plots on a page, and so on)
‘printer’network name of a PostScript printer or name of a directly attached printer
c

colors:

1 = Different thicknesses instead of colors

2 = Old BRO colors

3 = Colors from xtermc utility

4 = New BRO colors

ppage orientation (L = landscape, P = portrait)
[blank line]a blank line is required at the end of the file

Remarks


  1. In Syntax 1, you are prompted for input to satisfy the same options in file.
  2. In Syntax 2, the options are all given in the text file, file.
