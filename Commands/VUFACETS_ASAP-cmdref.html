VUFACETS (ASAP Command)

Shortcut for faceting and viewing current objects.

Syntax 

VUFACETS [ n n' [ m ] ] [ LIST ]

OptionDescription
n n'faceting numbers
mmaximum number of facets per object

Remarks


  1. VUFACETS is a shortcut for the $IO VECTOR REWIND, PLOT FACETS [ n n' ], and $VIEW command sequence. However, it temporarily suppresses any 2D plotting to minimize computation time and disk space.
  2. Replots the current accumulated 3D VECTOR graphics taking into account the current CONSIDER settings.
  3. If possible, the maximum number of total facets on any object is kept below m (the default is 1000; use 0 to turn off).
  4. Remembers the faceting numbers n n' m" from the previous VUFACETS command (an END command resets them to the default 1).
  5. A $VIEW will list the facets in the Command Output window.
  6. The numbers of facets created for each object can also be LISTed (a $VIEW is not automatically issued in this case).
