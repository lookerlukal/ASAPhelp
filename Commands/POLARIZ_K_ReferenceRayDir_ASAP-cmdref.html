POLARIZ K (Reference Ray Direction) (ASAP Command)

Sets the direction of the reference ray.

Syntax 

POLARIZ K [a b c]
POLARIZ K [X|Y|Z]

Option

Description

K Setting reference ray direction for the non-collimated rays
a b c Direction cosine of the reference ray. Alternatively, you can specify a global axis as a reference ray direction.
