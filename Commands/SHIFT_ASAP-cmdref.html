SHIFT (ASAP Command)

Specifies a relative shift of an entity along the coordinate axes.

Syntax 1 

SHIFT [ x y z ] [ LIST ]
        X x
        Y y
        Z z

Syntax 2 

SHIFT d ALONG a,b,c [ LIST ]

OptionDescription
X, Y, Zshifts axis
x, y, zrelative shift along coordinate axis
d distance
a b cdirection vector
LISTdecodes transformation matrix into simple operations (if possible) and prints

Remarks


  1. Translates the entity BY the given distances (x,y,z). The default distances move the entity's reference point to the global origin.
  2. To shift a distance d along a given direction (a,b,c) use the second syntax.
  3. When used with REPEAT, group SHIFT with these commands: MATRIX; ROTATE; SCALE; SKEW; PLACE; ALIGN; XEQ

Examples 

!!++
!! SHIFT02.INR
!! Title: Shifting Rays
!! Category: Isolated Command
!! Keywords: SHIFT 
!! Description: SHIFT command is used in same
!! way for an Object or an Entity.
!! Edit History (latest first)
!! 02/018/2002 - cl - creation
!!--

!! Shift rays 15 units parallel to Z axis
RAYS
    SHIFT Z 15
