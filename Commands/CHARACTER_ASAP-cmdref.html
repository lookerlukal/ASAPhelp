CHARACTER (ASAP Command)

Creates an edge of a specified string.

Syntax 

CHARACTER X x y [ z ] string
          Y y z   x   'string'
          Z z x   y

OptionDescription
X, Y or Zcharacter plane
x, y or zcharacter plane location
y, z or xcharacter width
z, x or ycharacter height (defaulted to the width)
string or 'string'edge character string

Reference Point

First point of character

Remarks


  1. Creates an edge patterned after the character string.
  2. This edge is a combination of coplanar straight line segments.
