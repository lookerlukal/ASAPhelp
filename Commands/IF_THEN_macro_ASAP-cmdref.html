$IF THEN (ASAP Macro)

Use the THEN keyword at the end of a $IF statement to trigger processing of conditional block structures.

Syntax 

   
  $IF a rel b [ log a’ rel’ b’ log’ ... ] THEN
   :
[ $ELSEIF ... THEN ]
   :
[ $ELSE ]
   :
  $END[IF]

Remarks

These structures can be nested up to 99 deep. However, the control words must be uniquely and identically indented at each level of nesting. This mandatory indentation significantly speeds processing and is also a good standard programming practice. The following example shows an IF structure inside a DO loop block.

Example 

$DO 1 3
  { $IF ?\2 EQ 0 THEN
       A?=?
    $ELSE
       B?=?
    $ENDIF }

  1. Notice the required vertical alignment of the $IF, $ELSE, $ENDIF commands.
  2. Using a $GO to jump into a block may cause unexpected results.
  3. Only a forward $GO to a label can be safely used to jump out of a block.
