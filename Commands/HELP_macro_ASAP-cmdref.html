$HELP (ASAP Macro)

Displays information on all or one command or a listing of the MACROS in the current library.

Syntax 

$HELP [ command ]

Remarks


  1. If no entry is given, an abbreviated list of all commands is displayed.
  2. If only one command (up to two literals) is specified, a complete one or two paragraph explanation of the command is shown.
  3. The &HELP form displays only the format of the command with no accompanying text.
