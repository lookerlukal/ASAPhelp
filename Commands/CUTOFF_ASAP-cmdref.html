CUTOFF (ASAP Command)

Sets the conditions controlling ray termination.

Syntax 

CUTOFF [ t ] [ n ]

OptionDescription
tabsolute flux threshold
nmaximum number of total object intersections for any ray

Remarks


  1. Causes rays to be ignored.
  2. Sets the absolute flux threshold; below this value, ASAP ignores the rays to the decimal number t (default 1.E-18).
  3. The maximum number of total object intersections for any ray can be set to the integer n (default 1000).
  4. Tracing of non-existent CUTOFF rays cannot be continued.
