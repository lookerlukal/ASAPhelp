$ECHO (ASAP Macro)

Controls display of input commands.

Syntax 

$ECHO [ ALL ]
        NONE

Remarks


  1. Macro echoes (redisplays) ALL input commands or NONE. The macro overrides the control of echoing as specified on the call to a user macro.
  2. A $ECHO command by itself (no argument) restores this default mode.
