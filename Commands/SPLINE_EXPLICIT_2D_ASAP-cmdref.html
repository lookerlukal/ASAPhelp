SPLINE, EXPLICIT 2D (ASAP Command)

Explicit 2D curvature (G2) continuous, piecewise cubic spline curve in the given plane (default 0).

Syntax 

SPLINE X [ x ]
       Y   y
       Z   z
h  s   d   [ c ]
h' s' [ d']
[ h" s" [ d"] ]
:

Remarks


  1. Defined in terms of subsequent heights (next coordinate), slopes, optional distances (remaining coordinate), and starting curvature c (inverse radius).
