DIMENSIONS (ASAP Command)

Displays a table of maximum array dimensions for the most important program arrays.

Syntax 

DIMENSIONS

Remarks


  1. The table information summarizes the storage capabilities of a given version of the program.
  2. Values listed in the form "number-1" usually indicate a limitation due to decimal encoding (either internally or for text output).
  3. The limit for "Ray Selection Flags" is set to 0. The 0 indicates that ASAP has not yet allocated any memory, but it will do so after the first SELECT command.
