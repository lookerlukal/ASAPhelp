$DBG (ASAP Macro)

Creates a debug listing of parser output.

Syntax 

$DBG  [ ... ]
      [ ... ]

Remarks


  1. Displays how the parser decoded a line of input into entries.
  2. The start, length, type, and value of each entry on the line is listed.
  3. If the line of input is on the $DBG macro itself, the line is not passed to ASAP.
