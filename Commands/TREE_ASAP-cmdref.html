TREE (ASAP Command)

Displays object name tree.

Syntax 

TREE [ ENTITIES ] [ i ] [ 'delimiter' ]

OptionDescription
ENTITIESdisplays the base entities and BOUNDS entities for each object
iindent space to use when displaying object name tree (default 2)
'delimiter'character delimiter separating each object name tree level (default is a period)

Remarks


  1. Displays object name tree indented by i (default 2) at each level, assuming they are separated by the single character delimiter (default is a period ( . )).
  2. The ENTITIES option also displays the base entities and signed BOUNDS entities for each object.
  3. The corresponding object (and entity) numbers are shown inside angle brackets < >.
