SEGMENTS (ASAP Command)

Controls number of segments plotted per arc.

Syntax 

SEGMENTS [ n ]

OptionDescription
nnumber of straight line segments used to approximate a 45 degree arc

Remarks


  1. Controls the number of straight-line segments used to draw conic arcs.
  2. The default number of line segments is 3. The minimum and maximum numbers are 1 and 45, respectively.
