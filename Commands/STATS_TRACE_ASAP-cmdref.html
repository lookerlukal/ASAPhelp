STATS TRACE (ASAP Command)

Prints statistics of a previous TRACE.

Syntax 

STATS TRACE [ MEDIA ] [ MODELS ] [ OBJECTS ]

Remarks


  1. If the previous TRACE had a STATS or ACCUM option on it, selectively print the given statistics.
  2. For OBJECTS (the default), only those currently CONSIDERed are printed.
