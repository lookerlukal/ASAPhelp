PLOT3D (ASAP Command)
PLOT3D (ASAP Command)

Creates a 3D representation of the current distribution data file with 1D cross-sections.

Syntax 

PLOT3D [ s ] [ t [ r ] ]

OptionDescription
sreadjusts the vertical scale
ttilt (default 18) (see Remarks)
rrotation (default 33) (see Remarks)

Remarks


  1. Produces a 3D hidden line representation of the data and a set of 1D (one-dimensional) cross-sections on the same plot.
  2. The s can be used to readjust the vertical scale of the plot downward (that is, s less 1).
  3. The actual view is controlled by the tilt t and rotation r angles in degrees (defaults 18 and 33). Both these angles should be greather than 1.5 degrees and less than 90.
  4. The OVERLAY command argument tells ASAP not to issue an end of plot so that the next plot created is overlaid with the current plot. This is useful for combining system plots with ray trace plots (assuming that the WINDOW is the same for all plots), multiple spot diagrams, and so on.
