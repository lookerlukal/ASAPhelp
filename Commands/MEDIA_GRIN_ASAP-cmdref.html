MEDIA GRIN (ASAP Command)

Creates a GRIN refractive material.

Syntax 

MEDIA [ m ]
n  [ n' n" . . . ]  GRIN k p t [ l ] [ 'name' ]
             :

OptionDescription
mstarting media number
nreal (or complex) refractive index
kGRIN function number
pexponent of GRIN function
tstep length to be used during ray trace
lmaximum number of ray steps in medium; default is 1000
namedescriptive name that can be assigned to this medium (only the first 24 characters after the comment delimiter are stored).

  4. The GRIN function is defined in global coordinates. This is of little consequence if the gradients are radial (there is no position dependence along the axis), but if the desired gradient is axial, you may have to SHIFT the gradient to the correct global coordinates to align the gradient with the object.
